## List Available Models

```bash
curl https://ai.kaiho.cc/v1/models \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Query a Single Model

```bash
curl https://ai.kaiho.cc/v1/models/{model_name} \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## Billing Types

| Type | Description | Applies To |
|------|-------------|------------|
| Token-based | Charged by input/output token count | Text generation, embeddings |
| Per-request | Fixed cost per request | Certain special endpoints |
| Image-based | Charged by quantity and resolution | Image generation |
| Video-based | Charged by duration and resolution | Video generation |
| Audio-based | Charged by audio duration | TTS, STT |
