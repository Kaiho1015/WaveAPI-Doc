Async processing mode -- submit a request and receive a task ID. Use the [Query Task Status](/en/api-reference/task/status) endpoint to retrieve generation results. Image links are valid for 24 hours -- please save them promptly.

<RequestExample>
```bash cURL
curl -X POST https://qingbo.dev/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "seedream-5.0",
    "action": "generate",
    "prompt": "一只柴犬穿宇航服站在月球上",
    "size": "16:9",
    "resolution": "2K",
    "watermark": false
  }'
```

## Available Models

| Model ID | Description |
|----------|-------------|
| `seedream-5.0` | Seedream 5.0, supports multi-image sets, web search, and streaming output |
| `seedream-4.5` | Seedream 4.5, supports multi-image sets and 4K resolution |
| `seedream-4.0` | Seedream 4.0, supports multi-image sets, 1K-4K resolution |

## Common Parameters

<ParamField body="model" type="string" required>
  Model group name: `seedream-5.0`, `seedream-4.5`, `seedream-4.0`
</ParamField>

<ParamField body="action" type="string" default="generate">
  Operation type
</ParamField>

<ParamField body="prompt" type="string" required>
  Text prompt, supports Chinese and English, recommended no more than 300 Chinese characters
</ParamField>

<ParamField body="image_urls" type="string[]">
  Reference image URL array, up to 14 images, each up to 10MB
</ParamField>

<ParamField body="size" type="string" default="1:1">
  Aspect ratio: `1:1`, `2:3`, `3:2`, `3:4`, `4:3`, `9:16`, `16:9`, `21:9`
</ParamField>

<ParamField body="resolution" type="string" default="2K">
  Resolution, available values depend on the model (see model tabs)
</ParamField>

<ParamField body="n" type="integer" default="1">
  Number of images to generate. Values greater than 1 enable multi-image set mode
</ParamField>

<ParamField body="watermark" type="boolean" default="true">
  Whether to add a watermark
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook callback URL upon task completion
</ParamField>

<ParamField body="extra" type="object">
  Additional parameters passed through to the upstream model (see model tabs)
</ParamField>

## Generation Modes

| Mode | Parameters | Description |
|------|-----------|-------------|
| Text-to-image | `prompt` | Pure text generation |
| Single image-to-image | `prompt` + `image_urls` (1 image) | Reference image + text |
| Multi-image-to-image | `prompt` + `image_urls` (2-14 images) | Multiple reference image fusion |
| Image set | `n > 1` | Generate a set of related images |

<Note>
In image set mode, the total of reference images + generated images must be 15 or fewer.
</Note>
