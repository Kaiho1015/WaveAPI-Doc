Async processing mode -- submit a request and receive a task ID. Use the [Query Task Status](/en/api-reference/task/status) endpoint to retrieve generation results.

<RequestExample>
```bash cURL
curl -X POST https://qingbo.dev/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini-3.1-flash-image-preview",
    "prompt": "一只橘猫趴在窗台上晒太阳，水彩画风格",
    "size": "16:9"
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    base_url="https://qingbo.dev/v1",
    api_key="YOUR_API_KEY"
)

response = client.images.generate(
    model="gemini-3.1-flash-image-preview",
    prompt="一只橘猫趴在窗台上晒太阳，水彩画风格",
    size="16:9"
)

print(response.data[0])
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":  "gemini-3.1-flash-image-preview",
        "prompt": "一只橘猫趴在窗台上晒太阳，水彩画风格",
        "size":   "16:9",
    }

    body, _ := json.Marshal(payload)
    req, _ := http.NewRequest("POST", "https://qingbo.dev/v1/tasks", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    resp, _ := http.DefaultClient.Do(req)
    defer resp.Body.Close()
    result, _ := io.ReadAll(resp.Body)
    fmt.Println(string(result))
}
```

```java Java
import java.net.http.*;
import java.net.URI;

public class Main {
    public static void main(String[] args) throws Exception {
        String payload = """
        {
          "model": "gemini-3.1-flash-image-preview",
          "prompt": "一只橘猫趴在窗台上晒太阳，水彩画风格",
          "size": "16:9"
        }
        """;

        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://qingbo.dev/v1/tasks"))
            .header("Authorization", "Bearer YOUR_API_KEY")
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(payload))
            .build();

        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString());
        System.out.println(response.body());
    }
}
```

## Available Models

| Model ID | Description |
|----------|-------------|
| `gemini-2.5-flash-image-preview` | Gemini 2.5 Flash image preview |
| `gemini-3-pro-image-preview` | Gemini 3 Pro image preview |
| `gemini-3.1-flash-image-preview` | Gemini 3.1 Flash image preview with high-resolution and search enhancement support |

## Common Parameters

<ParamField body="model" type="string" required>
  Model ID, see the model list above
</ParamField>

<ParamField body="prompt" type="string" required>
  Image description text, maximum 1000 characters
</ParamField>

<ParamField body="image_urls" type="array">
  Reference image URL array, up to 14 images, each up to 10MB
</ParamField>

<ParamField body="n" type="integer" default="1">
  Number of images to generate, fixed at 1
</ParamField>

## Response

Returns a task ID. Use [Query Task Status](/en/api-reference/task/status) to retrieve the final image result.

<ResponseField name="code" type="string">
  Status code, `"success"` on success
</ResponseField>

<ResponseField name="data" type="string">
  Task ID, formatted as `task-wave{timestamp}{userId}{random}`
</ResponseField>
