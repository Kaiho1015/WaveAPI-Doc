---
title: "Document Reranking"
description: "Rerank candidate documents by relevance to a query"
api: "POST https://ai.kaiho.cc/v1/rerank"
---

Synchronous endpoint -- reranks candidate documents by their relevance to a query text. Commonly used for RAG (Retrieval-Augmented Generation). Parameters are passed through directly to the upstream model.

<RequestExample>
```bash cURL
curl https://ai.kaiho.cc/v1/rerank \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "jina-reranker-v2-base-multilingual",
    "query": "什么是机器学习？",
    "documents": [
      "机器学习是人工智能的一个分支",
      "今天天气很好",
      "深度学习使用神经网络进行特征学习"
    ]
  }'
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model": "jina-reranker-v2-base-multilingual",
        "query": "什么是机器学习？",
        "documents": []string{
            "机器学习是人工智能的一个分支",
            "今天天气很好",
            "深度学习使用神经网络进行特征学习",
        },
    }

    body, _ := json.Marshal(payload)
    req, _ := http.NewRequest("POST", "https://ai.kaiho.cc/v1/rerank", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    resp, _ := http.DefaultClient.Do(req)
    defer resp.Body.Close()
    result, _ := io.ReadAll(resp.Body)
    fmt.Println(string(result))
}
```

```java Java
import java.net.http.*;
import java.net.URI;

public class Main {
    public static void main(String[] args) throws Exception {
        String payload = """
        {
          "model": "jina-reranker-v2-base-multilingual",
          "query": "什么是机器学习？",
          "documents": [
            "机器学习是人工智能的一个分支",
            "今天天气很好",
            "深度学习使用神经网络进行特征学习"
          ]
        }
        """;

        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://ai.kaiho.cc/v1/rerank"))
            .header("Authorization", "Bearer YOUR_API_KEY")
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(payload))
            .build();

        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString());
        System.out.println(response.body());
    }
}
```

## Request Parameters

<ParamField body="model" type="string" required>
  Reranking model ID, e.g., `jina-reranker-v2-base-multilingual`
</ParamField>

<ParamField body="query" type="string" required>
  Query text
</ParamField>

<ParamField body="documents" type="string[]" required>
  Candidate documents array
</ParamField>

<ParamField body="top_n" type="integer">
  Return the top N most relevant documents
</ParamField>
