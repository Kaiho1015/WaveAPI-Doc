Async processing mode -- submit a request and receive a task ID. Use the [Query Task Status](/en/api-reference/task/status) endpoint to retrieve generation results. Video links are valid for 24 hours -- please download promptly.

<RequestExample>
```bash cURL
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "seedance-2.0",
    "action": "generate",
    "prompt": "一只柴犬穿宇航服在月球上行走，电影感光影",
    "aspect_ratio": "16:9",
    "duration": 5,
    "resolution": "1080p",
    "watermark": false,
    "extra": {
      "generate_audio": true
    }
  }'
```

```go Go
    req.Header.Set("Content-Type", "application/json")

    resp, _ := http.DefaultClient.Do(req)
    defer resp.Body.Close()
    result, _ := io.ReadAll(resp.Body)
    fmt.Println(string(result))
}
```

```java Java
import java.net.http.*;
import java.net.URI;

public class Main {
    public static void main(String[] args) throws Exception {
        String payload = """
        {
          "model": "seedance-2.0",
          "action": "generate",
          "prompt": "一只柴犬穿宇航服在月球上行走，电影感光影",

          "resolution": "1080p",

          "extra": {
            "generate_audio": true
          }
        }
        """;

        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://www.qingbo.dev/v1/tasks"))
            .header("Authorization", "Bearer YOUR_API_KEY")
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(payload))
            .build();

        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString());
        System.out.println(response.body());
    }
}
```

## Available Models

| Model ID | Description |
|----------|-------------|
| `seedance-2.0` | Seedance 2.0, supports native audio, first/last frame, multimodal reference, up to 2K |
| `seedance-2.0-fast` | Seedance 2.0 fast version |
| `seedance-1.5-pro` | Seedance 1.5 professional version |
| `seedance-1.0-pro-fast` | Seedance 1.0 fast version |
| `seedance-1.0-pro-quality` | Seedance 1.0 high-quality version |

## Common Parameters

<ParamField body="model" type="string" required>
  Model ID
</ParamField>

<ParamField body="action" type="string" default="generate">
  Operation type
</ParamField>

<ParamField body="prompt" type="string" required>
  Video description text, supports Chinese and English. 2.0 supports referencing assets in the prompt with `@image1`, `@video1`, `@audio1`
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
  Aspect ratio: `16:9`, `9:16`, `4:3`, `3:4`, `21:9`, `1:1`
</ParamField>

<ParamField body="duration" type="integer" default="5">
  Video duration (seconds), range depends on the model (see model tabs)
</ParamField>

<ParamField body="resolution" type="string">
  Output resolution, available values depend on the model (see model tabs)
</ParamField>

<ParamField body="image_urls" type="string[]">
  Reference image URL array, quantity limits depend on the model version
</ParamField>

<ParamField body="first_frame" type="string">
  First frame image URL
</ParamField>

<ParamField body="last_frame" type="string">
  Last frame image URL (requires `first_frame`)
</ParamField>

<ParamField body="seed" type="integer" default="-1">
  Random seed, range [-1, 2^32-1]. `-1` for random. Same seed + same request produces similar results
</ParamField>

<ParamField body="watermark" type="boolean" default="false">
  Whether to add a watermark
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook callback URL
</ParamField>

<ParamField body="extra" type="object">
  Additional parameters passed through to the upstream model (see model tabs)
</ParamField>

## Model-Specific Parameters

<Tabs>
  <Tab title="Seedance 2.0">
    **duration range:** 4-15 seconds

    **resolution options:** `480p`, `720p`, `1080p`, `2K`

    **size additional support:** `adaptive` (auto-match input asset aspect ratio)

    **Input limits:** Up to 9 images (up to 30MB), up to 3 videos (up to 50MB, total duration up to 15s), up to 3 audio files (MP3/WAV, up to 15MB, total duration up to 15s)

    <ParamField body="extra.generate_audio" type="boolean" default="false">
      Native audio-video joint generation, supports dialogue lip-sync (8+ languages)
    </ParamField>

    <ParamField body="extra.return_last_frame" type="boolean" default="false">
      Return last frame image URL, useful for video clip continuation
    </ParamField>

    <ParamField body="extra.tools" type="array">
      Model tools, supports web search

      ```json
      "extra": { "tools": [{"type": "web_search"}] }
      ```
    </ParamField>

    <ParamField body="extra.service_tier" type="string">
      Service tier, controls the trade-off between latency and cost
    </ParamField>

    <Note>
    When using first/last frame mode (`first_frame` + `last_frame`) in 2.0, `video_urls` and `audio_urls` cannot be passed simultaneously.
    </Note>
  </Tab>

  <Tab title="Seedance 2.0 Fast">
    **duration range:** 4-15 seconds

    **resolution options:** `480p`, `720p`, `1080p`, `2K`

    Same parameters as Seedance 2.0, with faster processing.

    <ParamField body="extra.generate_audio" type="boolean" default="false">
      Native audio-video joint generation
    </ParamField>

    <ParamField body="extra.return_last_frame" type="boolean" default="false">
      Return last frame image
    </ParamField>
  </Tab>

  <Tab title="Seedance 1.5 Pro">
    **duration range:** 4-12 seconds

    **resolution options:** `480p`, `720p`, `1080p`

    **Input limits:** Up to 2 images (1 first frame + 1 last frame)

    <ParamField body="extra.audio" type="boolean" default="false">
      Whether to generate accompanying audio
    </ParamField>

    <ParamField body="extra.camerafixed" type="boolean" default="false">
      Fixed camera mode, keeps the camera position stationary when enabled
    </ParamField>
  </Tab>

  <Tab title="Seedance 1.0 Pro">
    **Two sub-versions:** `seedance-1.0-pro-fast` (fast) and `seedance-1.0-pro-quality` (high quality)

    **duration range:** 2-12 seconds

    **resolution options:** `480p`, `720p`, `1080p`

    **Input limits:** Up to 1 image

    <Note>
    The 1.0 Quality version supports `last_frame` and `reference` (style reference) roles. 1080p resolution is not supported when using the reference role.
    </Note>
  </Tab>
</Tabs>

## Generation Modes

| Mode | Parameters | Supported Versions |
|------|-----------|-------------------|
| Text-to-video | `prompt` | All |
| Image-to-video | `prompt` + `image_urls` (1 image) or `first_frame` | All |
| First/Last frame | `first_frame` + `last_frame` | 1.0-quality / 1.5 / 2.0 |
| Multimodal reference | `image_urls` + `video_urls` + `audio_urls` | 2.0 only |
