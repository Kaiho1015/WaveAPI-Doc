Async processing mode -- submit a request and receive a task ID. Use the [Query Task Status](/en/api-reference/task/status) endpoint to retrieve generation results. Video links are valid for 24 hours.

<RequestExample>
```bash cURL
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "minimax-hailuo-2.3",
    "action": "generate",
    "prompt": "一只白猫慵懒地躺在阳光下的窗台上",
    "duration": 6,
    "resolution": "1080p"
  }'
```

```python Python
import requests

response = requests.post(
    "https://www.qingbo.dev/v1/tasks",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "minimax-hailuo-2.3",
        "action": "generate",
        "prompt": "一只白猫慵懒地躺在阳光下的窗台上",
        "duration": 6,
        "resolution": "1080p"
    }
)

task_id = response.json()["task_id"]
print(f"Task submitted: {task_id}")
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":      "minimax-hailuo-2.3",
        "action":     "generate",
        "prompt":     "一只白猫慵懒地躺在阳光下的窗台上",
        "duration":   6,
        "resolution": "1080p",
    }

    body, _ := json.Marshal(payload)
    req, _ := http.NewRequest("POST", "https://www.qingbo.dev/v1/tasks", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    resp, _ := http.DefaultClient.Do(req)
    defer resp.Body.Close()
    result, _ := io.ReadAll(resp.Body)
    fmt.Println(string(result))
}
```

```java Java
import java.net.http.*;
import java.net.URI;

public class Main {
    public static void main(String[] args) throws Exception {
        String payload = """
        {
          "model": "minimax-hailuo-2.3",
          "action": "generate",
          "prompt": "一只白猫慵懒地躺在阳光下的窗台上",
          "duration": 6,
          "resolution": "1080p"
        }
        """;

        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://www.qingbo.dev/v1/tasks"))
            .header("Authorization", "Bearer YOUR_API_KEY")
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(payload))
            .build();

        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString());
        System.out.println(response.body());
    }
}
```

## Available Models

| Model ID | Default Resolution | Supported Duration | Description |
|----------|-------------------|-------------------|-------------|
| `minimax-hailuo-2.3` | 768P | 6s / 10s | Hailuo 2.3, supports character reference |
| `minimax-hailuo-2.3-fast` | 768P | 6s / 10s | Hailuo 2.3 fast version, image-to-video only |

## Common Parameters

<ParamField body="model" type="string" required>
  Model ID
</ParamField>

<ParamField body="action" type="string" default="generate">
  Operation type
</ParamField>

<ParamField body="prompt" type="string" required>
  Video description text
</ParamField>

<ParamField body="duration" type="integer" default="6">
  Video duration (seconds): `6`, `10`
</ParamField>

<ParamField body="resolution" type="string" default="768p">
  Output resolution, available values depend on the model (see model tabs)
</ParamField>

<ParamField body="image_urls" type="string[]">
  First frame image URL for image-to-video
</ParamField>

<ParamField body="last_frame" type="string">
  Last frame image URL, requires `image_urls`
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook callback URL
</ParamField>

<ParamField body="extra" type="object">
  Additional parameters passed through to Hailuo (see model tabs)
</ParamField>

## Model-Specific Parameters

<Tabs>
  <Tab title="Hailuo 2.3 / 2.3 Fast">

    <ParamField body="extra.fast_pretreatment" type="boolean">

    <ParamField body="extra.subject_reference" type="array">

    </ParamField>
  </Tab>


</Tabs>

## Generation Modes

| Mode | Parameters | Supported Models |
|------|-----------|-----------------|
| Text-to-video | `prompt` | All |
| Image-to-video | `prompt` + `image_urls` (1 image) | All |
| Character reference | `prompt` + `extra.subject_reference` | minimax-hailuo-2.3 |
