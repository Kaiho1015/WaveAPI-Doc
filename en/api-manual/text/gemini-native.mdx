## Overview

The Gemini Native Format API provides a native interface for interacting with Google Gemini models, supporting multimodal inputs and advanced features.

## Request Example

<CodeGroup>

```python Python
import google.generativeai as genai

genai.configure(
    api_key="YOUR_API_KEY",
    transport="rest",
    client_options={"api_endpoint": "https://www.qingbo.dev/v1"}
)

model = genai.GenerativeModel('gemini-2.0-flash-exp')

response = model.generate_content(
    "Explain the basic principles of quantum computing",
    generation_config={
        "temperature": 0.7,
        "max_output_tokens": 1000
    }
)

print(response.text)
```

</CodeGroup>

<Tip>
  **Long Context Processing:** Gemini 1.5 Pro supports up to 2 million tokens of context, ideal for processing long documents and video analysis.
</Tip>
