OpenAI-compatible unified API endpoint for GPT-4o, Claude, Gemini, Sora, VEO, and more. Migrate in under 5 minutes — just change your `base_url`, no code rewrite needed.

<CodeGroup>

```python Python
from openai import OpenAI

client = OpenAI(
    base_url="https://ai.kaiho.cc/v1",
    api_key="your-api-key"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(response.choices[0].message.content)
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'https://ai.kaiho.cc/v1',
  apiKey: 'your-api-key'
});

const response = await client.chat.completions.create({
  model: 'gpt-4o',
  messages: [{ role: 'user', content: 'Hello!' }]
});

console.log(response.choices[0].message.content);
```

```bash cURL
curl https://ai.kaiho.cc/v1/chat/completions \
  -H "Authorization: Bearer your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
```

## Quick Start

<Steps>
  <Step title="Get Your API Key">
    Sign up at the [Dashboard](https://ai.kaiho.cc/token) to get your API key
  </Step>
  <Step title="Replace Base URL">
    Point your OpenAI SDK base URL to `https://ai.kaiho.cc/v1`
  </Step>
  <Step title="Choose a Model and Go">
    Set the `model` parameter to your desired model and send your request
  </Step>
</Steps>
