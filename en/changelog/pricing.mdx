This page records **pricing changes** for QWave API. For each model's live unit price, see the [Models overview](/en/docs/models) and the console "Model Market".

<Note>
Pricing changes apply only to calls made **after** the change takes effect; historical calls are unaffected. The cost of each call is authoritative as returned in `cost` / `usage.cost`.
</Note>

**Pricing synced for some video models**

</Update>
