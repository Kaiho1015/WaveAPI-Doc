本页记录清波 API 的**价格调整**。各模型的实时单价请以 [模型总览](/cn/docs/models) 与控制台「模型市场」为准。

<Note>
价格调整仅对**调整生效后**的调用计费，历史调用不受影响。每次费用以接口返回的 `cost` / `usage.cost` 为准。
</Note>

<Update label="2026-06-17" description="价格调整">
**部分视频模型价格同步**

- 部分视频模型的单价已按官方价同步微调，具体以 [模型总览](/cn/docs/models) 与控制台「模型市场」展示的最新价格为准。
</Update>
