**价格调整**

- 部分视频模型按官方价同步微调，最新单价以 [模型总览](/cn/docs/models) 与控制台「模型市场」为准。

<Note>价格调整仅对调整生效后的调用计费，历史调用不受影响。</Note>
</Update>

<Update label="示例条目" description="模板（发布前删除）">
写一条更新时，复制本块，改 `label`（日期 `YYYY-MM-DD`）与 `description`（类型：模型上架 / 模型下架 / 价格调整 / 功能发布 / 通知），正文用 Markdown：

- **上架**：列模型名 + 模态，链到模型总览。
- **下架**：列模型名 + 下架日期 + 建议替代模型。
- **价格调整**：说明哪些模型、新价从哪查（不写死数字，避免文档与实际漂移）。
</Update>
