## 概述

TTS（Text-to-Speech）API 使用先进的语音合成技术，将文本转换为自然、流畅的人声，支持多种语言和音色。

## 请求参数

<ParamField body="model" type="string" required>
  TTS 模型：
  - `tts-1` - 标准质量，速度快
  - `tts-1-hd` - 高清质量，更自然
</ParamField>

<ParamField body="input" type="string" required>
  要转换为语音的文本，最多 4096 个字符。
</ParamField>

<ParamField body="voice" type="string" required>
  使用的声音：`alloy`, `echo`, `fable`, `onyx`, `nova`, `shimmer`
</ParamField>

<ParamField body="response_format" type="string" default="mp3">
  音频格式：
  - `mp3` - MP3 格式（默认）
  - `opus` - Opus 格式，适合流媒体
  - `aac` - AAC 格式
  - `flac` - FLAC 无损格式
  - `wav` - WAV 格式
  - `pcm` - 原始 PCM 格式
</ParamField>

<ParamField body="speed" type="number" default={1.0}>
  语速，范围 0.25 - 4.0。
  - 1.0 = 正常速度
  - 0.5 = 半速
  - 2.0 = 双速
</ParamField>

## 请求示例

<CodeGroup>

```bash cURL
curl https://ai.kaiho.cc/v1/audio/speech \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "tts-1-hd",
    "input": "欢迎使用 Kaihoxz 文本转语音服务，我们提供最自然的 AI 语音合成技术。",
    "voice": "nova",
    "speed": 1.0
  }' \
  --output speech.mp3
```

```python Python
import openai
from pathlib import Path

client = openai.OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://ai.kaiho.cc/v1"
)

# 生成语音
response = client.audio.speech.create(
    model="tts-1-hd",
    voice="nova",
    input="欢迎使用 Kaihoxz 文本转语音服务，我们提供最自然的 AI 语音合成技术。",
    speed=1.0
)

# 保存为文件
speech_file_path = Path("speech.mp3")
response.stream_to_file(speech_file_path)

print(f"语音已保存到: {speech_file_path}")
```

```javascript JavaScript
import fs from 'fs';
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'YOUR_API_KEY',
  baseURL: 'https://ai.kaiho.cc/v1'
});

async function generateSpeech() {
  const response = await client.audio.speech.create({
    model: 'tts-1-hd',
    voice: 'nova',
    input: '欢迎使用 Kaihoxz 文本转语音服务，我们提供最自然的 AI 语音合成技术。',
    speed: 1.0
  });

  const buffer = Buffer.from(await response.arrayBuffer());
  await fs.promises.writeFile('speech.mp3', buffer);
  
  console.log('语音已保存到: speech.mp3');
}

generateSpeech();
```

## 流式输出

对于实时播放，可以使用流式输出：

```python
from openai import OpenAI
import pyaudio

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://ai.kaiho.cc/v1"
)

# 初始化音频播放
p = pyaudio.PyAudio()
stream = p.open(format=pyaudio.paInt16, channels=1, rate=24000, output=True)

# 流式生成并播放
with client.audio.speech.with_streaming_response.create(
    model="tts-1",
    voice="nova",
    input="这是一段实时生成并播放的语音。",
    response_format="pcm"
) as response:
    for chunk in response.iter_bytes(chunk_size=1024):
        stream.write(chunk)

stream.stop_stream()
stream.close()
p.terminate()
```

## 文本优化建议

<AccordionGroup>
  <Accordion title="使用标点符号" icon="period">
    标点符号会影响语音的停顿和语气：
    - 句号（。）：较长停顿
    - 逗号（，）：短暂停顿
    - 问号（？）：上扬语调
    - 感叹号（！）：强调语气
  </Accordion>
  
  <Accordion title="SSML 标记（实验性）" icon="code">
    使用 SSML 标记控制语音细节：
    ```xml
    <speak>
      这是<emphasis>重要</emphasis>的内容。
      <break time="1s"/>
      这是一秒钟后的内容。
    </speak>
    ```
  </Accordion>
  
  <Accordion title="多语言混合" icon="language">
    TTS 支持自动识别并正确发音混合语言：
    ```
    欢迎使用 Kaihoxz API，这是一个 powerful 的 AI 平台。
    ```
  </Accordion>
  
  <Accordion title="数字和符号" icon="hashtag">
    数字和特殊符号会被智能转换：
    - 123 → "一百二十三"
    - $100 → "一百美元"
    - 50% → "百分之五十"
  </Accordion>
</AccordionGroup>

## 使用限制

- 单次请求最大 4096 个字符
- 长文本请分段处理
- 生成的音频自动返回，不需要轮询

<Warning>
  **使用规范：** 请确保生成的语音内容符合当地法律法规，不得用于欺诈、冒充他人等非法用途。
</Warning>

<Tip>
  **性能优化：** 对于实时应用，使用 `tts-1` 模型和 `opus` 格式可以获得最低延迟。
</Tip>
