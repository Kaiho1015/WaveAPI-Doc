## 概述

Whisper-1 是 OpenAI 的语音识别模型，能够准确地将音频转录为文本，支持多种语言和音频格式。

## 请求参数

<ParamField body="file" type="file" required>
  要转录的音频文件。
  
  支持的格式：mp3, mp4, mpeg, mpga, m4a, wav, webm
  
  最大文件大小：25 MB
</ParamField>

<ParamField body="model" type="string" required>
  模型 ID，使用 `whisper-1`。
</ParamField>

<ParamField body="language" type="string">
  音频的语言代码（ISO-639-1），如 `zh` 或 `en`。
  
  指定语言可以提高准确度和速度。
</ParamField>

<ParamField body="prompt" type="string">
  可选的文本，用于引导模型的风格或提供上下文。
  
  提示词应与音频语言匹配。
</ParamField>

<ParamField body="response_format" type="string" default="json">
  输出格式：
  - `json` - JSON 格式
  - `text` - 纯文本
  - `srt` - 字幕文件格式
  - `verbose_json` - 包含时间戳的详细 JSON
  - `vtt` - WebVTT 字幕格式
</ParamField>

<ParamField body="temperature" type="number" default={0}>
  采样温度，范围 0-1。
</ParamField>

<ParamField body="timestamp_granularities" type="array">
  时间戳粒度：
  - `segment` - 段级时间戳
  - `word` - 词级时间戳
</ParamField>

## 请求示例

<CodeGroup>

```bash cURL
curl https://ai.kaiho.cc/v1/audio/transcriptions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "file=@audio.mp3" \
  -F "model=whisper-1" \
  -F "language=zh" \
  -F "response_format=verbose_json" \
  -F "timestamp_granularities[]=word"
```

```python Python
import openai

client = openai.OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://ai.kaiho.cc/v1"
)

# 基础转录
with open("audio.mp3", "rb") as audio_file:
    transcription = client.audio.transcriptions.create(
        model="whisper-1",
        file=audio_file,
        language="zh"
    )

print(transcription.text)

# 带时间戳的转录
with open("audio.mp3", "rb") as audio_file:
    transcription = client.audio.transcriptions.create(
        model="whisper-1",
        file=audio_file,
        language="zh",
        response_format="verbose_json",
        timestamp_granularities=["word", "segment"]
    )

# 打印带时间戳的文本
for word in transcription.words:
    print(f"{word.start:.2f}s - {word.end:.2f}s: {word.word}")
```

```javascript JavaScript
import fs from 'fs';
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'YOUR_API_KEY',
  baseURL: 'https://ai.kaiho.cc/v1'
});

// 基础转录
const transcription = await client.audio.transcriptions.create({
  file: fs.createReadStream('audio.mp3'),
  model: 'whisper-1',
  language: 'zh'
});

console.log(transcription.text);

// 带时间戳的转录
const detailedTranscription = await client.audio.transcriptions.create({
  file: fs.createReadStream('audio.mp3'),
  model: 'whisper-1',
  language: 'zh',
  response_format: 'verbose_json',
  timestamp_granularities: ['word', 'segment']
});

// 打印带时间戳的文本
detailedTranscription.words.forEach(word => {
  console.log(`${word.start.toFixed(2)}s - ${word.end.toFixed(2)}s: ${word.word}`);
});
```

</CodeGroup>

## 最佳实践

<AccordionGroup>
  <Accordion title="音频质量" icon="microphone">
    - 使用清晰的音频源
    - 减少背景噪音
    - 采样率至少 16kHz
    - 单声道或立体声均可
  </Accordion>
  
  <Accordion title="文件大小" icon="file">
    - 单个文件最大 25MB
    - 超大文件请先分割
    - 使用压缩格式（如 mp3）节省空间
  </Accordion>
  
  <Accordion title="指定语言" icon="language">
    - 指定正确的语言代码提高准确度
    - 混合语言时可以不指定
    - 提示词应与音频语言一致
  </Accordion>
  
  <Accordion title="时间戳用途" icon="clock">
    - 生成字幕：使用 srt 或 vtt 格式
    - 精确定位：使用词级时间戳
    - 分段处理：使用段级时间戳
  </Accordion>
</AccordionGroup>
