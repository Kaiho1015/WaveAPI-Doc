## 概述

Sora2 是 OpenAI 的最新视频生成模型，能够根据文本描述创建逼真的视频内容，支持多种分辨率和时长。

<Warning>
  **异步任务：** 视频生成是异步操作，需要使用[任务管理 API](/cn/api-manual/task/get-status) 查询生成状态。
</Warning>

## 请求参数

<ParamField body="prompt" type="string" required>
  视频描述文本，最多 500 个字符。详细描述场景、动作、镜头运动等。
</ParamField>

<ParamField body="model" type="string" default="sora-2">
  使用的模型：
  - `sora-2` - 最新最强大
  - `sora-turbo` - 快速生成
  - `runway-gen3` - 高质量
  - `pika-1.5` - 创意风格
</ParamField>

<ParamField body="duration" type="integer" default={5}>
  视频时长（秒），范围 2-20 秒。
</ParamField>

<ParamField body="resolution" type="string" default="1280x720">
  视频分辨率：
  - `1920x1080` - Full HD
  - `1280x720` - HD
  - `640x360` - SD
  - `1024x1024` - 方形（社交媒体）
</ParamField>

<ParamField body="fps" type="integer" default={24}>
  帧率，可选值：24, 30, 60
</ParamField>

<ParamField body="style" type="string">
  视频风格：
  - `realistic` - 写实风格
  - `cinematic` - 电影风格
  - `animated` - 动画风格
  - `artistic` - 艺术风格
</ParamField>

<ParamField body="webhook_url" type="string">
  任务完成时的回调 URL（可选）。
</ParamField>

```python Python
import requests
import time

# 创建视频生成任务
response = requests.post(
    "https://ai.kaiho.cc/v1/videos/generations",
    headers={
        "Content-Type": "application/json",
        "Authorization": "Bearer YOUR_API_KEY"
    },
    json={
        "model": "sora-2",
        "prompt": "一只金毛犬在秋天的公园里追逐飘落的树叶，阳光透过树叶洒下，慢镜头，电影级画质",
        "duration": 10,
        "resolution": "1920x1080",
        "fps": 30,
        "style": "cinematic"
    }
)

task = response.json()
task_id = task["id"]
print(f"任务ID: {task_id}")

# 轮询任务状态
while True:
    status_response = requests.get(
        f"https://ai.kaiho.cc/v1/tasks/{task_id}",
        headers={"Authorization": "Bearer YOUR_API_KEY"}
    )
    
    status = status_response.json()
    
    if status["status"] == "completed":
        print(f"视频生成完成: {status['result']['video_url']}")
        break
    elif status["status"] == "failed":
        print(f"生成失败: {status['error']}")
        break
    
    print(f"进度: {status['progress']}%")
    time.sleep(5)
```

```javascript JavaScript
import axios from 'axios';

const apiKey = 'YOUR_API_KEY';
const baseURL = 'https://ai.kaiho.cc/v1';

async function generateVideo() {
  // 创建视频生成任务
  const response = await axios.post(
    `${baseURL}/videos/generations`,
    {
      model: 'sora-2',
      prompt: '一只金毛犬在秋天的公园里追逐飘落的树叶，阳光透过树叶洒下，慢镜头，电影级画质',
      duration: 10,
      resolution: '1920x1080',
      fps: 30,
      style: 'cinematic'
    },
    {
      headers: {
        'Content-Type': 'application/json',
        'Authorization': `Bearer ${apiKey}`
      }
    }
  );

  const taskId = response.data.id;
  console.log('任务ID:', taskId);

  // 轮询任务状态
  while (true) {
    const statusResponse = await axios.get(
      `${baseURL}/tasks/${taskId}`,
      {
        headers: { 'Authorization': `Bearer ${apiKey}` }
      }
    );

    const status = statusResponse.data;

    if (status.status === 'completed') {
      console.log('视频生成完成:', status.result.video_url);
      break;
    } else if (status.status === 'failed') {
      console.log('生成失败:', status.error);
      break;
    }

    console.log('进度:', status.progress + '%');
    await new Promise(resolve => setTimeout(resolve, 5000));
  }
}

generateVideo();
```

## 使用 Webhook

配置 webhook 以自动接收完成通知：

```python
response = requests.post(
    "https://ai.kaiho.cc/v1/videos/generations",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "sora-2",
        "prompt": "视频描述...",
        "webhook_url": "https://your-server.com/webhook",
        "webhook_secret": "your_secret_key"  # 用于验证请求
    }
)
```

Webhook 回调格式：

```json
{
  "event": "video.completed",
  "task_id": "task_abc123",
  "status": "completed",
  "result": {
    "video_url": "https://storage.kaiho.cc/videos/video-123.mp4",
    "duration": 10
  }
}
```

## 生成时间估算

| 时长 | 分辨率 | 预计时间 |
|------|--------|---------|
| 5秒 | 720p | 1-2 分钟 |
| 5秒 | 1080p | 2-3 分钟 |
| 10秒 | 720p | 3-5 分钟 |
| 10秒 | 1080p | 5-8 分钟 |
| 20秒 | 1080p | 10-15 分钟 |

<Tip>
  **节省时间：** 在测试提示词时，先使用较低分辨率和较短时长，确定效果后再生成高质量版本。
</Tip>
