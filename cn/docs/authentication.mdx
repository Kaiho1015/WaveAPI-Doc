## 认证方式

所有 WaveAPI 请求使用 Bearer Token 认证：

```bash
Authorization: Bearer YOUR_API_KEY
```

## 获取 API Key

<Steps>
  <Step title="注册账户">
    访问 [Dashboard](https://qingbo.dev/dashboard/keys) 注册
  </Step>
  <Step title="创建密钥">
    在密钥管理页面点击"创建新密钥"
  </Step>
  <Step title="保存密钥">
    复制并安全保存，密钥只显示一次
  </Step>
</Steps>

## Base URL

```
https://www.qingbo.dev/v1
```

所有端点都基于此 URL。如果你已经在使用 OpenAI SDK，只需替换 `base_url` 即可。

<Warning>
请勿在前端代码或公开仓库中暴露 API Key。
</Warning>
