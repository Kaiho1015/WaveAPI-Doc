异步处理模式 — 提交请求后返回任务 ID，通过 [查询任务状态](/cn/api-reference/task/status) 接口获取生成结果。

<Warning>
Sora 2 系列模型将于 2026 年 9 月 24 日停止服务。
</Warning>

<RequestExample>
```bash cURL
curl -X POST https://qingbo.dev/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "sora-2-pro",
    "action": "generate",
    "prompt": "A cat playing piano in a jazz club, cinematic lighting",
    "aspect_ratio": "16:9",
    "duration": 8,
    "resolution": "1080p"
  }'
```

```python Python
import requests

response = requests.post(
    "https://qingbo.dev/v1/tasks",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "sora-2-pro",
        "action": "generate",
        "prompt": "A cat playing piano in a jazz club, cinematic lighting",
        "aspect_ratio": "16:9",
        "duration": 8,
        "resolution": "1080p"
    }
)

task_id = response.json()["task_id"]
print(f"任务已提交: {task_id}")
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":        "sora-2-pro",
        "action":       "generate",
        "prompt":       "A cat playing piano in a jazz club, cinematic lighting",
        "aspect_ratio": "16:9",
        "duration":     8,
        "resolution":   "1080p",
    }

    body, _ := json.Marshal(payload)
    req, _ := http.NewRequest("POST", "https://qingbo.dev/v1/tasks", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    resp, _ := http.DefaultClient.Do(req)
    defer resp.Body.Close()
    result, _ := io.ReadAll(resp.Body)
    fmt.Println(string(result))
}
```

```java Java
import java.net.http.*;
import java.net.URI;

public class Main {
    public static void main(String[] args) throws Exception {
        String payload = """
        {
          "model": "sora-2-pro",
          "action": "generate",
          "prompt": "A cat playing piano in a jazz club, cinematic lighting",
          "aspect_ratio": "16:9",
          "duration": 8,

        """;

        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://qingbo.dev/v1/tasks"))
            .header("Authorization", "Bearer YOUR_API_KEY")
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(payload))
            .build();

        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString());
        System.out.println(response.body());
    }
}
```

## 可用模型

| 模型 ID | resolution 支持 | 说明 |
|---------|----------------|------|
| `sora-2-pro` | `720p`、`1080p` | 高画质，支持 1080p |
| `sora-2` | `720p` | 速度更快 |

## 统一参数

<ParamField body="model" type="string" required>
  模型 ID
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作类型：`generate`、`remix`、`extend`
</ParamField>

<ParamField body="prompt" type="string" required>
  视频描述文本
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook 回调地址
</ParamField>

<ParamField body="extra" type="object">
  透传给 Sora 的额外参数
</ParamField>

## 各操作类型参数

<Tabs>
  <Tab title="生成 (generate)">
    文生视频或图生视频。

    <ParamField body="aspect_ratio" type="string" default="9:16">
      宽高比：`16:9`、`9:16`
    </ParamField>

    <ParamField body="duration" type="integer" default="4">
      视频时长（秒）：`4`、`8`、`12`、`16`、`20`
    </ParamField>

    <ParamField body="resolution" type="string" default="720p">
      输出分辨率。`1080p` 仅 `sora-2-pro` 支持
    </ParamField>

    <ParamField body="image_urls" type="string[]">
      参考图片 URL，用于首帧引导。图片宽高比需与 `aspect_ratio` 一致
    </ParamField>

    <ParamField body="extra.characters" type="array">
      角色资产数组，最多 2 个。角色名称必须在 prompt 中被提及

      <Expandable title="属性">
        <ParamField body="name" type="string" required>
          角色名称
        </ParamField>
        <ParamField body="asset_id" type="string" required>
          角色资产 ID
        </ParamField>
      </Expandable>
    </ParamField>
  </Tab>

  <Tab title="Remix">
    基于已有视频进行修改。需提供原始视频的 task_id。

    <ParamField body="extra.video_id" type="string" required>
      要 remix 的原始视频 ID
    </ParamField>

    <ParamField body="extra.strength" type="string" default="Mild">
      变化程度：`Subtle`、`Mild`、`Strong`，或数字 `1`-`8`
    </ParamField>

    <Note>
    Remix 即将被 Edits 替代。
    </Note>
  </Tab>

  <Tab title="续写 (extend)">
    延续已有视频，保持运动、镜头和场景的连续性。

    <ParamField body="duration" type="integer" required>
      续写片段时长（秒）：`4`、`8`、`12`、`16`、`20`
    </ParamField>

    <ParamField body="extra.video_id" type="string" required>
      要续写的原始视频 ID（必须是已完成的 Sora 视频）
    </ParamField>

    **限制：**
    - 每条视频链最多续写 6 次（总计 120 秒）
    - 不支持 `image_urls` 和 `characters`
  </Tab>
</Tabs>
