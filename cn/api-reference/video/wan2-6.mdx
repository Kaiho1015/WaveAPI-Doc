<Note>
  该接口为异步执行，请求成功后会返回任务 ID。生成通常需要 1-3 分钟，生成结果在服务器保留 24 小时。
</Note>

```go Go theme={null}
package main

import (
	"bytes"
	"encoding/json"
	"fmt"
	"net/http"
)

func main() {
	url := "https://qingbo.dev/v1/videos/generations"
	payload := map[string]interface{}{
		"model": "wan2.6",
		"prompt": "黄昏时分，渔船归港的宁静海面",
		"resolution": "720p",
		"duration": 5,
	}
	jsonPayload, _ := json.Marshal(payload)

	req, _ := http.NewRequest("POST", url, bytes.NewBuffer(jsonPayload))
	req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
	req.Header.Set("Content-Type", "application/json")

	client := &http.Client{}
	resp, _ := client.Do(req)
	defer resp.Body.Close()

	var result map[string]interface{}
	json.NewDecoder(resp.Body).Decode(&result)
	fmt.Println(result)
}
```

## 认证方式

在请求头中加入 `Authorization: Bearer YOUR_API_KEY`。您可以从 [API Key 页面](https://qingbo.dev/dashboard/keys) 获取您的密钥。

## 请求参数

<ParamField body="model" type="string" required>
  模型 ID：`wan2.6`
</ParamField>

<ParamField body="prompt" type="string">
  视频生成的提示词。支持提示词自动扩展以优化效果。
</ParamField>

<ParamField body="image_urls" type="array">
  参考图片链接，用于图生视频任务。
</ParamField>

<ParamField body="resolution" type="string">
  分辨率。支持 `720p`, `1080p`。
</ParamField>

<ParamField body="duration" type="integer">
  视频时长。可选值：`5`, `10`, `15`。
</ParamField>

## 响应参数

<ResponseField name="code" type="string">
  响应状态码 * `success` - 请求成功，任务已提交
</ResponseField>

<ResponseField name="data" type="string">
  任务 ID，格式为 `task-wave{timestamp}{userId}{random}`，用于通过 [查询任务状态](/cn/api-reference/task/get-status) 接口获取生成结果
</ResponseField>

## 使用示例

### 文生视频
```json theme={null}
{
  "model": "wan2.6",
  "prompt": "黄昏时分，渔船归港的宁静海面",
  "resolution": "1080p",
  "duration": 10
}
```

### 图生视频
```json theme={null}
{
  "model": "wan2.6",
  "image_urls": ["https://example.com/scene.jpg"],
  "prompt": "镜头缓缓推进，展现海浪拍打礁石的细节"
}
```

## 注意事项

- 支持音频同步生成。
- 建议开启提示词自动扩展功能以获得更精细的画面。
- 生成进度请参考 [查询任务状态](/cn/api-reference/task/get-status)。
