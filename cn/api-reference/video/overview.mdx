## 概述

WaveAPI 支持多种视频生成模型，统一通过异步任务接口 `/v1/tasks` 调用。视频生成通常耗时较长，请使用轮询或 Webhook 回调获取结果。

## 请求端点

```
POST https://www.qingbo.dev/v1/tasks
```

## 通用请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `model` | string | 是 | 模型名称 |
| `action` | string | 否 | 默认 `"generate"` |
| `prompt` | string | 是 | 视频描述文本 |
| `duration` | integer | 否 | 视频时长（秒） |
| `aspect_ratio` | string | 否 | 宽高比，如 `"16:9"` |
| `resolution` | string | 否 | 分辨率，如 `"1080p"` |
| `first_frame` | string | 否 | 首帧图片 URL |
| `last_frame` | string | 否 | 尾帧图片 URL |
| `image_urls` | array | 否 | 参考图片 URL 数组 |
| `callback_url` | string | 否 | 完成后的回调地址 |
| `extra` | object | 否 | 模型特定的透传参数 |

## 响应格式

```json
{
  "task_id": "task-wave1775285160b950328499",
  "model": "veo-3.1-fast",
  "action": "generate",
  "status": "queued",
  "created_at": 1775285160040,
  "progress": 0
}
```

## 生成模式

| 模式 | 参数 | 说明 |
|------|------|------|
| 文生视频 | `prompt` | 纯文本描述生成视频 |
| 图生视频 | `prompt` + `first_frame` 或 `image_urls` | 基于图片生成视频 |
| 首尾帧 | `first_frame` + `last_frame` | 指定首尾画面生成过渡视频 |

<Note>
并非所有模型都支持全部模式，请查看各模型文档了解具体支持情况。
</Note>
