<Note>
本接口采用异步模式。提交成功后，接口会返回一个任务 ID。您需要使用该 ID 调用 [查询任务状态](/cn/api-reference/task/get-status) 接口获取最终生成结果。视频链接有效期为 24 小时，请及时下载。
</Note>

## 认证方式

在请求头中加入 `Authorization: Bearer YOUR_API_KEY` 进行身份验证。您可以在 [令牌管理](https://qingbo.dev/dashboard/keys) 页面获取您的 API 密钥。

## 请求参数

<ParamField body="model" type="string" required>
可选值：`veo3.1-fast`（快速生成）, `veo3.1-quality`（高画质）
</ParamField>

<ParamField body="prompt" type="string" required>
视频生成的文本描述。例如："海豚在蔚蓝的海洋中跃出水面"
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
视频宽高比，支持 `16:9` 和 `9:16`。
</ParamField>

<ParamField body="duration" type="number" default="8">
固定为 8 秒。
</ParamField>

<ParamField body="image_urls" type="array">
支持最多 3 张图片 URL。用于首尾帧模式或参考图模式。
</ParamField>

<ParamField body="generation_type" type="string" default="frame">
生成模式：`frame`（首尾帧模式）, `reference`（参考图模式）。注意：`veo3.1-quality` 不支持 `reference` 模式。
</ParamField>

<ParamField body="resolution" type="string" default="720p">
分辨率，可选值：`720p`, `1080p`, `4k`。
</ParamField>

<ParamField body="enable_gif" type="boolean" default="false">
是否生成 GIF 动图。注意：开启后不能选择 `1080p` 或 `4k` 分辨率。
</ParamField>

## 响应参数

<ResponseField name="code" type="string">
响应状态码 * `success` - 请求成功，任务已提交
</ResponseField>

<ResponseField name="data" type="string">
任务 ID，格式为 `task-wave{timestamp}{userId}{random}`，用于通过 [查询任务状态](/cn/api-reference/task/get-status) 接口获取生成结果
</ResponseField>

## 使用示例

### 文生视频 (高画质)

```json theme={null}
{
  "model": "veo3.1-quality",
  "prompt": "海豚在蔚蓝的海洋中跃出水面",
  "aspect_ratio": "16:9",
  "resolution": "1080p"
}
```

### 首尾帧模式 (快速)

```json theme={null}
{
  "model": "veo3.1-fast",
  "prompt": "一个人在草地上行走",
  "image_urls": ["https://example.com/start.jpg", "https://example.com/end.jpg"],
  "generation_type": "frame"
}
```

## 注意事项

- 接口为异步处理，返回任务 ID 后请通过查询接口获取结果。
- 视频链接有效期为 24 小时，过期后将无法访问，请及时转存。
- `veo3.1-quality` 模式下暂不支持参考图模式。
- `enable_gif` 与高分辨率（1080p/4k）不可同时使用。
