Google Veo 3.1 视频生成模型,提供 **官方直连** 与 **逆向接入** 两种渠道,共 **4 个子档**。

- **官方档**(Vertex AI 直连):按 **秒** 计费,支持 4 / 6 / 8 秒时长,720P / 1080P / 4K 三种分辨率。
- **逆向档**:按 **次** 计费(单次固定价),仅支持 **8 秒** 固定时长。

| 档位 | 渠道 | 计费方式 | 定位 |
|---|---|---|---|
| `veo3.1-quality-official` | 官方直连 | 按秒 | 旗舰档,最终交付质量 |
| `veo3.1-fast-official` | 官方直连 | 按秒 | 加速档,适合迭代试稿 |
| `veo3.1-quality` | 逆向接入 | 按次 | 逆向旗舰子型,固定 8 秒 |
| `veo3.1-lite` | 逆向接入 | 按次 | 逆向轻量子型,固定 8 秒 |

两种渠道均原生支持音频、首尾帧关键帧、Remix 续写。

## 定价

<Note>
**4K 分辨率不支持静音**,必须设 `generate_audio=true`(720p / 1080p 可静音也可有声)。下表中 `720p_sound` / `1080p_sound` 为带音频加价档,`4k_sound` 是 4K 唯一档。
</Note>

### 官方档(按秒)

720P 与 1080P 同价(Google 官方阶梯定价),开启 `generate_audio=true` 后单价**翻倍**。

#### `veo3.1-fast-official`

| 规格 | 单价(每秒) | 8 秒视频 |
|---|---|---|
| 720P 静音 | `$0.08` | `$0.64` |
| 1080P 静音 | `$0.08` | `$0.64` |
| 720P + 音频 | `$0.12` | `$0.96` |
| 1080P + 音频 | `$0.12` | `$0.96` |
| 4K + 音频 | `$0.28` | `$2.24` |

#### `veo3.1-quality-official`

| 规格 | 单价(每秒) | 8 秒视频 |
|---|---|---|
| 720P 静音 | `$0.16` | `$1.28` |
| 1080P 静音 | `$0.16` | `$1.28` |
| 720P + 音频 | `$0.32` | `$2.56` |
| 1080P + 音频 | `$0.32` | `$2.56` |
| 4K + 音频 | `$0.48` | `$3.84` |

### 逆向档(按次)

逆向档为**单次固定价**,与时长、分辨率、是否带音频无关(仅支持 8 秒)。具体价格以 admin 面板 / 控制台公示为准。

| 模型 ID | 计费方式 | 时长 | 单次价 |
|---|---|---|---|
| `veo3.1-quality` | 按次 | 8 秒(固定) | TBD(参考控制台) |
| `veo3.1-lite` | 按次 | 8 秒(固定) | TBD(参考控制台) |

## 调用示例

<CodeGroup>

```bash 文生视频(最简)
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "veo3.1-fast-official",
    "prompt": "雨夜霓虹街道,一辆红色跑车驶过水洼,电影感慢镜头",
    "duration": 8,
    "resolution": "1080p",
    "aspect_ratio": "16:9"
  }'
```

```bash 图生视频(I2V 首帧)
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "veo3.1-quality-official",
    "action": "image2video",
    "prompt": "人物缓缓抬头微笑,头发被风吹起",
    "first_frame_image": "https://cdn.example.com/portrait.jpg",
    "duration": 6,
    "resolution": "1080p",
    "resize_mode": "pad"
  }'
```

```bash 带原生音频(generate_audio=true)
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "veo3.1-quality-official",
    "prompt": "海浪拍打礁石,海鸥鸣叫,远处轮船汽笛声",
    "duration": 8,
    "resolution": "1080p",
    "aspect_ratio": "16:9",
    "generate_audio": true
  }'
```

```bash Remix 续写
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "veo3.1-fast-official",
    "action": "remix",
    "prompt": "镜头继续推近,人物转身离去",
    "first_frame_image": "https://cdn.example.com/last-frame.jpg",
    "duration": 4,
    "resolution": "1080p",
    "raw": true
  }'
```

```bash 逆向档(8 秒固定)
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "veo3.1-quality",
    "prompt": "夕阳下的海滨小镇,鸽群飞过钟楼",
    "duration": 8,
    "aspect_ratio": "16:9"
  }'
```

## 通用参数

<ParamField body="model" type="string" required>
  模型 ID,可选 `veo3.1-quality-official` / `veo3.1-fast-official` / `veo3.1-quality` / `veo3.1-lite`
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作类型,可选值:
  - `generate` — 文生视频(T2V)
  - `image2video` — 图生视频(I2V),需配合 `first_frame_image`
  - `first_last_frame` — 首尾帧关键帧,需同时传 `first_frame_image` 与 `last_frame_image`
  - `remix` — 基于已有片段续写
</ParamField>

<ParamField body="prompt" type="string" required>
  视频描述文本,所有 action 必填
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
  画面宽高比,可选值:
  - `16:9` — 横版宽屏
  - `9:16` — 竖版长屏
</ParamField>

<ParamField body="resolution" type="string" default="1080p">
  输出分辨率,可选值:
  - `720p`
  - `1080p`
  - `4k` — **仅官方档支持**,且**必须** `generate_audio=true`(4K 不支持静音)
</ParamField>

<ParamField body="duration" type="integer" default="8">
  视频时长(秒):
  - **官方档**:可选 `4` / `6` / `8`
  - **逆向档**:仅 `8`(固定)
</ParamField>

<ParamField body="image_urls" type="string[]">
  通用图像引用数组(部分场景下与 `first_frame_image` 等价)
</ParamField>

<ParamField body="first_frame_image" type="string">
  视频首帧图 URL。`image2video` / `first_last_frame` / `remix` 模式必填
</ParamField>

<ParamField body="last_frame_image" type="string">
  视频尾帧图 URL,**仅 `first_last_frame` 模式必填**
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook 回调地址,任务终态时调用。详见 [回调机制](/cn/docs/sync-async#异步任务-webhook)
</ParamField>

<ParamField body="callback_events" type="string[]">
  订阅的回调事件类型,默认订阅终态(`completed` / `failed`)
</ParamField>

## 资源限制

| 项目 | 限制 |
|---|---|
| 参考图片 | JPG / PNG / WEBP,单张 ≤ 20MB,建议 ≥ 720P |
| 输出视频 | MP4,链接 24 小时有效 |
| 时长 | 官方档 4 / 6 / 8 秒;逆向档仅 8 秒(固定) |
| 分辨率 | 720P / 1080P 任意;**4K 仅官方档**且必须 `generate_audio=true` |
| 音频 | 原生生成,无需额外 TTS;4K 必须有音频,720P / 1080P 音频可选 |
| 人物 | 受 `person_generation` 策略约束,违规内容会被上游拦截 |
