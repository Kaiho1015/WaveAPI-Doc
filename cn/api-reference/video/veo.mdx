Google Vertex AI 上的 Veo 3.1 视频生成模型,**官方直连**。

提供 **Fast** 与 **Quality** 两档,参数完全一致,差别在生成质量与单价。两档均原生支持音频、首尾帧关键帧、Remix 续写,以及 720P / 1080P / 4K 三种分辨率。

| 档位 | 定位 | 速度 | 单价基准(720P 静音) |
|---|---|---|---|
| `veo3.1-fast-official` | 加速档,适合迭代试稿 | 更快 | `$0.08` / 秒 |
| `veo3.1-quality-official` | 旗舰档,最终交付质量 | 较慢 | `$0.16` / 秒 |

按 **秒** 计费,支持 4 / 6 / 8 秒三档时长。

## 定价

**注意** — 720P 与 1080P 同价(Google 官方阶梯定价),但开启 `generate_audio=true` 后单价**翻倍**;4K 仅在含音频档位提供。

## 调用示例

<CodeGroup>

```bash 文生视频(最简)
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "veo3.1-fast-official",
    "prompt": "雨夜霓虹街道,一辆红色跑车驶过水洼,电影感慢镜头",
    "duration": 8,
    "resolution": "1080p",
    "aspect_ratio": "16:9"
  }'
```

```bash 图生视频(I2V 首帧)
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "veo3.1-quality-official",
    "action": "image2video",
    "prompt": "人物缓缓抬头微笑,头发被风吹起",
    "first_frame_image": "https://cdn.example.com/portrait.jpg",
    "duration": 6,
    "resolution": "1080p",
    "resize_mode": "pad"
  }'
```

```bash 带原生音频(generate_audio=true)
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "veo3.1-quality-official",
    "prompt": "海浪拍打礁石,海鸥鸣叫,远处轮船汽笛声",
    "duration": 8,
    "resolution": "1080p",
    "aspect_ratio": "16:9",
    "generate_audio": true
  }'
```

```bash Remix 续写
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "veo3.1-fast-official",
    "action": "remix",
    "prompt": "镜头继续推近,人物转身离去",
    "first_frame_image": "https://cdn.example.com/last-frame.jpg",
    "duration": 4,
    "resolution": "1080p",
    "raw": true
  }'
```

</CodeGroup>

## 通用参数

<ParamField body="model" type="string" required>
  模型 ID,可选 `veo3.1-fast-official` 或 `veo3.1-quality-official`
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作类型,可选值:
  - `generate` — 文生视频(T2V)
  - `image2video` — 图生视频(I2V),需配合 `first_frame_image`
  - `first_last_frame` — 首尾帧关键帧,需同时传 `first_frame_image` 与 `last_frame_image`
  - `remix` — 基于已有片段续写
</ParamField>

<ParamField body="prompt" type="string" required>
  视频描述文本,所有 action 必填
</ParamField>

<ParamField body="n" type="integer" default="1">
  生成数量,等价于 `sample_count`(取较大者)
</ParamField>

<ParamField body="seed" type="integer" default="-1">
  随机种子,`-1` 为随机;固定值可复现相似结果
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
  画面宽高比,可选值:
  - `16:9` — 横版宽屏
  - `9:16` — 竖版长屏
</ParamField>

<ParamField body="resolution" type="string" default="1080p">
  输出分辨率,可选值:
  - `720p`
  - `1080p`
  - `4k` — 仅在 `generate_audio=true` 时可用
</ParamField>

<ParamField body="duration" type="integer" default="8">
  视频时长(秒),可选 `4` / `6` / `8`
</ParamField>

<ParamField body="image_urls" type="string[]">
  通用图像引用数组(部分场景下与 `first_frame_image` 等价)
</ParamField>

<ParamField body="first_frame_image" type="string">
  视频首帧图 URL。`image2video` / `first_last_frame` / `remix` 模式必填
</ParamField>

<ParamField body="last_frame_image" type="string">
  视频尾帧图 URL,**仅 `first_last_frame` 模式必填**
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook 回调地址,任务终态时调用。详见 [回调机制](/cn/docs/sync-async#异步任务-webhook)
</ParamField>

<ParamField body="callback_events" type="string[]">
  订阅的回调事件类型,默认订阅终态(`completed` / `failed`)
</ParamField>

## 模型特定参数

<Note>
**Fast 与 Quality 两档参数完全一致**,仅生成质量与单价不同,以下参数对两个 Tab 都适用。
</Note>

<Tabs>
  <Tab title="veo3.1-fast-official">
    <ParamField body="generate_audio" type="boolean" default="false">
      是否生成音频轨道。开启后单价翻倍,4K 必须开启此项
    </ParamField>

    <ParamField body="sample_count" type="integer" default="1">
      生成样本数,范围 1-4
    </ParamField>

    <ParamField body="resize_mode" type="string" default="pad">
      图生视频时图片缩放策略,可选值:
      - `pad` — 边缘填充,保留完整画面
      - `crop` — 居中裁剪,贴合输出比例
    </ParamField>

    <ParamField body="enhance_prompt" type="boolean" default="true">
      上游提示词增强,**只能为 `true`**(Vertex AI 强制开启)
    </ParamField>

    <ParamField body="person_generation" type="string" default="allow_adult">
      人物生成策略,可选值:
      - `allow_adult` — 允许生成成年人物
      - `disallow` — 禁止生成人物
    </ParamField>

    <ParamField body="raw" type="boolean" default="false">
      Remix 模式专用,`true` 时仅返回延长部分,不拼接原片段
    </ParamField>
  </Tab>

  <Tab title="veo3.1-quality-official">
    <ParamField body="generate_audio" type="boolean" default="false">
      是否生成音频轨道。开启后单价翻倍,4K 必须开启此项
    </ParamField>

    <ParamField body="sample_count" type="integer" default="1">
      生成样本数,范围 1-4
    </ParamField>

    <ParamField body="resize_mode" type="string" default="pad">
      图生视频时图片缩放策略,可选值:
      - `pad` — 边缘填充,保留完整画面
      - `crop` — 居中裁剪,贴合输出比例
    </ParamField>

    <ParamField body="enhance_prompt" type="boolean" default="true">
      上游提示词增强,**只能为 `true`**(Vertex AI 强制开启)
    </ParamField>

    <ParamField body="person_generation" type="string" default="allow_adult">
      人物生成策略,可选值:
      - `allow_adult` — 允许生成成年人物
      - `disallow` — 禁止生成人物
    </ParamField>

    <ParamField body="raw" type="boolean" default="false">
      Remix 模式专用,`true` 时仅返回延长部分,不拼接原片段
    </ParamField>
  </Tab>
</Tabs>

## 资源限制

| 项目 | 限制 |
|---|---|
| 参考图片 | JPG / PNG / WEBP,单张 ≤ 20MB,建议 ≥ 720P |
| 输出视频 | MP4,链接 24 小时有效 |
| 时长 | 4 / 6 / 8 秒(整秒固定档) |
| 分辨率 | 720P / 1080P 任意,4K 仅 `generate_audio=true` |
| 音频 | 原生生成,无需额外 TTS |
| 人物 | 受 `person_generation` 策略约束,违规内容会被 Vertex 端拦截 |
