异步处理模式 — 提交请求后返回任务 ID，通过 [查询任务状态](/cn/api-reference/task/status) 接口获取生成结果。视频链接有效期 24 小时，请及时下载。

```java Java
import java.net.http.*;
import java.net.URI;

public class Main {
    public static void main(String[] args) throws Exception {
        String payload = """
        {
          "model": "seedance-2.0",
          "action": "generate",
          "prompt": "一只柴犬穿宇航服在月球上行走，电影感光影",

          "resolution": "1080p",

          }
        }
        """;

        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://www.qingbo.dev/v1/tasks"))
            .header("Authorization", "Bearer YOUR_API_KEY")
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(payload))
            .build();

        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString());
        System.out.println(response.body());
    }
}
```

## 可用模型

| 模型 ID | 说明 |
|---------|------|
| `seedance-2.0` | Seedance 2.0，支持原生音频、首尾帧、多模态参考、最高 2K |
| `seedance-2.0-fast` | Seedance 2.0 快速版 |
| `seedance-1.5-pro` | Seedance 1.5 专业版 |
| `seedance-1.0-pro-fast` | Seedance 1.0 快速版 |
| `seedance-1.0-pro-quality` | Seedance 1.0 高质量版 |

## 统一参数

<ParamField body="model" type="string" required>
  模型 ID
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作类型
</ParamField>

<ParamField body="prompt" type="string" required>
  视频描述文本，支持中英文。2.0 支持在 prompt 中用 `@image1`、`@video1`、`@audio1` 引用参考素材
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
  宽高比：`16:9`、`9:16`、`4:3`、`3:4`、`21:9`、`1:1`
</ParamField>

<ParamField body="duration" type="integer" default="5">
  视频时长（秒），可选范围取决于模型，见各模型 Tab
</ParamField>

<ParamField body="resolution" type="string">
  输出分辨率，可选值取决于模型，见各模型 Tab
</ParamField>

<ParamField body="image_urls" type="string[]">
  参考图片 URL 数组，数量限制取决于模型版本
</ParamField>

<ParamField body="first_frame" type="string">
  首帧图片 URL
</ParamField>

<ParamField body="last_frame" type="string">
  尾帧图片 URL（需配合 `first_frame`）
</ParamField>

<ParamField body="seed" type="integer" default="-1">
  随机种子，范围 [-1, 2^32-1]。`-1` 为随机。相同种子 + 相同请求可获得相似结果
</ParamField>

<ParamField body="watermark" type="boolean" default="false">
  是否添加水印
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook 回调地址
</ParamField>

<ParamField body="extra" type="object">
  透传给上游的额外参数，详见各模型 Tab
</ParamField>

## 模型特定参数

<Tabs>
  <Tab title="Seedance 2.0">
    **duration 范围：** 4-15 秒

    **resolution 可选值：** `480p`、`720p`、`1080p`、`2K`

    **size 额外支持：** `adaptive`（自适应，匹配输入素材宽高比）

    **输入限制：** 图片最多 9 张（≤ 30MB）、视频最多 3 个（≤ 50MB，总时长 ≤ 15s）、音频最多 3 个（MP3/WAV，≤ 15MB，总时长 ≤ 15s）

    <ParamField body="extra.generate_audio" type="boolean" default="false">
      原生音视频联合生成，支持对白口型同步（8+ 语言）
    </ParamField>

    <ParamField body="extra.return_last_frame" type="boolean" default="false">
      返回末帧图片 URL，便于视频片段衔接续拍
    </ParamField>

    <ParamField body="extra.tools" type="array">
      模型工具，支持联网搜索

      ```json
      "extra": { "tools": [{"type": "web_search"}] }
      ```
    </ParamField>

    <ParamField body="extra.service_tier" type="string">
      服务等级，控制延迟与成本的平衡
    </ParamField>

    <Note>
    2.0 使用首尾帧模式（`first_frame` + `last_frame`）时，不能同时传 `video_urls` 和 `audio_urls`。
    </Note>
  </Tab>

  <Tab title="Seedance 2.0 Fast">
    **duration 范围：** 4-15 秒

    **resolution 可选值：** `480p`、`720p`、`1080p`、`2K`

    参数与 Seedance 2.0 相同，处理速度更快。

    <ParamField body="extra.generate_audio" type="boolean" default="false">
      原生音视频联合生成
    </ParamField>

    <ParamField body="extra.return_last_frame" type="boolean" default="false">
      返回末帧图片
    </ParamField>
  </Tab>

  <Tab title="Seedance 1.5 Pro">
    **duration 范围：** 4-12 秒

    **resolution 可选值：** `480p`、`720p`、`1080p`

    **输入限制：** 图片最多 2 张（1 张首帧 + 1 张尾帧）

    <ParamField body="extra.audio" type="boolean" default="false">
      是否生成伴随音频
    </ParamField>

    <ParamField body="extra.camerafixed" type="boolean" default="false">
      固定机位模式，开启后摄像机位置不变
    </ParamField>
  </Tab>

  <Tab title="Seedance 1.0 Pro">
    **两个子版本：** `seedance-1.0-pro-fast`（快速）和 `seedance-1.0-pro-quality`（高画质）

    **duration 范围：** 2-12 秒

    **resolution 可选值：** `480p`、`720p`、`1080p`

    **输入限制：** 图片最多 1 张

    <Note>
    1.0 Quality 版本支持 `last_frame`（尾帧）和 `reference`（风格参考）角色。使用 reference 角色时不支持 1080p 分辨率。
    </Note>
  </Tab>

## 生成模式

| 模式 | 参数 | 支持版本 |
|------|------|---------|
| 文生视频 | `prompt` | 全部 |
| 图生视频 | `prompt` + `image_urls`(1张) 或 `first_frame` | 全部 |
| 首尾帧 | `first_frame` + `last_frame` | 1.0-quality / 1.5 / 2.0 |
| 多模态参考 | `image_urls` + `video_urls` + `audio_urls` | 仅 2.0 |
