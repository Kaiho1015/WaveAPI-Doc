字节跳动豆包(Doubao)的新一代有声视频生成模型 **`doubao-seedance-1.5-pro`** — 音视频联合生成,支持多语言对白 + 口型同步,可用首帧 / 尾帧约束镜头起止画面。

<Note>
这是 **视频** 模型 Seedance(动态视频),请勿与豆包的图像模型 **Seedream**(静态图像)混淆,两者是不同产品线。
</Note>

按 **分辨率 × 时长** 计费,**每秒单价**,支持 480P / 720P / 1080P 三档分辨率。

## 定价

按秒计费,单位 `$/秒`(售价即结算 USD 价)。

| 分辨率 | 单价 | 5 秒视频 |
|---|---|---|
| 480P | `$0.02295` / 秒 | `$0.114750` |
| 720P | `$0.0495` / 秒 | `$0.247500` |
| 1080P | `$0.1215` / 秒 | `$0.607500` |

## 调用示例

<CodeGroup>

```bash 文生视频(有声)
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "doubao-seedance-1.5-pro",
    "prompt": "海边日落,金色阳光洒在海面上,海浪轻轻拍打沙滩,电影感光影",
    "duration": 5,
    "resolution": "720p",
    "aspect_ratio": "16:9",
    "audio": true
  }'
```

```bash 图生视频(首帧驱动)
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "doubao-seedance-1.5-pro",
    "prompt": "人物缓缓走向镜头,镜头微微推进",
    "image_urls": ["https://cdn.example.com/portrait.jpg"],
    "duration": 5,
    "resolution": "1080p"
  }'
```

```bash 首尾帧(有声)
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "doubao-seedance-1.5-pro",
    "prompt": "她回头微笑,说:你来了。",
    "image_urls": [
      "https://cdn.example.com/start.jpg",
      "https://cdn.example.com/end.jpg"
    ],
    "duration": 10,
    "resolution": "1080p",
    "audio": true
  }'
```

</CodeGroup>

## 媒体字段路由

`doubao-seedance-1.5-pro` 仅支持 `generate` 一种 action,后端按 `image_urls` 数量自动决定首帧 / 尾帧用途,**无需显式传 `action`**。

| `image_urls` 数量 | 用途 | 说明 |
|---|---|---|
| `0` 张(或不传) | 文生视频 | 纯 prompt 驱动,画面比例由 `aspect_ratio` 控制 |
| `1` 张 | 图生视频 | 该图作为首帧 |
| `2` 张 | 首尾帧 | 第一张作首帧,第二张作尾帧,跟随首帧比例 |

## 通用参数

<ParamField body="model" type="string" required>
  固定为 `doubao-seedance-1.5-pro`
</ParamField>

<ParamField body="prompt" type="string" required>
  视频描述文本。建议详细描述场景 / 动作 / 风格 / 对白,以获得更好的生成效果
</ParamField>

<ParamField body="duration" type="integer" default="5">
  视频时长(秒),可选 `5` 或 `10`
</ParamField>

<ParamField body="resolution" type="string" default="720p">
  输出分辨率,可选值:
  - `480p` — 标清
  - `720p` — 高清(默认)
  - `1080p` — 全高清
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
  画面宽高比,**仅文生视频生效**;带 `image_urls` 时跟随首帧比例。可选值:
  - `16:9` — 横版宽屏
  - `9:16` — 竖版长屏
  - `1:1` — 正方形
</ParamField>

<ParamField body="image_urls" type="string[]">
  参考图片 URL 数组(0-2 张):
  - 1 张 = 首帧驱动(图生视频)
  - 2 张 = 首帧 + 尾帧约束插值
</ParamField>

<ParamField body="seed" type="integer">
  随机种子。相同请求 + 相同 seed 会生成类似结果(不保证完全一致);省略或传 `-1` 则随机
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook 回调地址,任务终态时调用。详见 [回调机制](/cn/docs/sync-async#异步任务-webhook)
</ParamField>

## 模型特定参数

<ParamField body="audio" type="boolean" default="false">
  是否生成音频(有声视频)。设为 `true` 时视频包含 AI 生成的配套音频,支持多语言对白 + 口型同步
</ParamField>

<ParamField body="camerafixed" type="boolean" default="false">
  是否固定摄像头。设为 `true` 时摄像头位置保持固定
</ParamField>

## 资源限制

| 项目 | 限制 |
|---|---|
| 参考图(`image_urls`) | 最多 2 张(首帧 + 尾帧),JPG / PNG / WEBP |
| Prompt 长度 | 建议详细描述场景 / 动作 / 对白 |
| 视频时长 | 5 或 10 秒 |
| 输出 | MP4,链接 24 小时有效 |
