昆仑万维 **SkyReels V4** 系列视频生成模型,覆盖 **文生视频 / 图生视频 / 参考视频** 三类场景,统一走异步任务入口。480p / 720p / 1080p 三档分辨率,3-15 秒任意整数时长。

两个档位:

- **`skyreels-v4-fast`** — 加速档,响应更快、价格更低,适合创意预览、批量草稿、A/B 比对
- **`skyreels-v4-std`** — 标准档,采样更深、画质更稳定,适合最终交付、客户成片、复杂运动镜头

两档共享同一套参数与能力,只是采样深度不同。建议先用 fast 跑通构思,再用 std 出终版。

## 参考视频加价

SkyReels V4 支持传入 **参考视频**(`video_urls`)作为运动 / 风格参考。是否传 `video_urls` 决定计费档位:

- **不传 `video_urls`(普通档)** — 文生视频或图生视频(传 `image_urls`),按各分辨率 **普通档** 单价计费。
- **传 `video_urls`(参考视频档)** — 以参考视频引导生成,按更贵的 **参考视频档** 单价计费(约为普通档的 1.8-2x)。

下方定价表分别列出两组单价。

## 定价

全系 **按秒计费**(`$/秒`),实际费用 = 对应档位单价 × 时长(秒)。售价即结算 USD 价。

### `skyreels-v4-fast`

| 分辨率 | 普通档 | 参考视频档 |
|---|---|---|
| 480p | `$0.072` / 秒 | `$0.135` / 秒 |
| 720p | `$0.099` / 秒 | `$0.18` / 秒 |
| 1080p | `$0.2475` / 秒 | `$0.45` / 秒 |

### `skyreels-v4-std`

| 分辨率 | 普通档 | 参考视频档 |
|---|---|---|
| 480p | `$0.099` / 秒 | `$0.162` / 秒 |
| 720p | `$0.126` / 秒 | `$0.225` / 秒 |
| 1080p | `$0.315` / 秒 | `$0.5625` / 秒 |

实际扣费 = 单价 / 秒 × `duration`。例:`skyreels-v4-fast` 720p 普通档 5 秒 = `$0.099 × 5 = $0.495`;同档传参考视频后 = `$0.18 × 5 = $0.9`。

<Note>
**参考视频加价** — 只要请求带 `video_urls`,即按"参考视频档"列计价;不带则按"普通档"列。传不传 `image_urls`(图生)不影响档位。
</Note>

## 调用示例

<CodeGroup>

```bash 文生视频(普通档)
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "skyreels-v4-fast",
    "prompt": "一只柴犬穿宇航服在月球漫步,镜头缓缓推近,电影感光影",
    "duration": 5,
    "resolution": "1080p",
    "aspect_ratio": "16:9"
  }'
```

```bash 图生视频(普通档)
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "skyreels-v4-std",
    "prompt": "人物缓缓走向镜头,长发飘动",
    "image_urls": ["https://cdn.example.com/portrait.jpg"],
    "duration": 8,
    "resolution": "1080p"
  }'
```

```bash 参考视频(参考视频档)
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "skyreels-v4-fast",
    "prompt": "复刻参考视频的镜头运动,替换为雪山场景",
    "video_urls": ["https://cdn.example.com/source.mp4"],
    "duration": 6,
    "resolution": "1080p"
  }'
```

</CodeGroup>

<ResponseExample>
```json 提交成功
{
  "task_id": "task-wave1775285160b950328499",
  "model": "skyreels-v4-fast",
  "action": "generate",
  "status": "queued",
  "created_at": 1775285160040,
  "progress": 0
}
```
</ResponseExample>

提交后用 [`GET /v1/tasks/{task_id}`](/cn/api-reference/task/status) 轮询状态,详见 [任务系统](/cn/docs/task-system)。

## 媒体字段路由

`skyreels-v4-fast` / `skyreels-v4-std` 均仅支持 `generate` 一种 action,后端按传入的媒体字段自动决定生成模式,**无需显式传 `action`**。

| 传入字段 | 模式 | 计费档 |
|---|---|---|
| 仅 `prompt` | 文生视频 | 普通档 |
| `prompt` + `image_urls` | 图生视频 | 普通档 |
| `prompt` + `video_urls` | 参考视频 | 参考视频档 |

## 可用模型

| 模型 ID | 档位 | 分辨率 | 时长 | 宽高比 | 说明 |
|---|---|---|---|---|---|
| `skyreels-v4-fast` | 加速档 | 480p / 720p / 1080p | 3-15 秒任意整数 | 16:9 / 9:16 / 1:1 | 响应更快、更便宜,适合预览 / 草稿 |
| `skyreels-v4-std` | 标准档 | 480p / 720p / 1080p | 3-15 秒任意整数 | 16:9 / 9:16 / 1:1 | 画质更稳定,适合最终交付 |

## 通用参数

<ParamField body="model" type="string" required>
  模型 ID,可选值:
  - `skyreels-v4-fast` — 加速档
  - `skyreels-v4-std` — 标准档
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作类型,固定为 `generate`。生成模式由媒体字段自动路由,见 [媒体字段路由](#媒体字段路由)
</ParamField>

<ParamField body="prompt" type="string" required>
  视频描述文本,建议描述 **动作 / 镜头 / 氛围 / 光影**。最多 5000 字符
</ParamField>

<ParamField body="resolution" type="string" default="720p">
  输出分辨率,可选值:
  - `480p`
  - `720p`
  - `1080p`
</ParamField>

<ParamField body="duration" type="integer" default="5">
  视频时长(秒),范围 **3-15** 任意整数
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
  画面宽高比,文生模式生效;带 `image_urls` / `video_urls` 时跟随源素材比例。可选值:
  - `16:9` — 横版宽屏
  - `9:16` — 竖版长屏
  - `1:1` — 正方形
</ParamField>

<ParamField body="image_urls" type="string[]">
  输入图片 URL 数组,传入后进入 **图生视频** 模式。需公网可下载链接
</ParamField>

<ParamField body="video_urls" type="string[]">
  参考视频 URL 数组,传入后进入 **参考视频** 模式,以视频引导运动 / 风格。**传入即触发参考视频加价档**(见 [定价](#定价))。需公网可下载链接
</ParamField>

<ParamField body="seed" type="integer">
  随机种子(≥0)。相同请求 + 相同 seed 会生成相似结果,不指定则每次不同
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook 回调地址,任务终态时调用。详见 [回调机制](/cn/docs/sync-async#异步任务-webhook)
</ParamField>

<ParamField body="callback_events" type="string[]">
  订阅的回调事件,默认订阅终态(`succeeded` / `failed` / `cancelled`)
</ParamField>

## 模型特定参数

两档参数完全一致,区别仅在生成速度与画质,按场景选择即可。

<Tabs>
  <Tab title="skyreels-v4-fast">
    **定位:** 加速档,响应更快、价格更低,适合预览 / 草稿 / 高吞吐场景。无额外特定参数,所有可选项见 [通用参数](#通用参数)。
  </Tab>

  <Tab title="skyreels-v4-std">
    **定位:** 标准档,画质更稳定、运动更自然,适合最终交付级别的视频。无额外特定参数,所有可选项见 [通用参数](#通用参数)。
  </Tab>
</Tabs>

## 资源限制

| 项目 | 限制 |
|---|---|
| 输入图片(`image_urls`) | JPG / PNG / WEBP,需公网可下载链接 |
| 参考视频(`video_urls`) | MP4 / MOV,需公网可下载链接 |
| Prompt | ≤ 5000 字符 |
| 视频时长 | 3-15 秒任意整数 |
| 输出 | MP4,链接 24 小时有效,请及时下载 |
