<Note>
本接口采用异步模式。提交成功后，接口会返回一个任务 ID。您需要使用该 ID 调用 [查询任务状态](/cn/api-reference/task/get-status) 接口获取最终生成结果。视频链接有效期为 24 小时，请及时下载。
</Note>

## 认证方式

在请求头中加入 `Authorization: Bearer YOUR_API_KEY` 进行身份验证。您可以在 [令牌管理](https://qingbo.dev/dashboard/keys) 页面获取您的 API 密钥。

## 请求参数

<ParamField body="model" type="string" required>
固定值：`kling-v2-6`
</ParamField>

<ParamField body="prompt" type="string" required>
视频生成的文本描述。例如："一只金色的猫在阳光明媚的草地上奔跑，慢动作，电影质感"
</ParamField>

<ParamField body="mode" type="string" default="std">
生成模式：`std`（标准 720P，仅静音）, `pro`（专业 1080P，支持音频生成）。
</ParamField>

<ParamField body="duration" type="number" default="5">
视频时长。支持 `5` 或 `10` 秒。
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
视频宽高比，支持 `16:9`, `9:16`, `1:1`。
</ParamField>

<ParamField body="negative_prompt" type="string">
负向提示词，用于排除不想要的内容。
</ParamField>

<ParamField body="image_urls" type="array">
支持最多 2 张图片 URL。1 张作为首帧，2 张作为首尾帧（需在 `pro` 模式下）。
</ParamField>

<ParamField body="audio" type="boolean" default="false">
是否生成配套音频。仅在 `pro` 模式下有效，且与尾帧输入互斥。
</ParamField>

<ParamField body="watermark" type="boolean" default="false">
是否添加水印。
</ParamField>

## 响应参数

<ResponseField name="code" type="string">
响应状态码 * `success` - 请求成功，任务已提交
</ResponseField>

<ResponseField name="data" type="string">
任务 ID，格式为 `task-wave{timestamp}{userId}{random}`，用于通过 [查询任务状态](/cn/api-reference/task/get-status) 接口获取生成结果
</ResponseField>

## 功能支持矩阵

| 功能 | 标准模式 (std) | 专业模式 (pro) |
| :--- | :--- | :--- |
| 分辨率 | 720P | 1080P |
| 视频时长 | 5s / 10s | 5s / 10s |
| 首尾帧 | 不支持 | 支持 |
| 音频生成 | 不支持 | 支持 |
| 运镜控制 | 支持 | 支持 |

## 使用示例

### 专业模式文生视频 (10s)

```json theme={null}
{
  "model": "kling-v2-6",
  "prompt": "赛博朋克风格的繁华都市街道，雨夜，霓虹灯反射在积水中",
  "mode": "pro",
  "duration": 10,
  "aspect_ratio": "16:9"
}
```

```json theme={null}
{
  "model": "kling-v2-6",
  "prompt": "镜头缓缓推进，展示人物表情变化",
  "mode": "pro",
  "image_urls": ["https://example.com/start.jpg", "https://example.com/end.jpg"],
  "duration": 5
}
```

## 注意事项

- 接口为异步处理，返回任务 ID 后请通过查询接口获取结果。
- 视频链接有效期为 24 小时，过期后将无法访问，请及时转存。
- `pro` 模式下的音频生成功能与双图输入（首尾帧）不可同时使用。
