异步处理模式 — 提交请求后返回任务 ID，通过 [查询任务状态](/cn/api-reference/task/status) 接口获取生成结果。视频链接有效期 24 小时。

<RequestExample>
```bash cURL
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "hailuo-2.3",
    "action": "generate",
    "prompt": "一只白猫慵懒地躺在阳光下的窗台上",
    "duration": 6,
    "resolution": "1080p"
  }'
```

```python Python
import requests

response = requests.post(
    "https://www.qingbo.dev/v1/tasks",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "hailuo-2.3",
        "action": "generate",
        "prompt": "一只白猫慵懒地躺在阳光下的窗台上",
        "duration": 6,
        "resolution": "1080p"
    }
)

task_id = response.json()["task_id"]
print(f"任务已提交: {task_id}")
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"

        "resolution": "1080p",
    }

    body, _ := json.Marshal(payload)
    req, _ := http.NewRequest("POST", "https://www.qingbo.dev/v1/tasks", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    resp, _ := http.DefaultClient.Do(req)
    defer resp.Body.Close()
    result, _ := io.ReadAll(resp.Body)
    fmt.Println(string(result))
}
```

```java Java
import java.net.http.*;
import java.net.URI;

public class Main {

            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(payload))
            .build();

        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString());
        System.out.println(response.body());
    }
}
```

## 可用模型

| 模型 ID | 默认分辨率 | 支持时长 | 说明 |
|---------|-----------|---------|------|
| `hailuo-2.3` | 768P | 6s / 10s | 海螺 2.3，支持 4K、角色参考 |
| `hailuo-2.3-fast` | 768P | 6s / 10s | 海螺 2.3 快速版 |
| `hailuo-02` | 768P | 6s / 10s | 海螺 02，支持首尾帧 |
| `T2V-01-Director` | 768P | 6s | 导演模式，支持镜头语法控制 |
| `I2V-01-Director` | 720P | 6s | 图生视频导演模式 |
| `S2V-01` | 720P | 6s | 角色参考模式，保持角色一致性 |

## 统一参数

<ParamField body="model" type="string" required>
  模型 ID
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作类型
</ParamField>

<ParamField body="prompt" type="string" required>
  视频描述文本
</ParamField>

<ParamField body="duration" type="integer" default="6">
  视频时长（秒）：`6`、`10`
</ParamField>

<ParamField body="resolution" type="string" default="768p">
  输出分辨率，可选值取决于模型，见各模型 Tab
</ParamField>

<ParamField body="image_urls" type="string[]">
  首帧图片 URL，用于图生视频
</ParamField>

<ParamField body="last_frame" type="string">
  尾帧图片 URL，需配合 `image_urls`。仅 `hailuo-02` 支持
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook 回调地址
</ParamField>

<ParamField body="extra" type="object">
  透传给海螺的额外参数，详见各模型 Tab
</ParamField>

## 生成模式

| 模式 | 参数 | 支持模型 |
|------|------|---------|
| 文生视频 | `prompt` | 全部 |
| 图生视频 | `prompt` + `image_urls`(1张) | 全部 |
| 首尾帧 | `image_urls`(1张) + `last_frame` | hailuo-02 |
| 角色参考 | `prompt` + `extra.subject_reference` | hailuo-2.3 系列 |
