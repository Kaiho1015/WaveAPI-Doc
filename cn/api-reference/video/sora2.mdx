<Note>
本接口采用异步模式。提交成功后，接口会返回一个任务 ID。您需要使用该 ID 调用 [查询任务状态](/cn/api-reference/task/get-status) 接口获取最终生成结果。视频链接有效期为 24 小时，请及时下载。
</Note>

## 认证方式

在请求头中加入 `Authorization: Bearer YOUR_API_KEY` 进行身份验证。您可以在 [令牌管理](https://ai.kaiho.cc/dashboard/keys) 页面获取您的 API 密钥。

## 请求参数

<ParamField body="model" type="string" required>
可选值：`sora-2`（标准版）, `sora-2-vip`（VIP版）, `sora-2-pro`（专业版,支持更长时长）
</ParamField>

<ParamField body="prompt" type="string" required>
视频生成的文本描述。例如："瀑布倾泻而下形成彩虹"。支持使用 `@username` 引用角色。
</ParamField>

<ParamField body="duration" type="number" default="10">
视频时长。`sora-2` 和 `sora-2-vip` 支持 `10`, `15` 秒；`sora-2-pro` 支持 `10`, `15`, `25` 秒。
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
视频宽高比，支持 `16:9` 和 `9:16`。
</ParamField>

<ParamField body="image_urls" type="array">
参考图 URL。用于图生视频或风格参考，支持 jpeg/jpg/png/webp 格式，最大 10MB。
</ParamField>

<ParamField body="watermark" type="boolean" default="false">
是否添加水印。
</ParamField>

<ParamField body="thumbnail" type="boolean" default="false">
是否生成并保存缩略图。注意：缩略图仅保存 1 小时。
</ParamField>

<ParamField body="private" type="boolean" default="false">
开启隐私模式。
</ParamField>

<ParamField body="style" type="string">
风格选择，支持：`thanksgiving`, `comic`, `news`, `selfie`, `nostalgic`, `anime`。
</ParamField>

<ParamField body="storyboard" type="boolean" default="false">
开启分镜控制。
</ParamField>

<ParamField body="character_url" type="string">
参考视频 URL。用于角色提取和一致性生成。
</ParamField>

<ParamField body="character_timestamps" type="string">
格式为 "start,end"（如 "1,3"），指定从参考视频中提取角色的起始和结束时间。
</ParamField>

## 响应参数

<ResponseField name="code" type="string">
响应状态码 * `success` - 请求成功，任务已提交
</ResponseField>

<ResponseField name="data" type="string">
任务 ID，格式为 `task-wave{timestamp}{userId}{random}`，用于通过 [查询任务状态](/cn/api-reference/task/get-status) 接口获取生成结果
</ResponseField>

## 使用示例

### VIP 文生视频 (15s)

```json theme={null}
{
  "model": "sora-2-vip",
  "prompt": "瀑布倾泻而下形成彩虹",
  "aspect_ratio": "16:9",
  "duration": 15

```

### 专业版角色参考

```json theme={null}
{
  "model": "sora-2-pro",
  "prompt": "一只戴着墨镜的猫在海边冲浪 @cat_surfer",
  "character_url": "https://example.com/character_source.mp4",
  "character_timestamps": "1,3",
  "duration": 10
}
```

## 注意事项

- 接口为异步处理，返回任务 ID 后请通过查询接口获取结果。
- 视频链接有效期为 24 小时，过期后将无法访问，请及时转存。
- 缩略图（如果开启）仅保存 1 小时。
- `character_timestamps` 需严格遵守 "起始秒数,结束秒数" 格式。
