异步处理模式 — 提交请求后返回任务 ID，通过 [查询任务状态](/cn/api-reference/task/status) 接口获取生成结果。视频链接有效期 24 小时。

<RequestExample>
```bash cURL
curl -X POST https://ai.kaiho.cc/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "kling-v2-master",
    "action": "generate",
    "prompt": "一条锦鲤在清澈的池塘中游动，水面波光粼粼",
    "aspect_ratio": "16:9",
    "duration": 5,
    "extra": {
      "mode": "pro"
    }
  }'
```

## 可用模型

| 模型 ID | 说明 |
|---------|------|
| `kling-v3` | 可灵 V3，支持 3-15 秒、多镜头、原生音频 |
| `kling-v2-master` | 可灵 V2 Master |
| `kling-v1-6` | 可灵 V1.6，支持镜头控制 |
| `kling-v1` | 可灵 V1 |

## 统一参数

<ParamField body="model" type="string" required>
  模型 ID
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作类型
</ParamField>

<ParamField body="prompt" type="string" required>
  视频描述文本，最多 2500 字符
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
  宽高比：`16:9`、`9:16`、`1:1`

  图生视频时忽略此参数，宽高比由输入图片决定
</ParamField>

<ParamField body="duration" type="integer" default="5">
  视频时长（秒）：`5`、`10`
</ParamField>

<ParamField body="negative_prompt" type="string">
  不希望出现的内容，最多 2500 字符。仅 V1 / V1.6 支持
</ParamField>

<ParamField body="image_urls" type="string[]">
  参考图片 URL（首帧），用于图生视频。JPEG/PNG/WebP，≤ 10MB
</ParamField>

<ParamField body="last_frame" type="string">
  尾帧图片 URL，需配合 `image_urls` 使用
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook 回调地址
</ParamField>

<ParamField body="extra" type="object">
  透传给可灵的额外参数，详见各模型 Tab
</ParamField>

## 模型特定参数

<Tabs>
  <Tab title="Kling V3">
    **duration 范围：** 3-15 秒

    <ParamField body="extra.mode" type="string" default="std">
      生成模式：`std`（720P）或 `pro`（1080P）
    </ParamField>

    <ParamField body="extra.enable_audio" type="boolean" default="true">
      原生音频生成，默认开启
    </ParamField>

    **多镜头模式：**

    支持 2-6 个镜头分段生成，使用 `shot_N_prompt` + `shot_N_duration` 替代 `prompt` + `duration`：

    ```json
    "extra": {
      "shot_1_prompt": "特写：一杯咖啡在桌上冒着热气",
      "shot_1_duration": 5,
      "shot_2_prompt": "镜头拉远，展示整个咖啡馆的场景",
      "shot_2_duration": 5
    }
    ```

    各镜头 duration 总和须在 3-15 秒范围内。
  </Tab>

  <Tab title="Kling V2 Master">
    <ParamField body="extra.mode" type="string" default="std">
      生成模式：`std`（720P，更快）或 `pro`（1080P，更高质量）
    </ParamField>

    <Note>
    V2 不支持 `negative_prompt`、`cfg_scale` 和 `camera_control`。
    </Note>
  </Tab>

  <Tab title="Kling V1.6">
    <ParamField body="extra.mode" type="string" default="std">
      生成模式：`std` 或 `pro`
    </ParamField>

    <ParamField body="extra.cfg_scale" type="number" default="0.5">
      创意度控制，范围 0-1。值越高越贴近提示词
    </ParamField>

    <ParamField body="extra.camera_control" type="object">
      镜头运动控制

      <Expandable title="预设模式">
        `type` 可选值：`down_back`（下降后退）、`forward_up`（前进上升）、`right_turn_forward`（右转前进）、`left_turn_forward`（左转前进）
      </Expandable>

      <Expandable title="自定义模式">
        设置 `type: "simple"`，通过 `config` 精确控制六轴运动（范围 -10 ~ 10）：

        - `horizontal` — X 平移，负值左移，正值右移
        - `vertical` — Y 平移，负值下移，正值上移
        - `pan` — X 旋转，负值下转，正值上转
        - `tilt` — Y 旋转，负值左转，正值右转
        - `roll` — Z 旋转，负值逆时针，正值顺时针
        - `zoom` — 焦距，负值拉近，正值拉远
      </Expandable>
    </ParamField>

    <ParamField body="extra.static_mask" type="string">
      静态遮罩图片 URL（PNG），定义不运动的区域。仅图生视频可用
    </ParamField>

    <ParamField body="extra.dynamic_masks" type="array">
      动态遮罩数组（最多 6 个），定义按轨迹运动的区域。仅图生视频可用

      <Expandable title="属性">
        <ParamField body="mask_url" type="string">
          遮罩图片 URL（PNG with alpha）
        </ParamField>
        <ParamField body="trajectories" type="array">
          轨迹坐标点数组，每个点包含 x 和 y 坐标，2-77 个点
        </ParamField>
      </Expandable>
    </ParamField>
  </Tab>

  <Tab title="Kling V1">
    <ParamField body="extra.mode" type="string" default="std">
      生成模式：`std` 或 `pro`
    </ParamField>

    <ParamField body="extra.cfg_scale" type="number" default="0.5">
      创意度控制，范围 0-1
    </ParamField>
  </Tab>
</Tabs>

## 生成模式

| 模式 | 参数 | 说明 |
|------|------|------|
| 文生视频 | `prompt` | 纯文本生成 |
| 图生视频 | `prompt` + `image_urls`(1张) | 首帧图 + 文本描述 |
| 首尾帧 | `image_urls`(1张) + `last_frame` | 起止画面过渡 |

<Note>
图生视频时 `aspect_ratio` 由输入图片决定，无需指定。
</Note>
