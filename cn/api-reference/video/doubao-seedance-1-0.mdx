```go Go theme={null}
package main

import (

	client := &http.Client{}
	resp, _ := client.Do(req)

}
```

## 认证方式

在请求头中加入 `Authorization: Bearer YOUR_API_KEY`。您可以从 [API Key 页面](https://qingbo.dev/dashboard/keys) 获取您的密钥。

## 请求参数

<ParamField body="model" type="string" required>
  模型 ID：`doubao-seedance-1-0-pro`
</ParamField>

<ParamField body="prompt" type="string">
  视频生成的提示词，描述画面内容、运动等。
</ParamField>

<ParamField body="image_urls" type="array">
  参考图片链接列表。支持文生视频、图生视频（首帧、尾帧或参考图）。
</ParamField>

<ParamField body="aspect_ratio" type="string">
  视频比例。可选值：`16:9`, `9:16`, `1:1`。
</ParamField>

<ParamField body="duration" type="integer">
  视频时长（秒）。
</ParamField>

## 响应参数

<ResponseField name="code" type="string">
  响应状态码 * `success` - 请求成功，任务已提交
</ResponseField>

<ResponseField name="data" type="string">
  任务 ID，格式为 `task-wave{timestamp}{userId}{random}`，用于通过 [查询任务状态](/cn/api-reference/task/get-status) 接口获取生成结果
</ResponseField>

## 使用示例

### 文生视频
```json theme={null}
{
  "model": "doubao-seedance-1-0-pro",
  "prompt": "樱花树下的古风女子缓缓转身",
  "aspect_ratio": "16:9"
}
```

### 图生视频
```json theme={null}
{
  "model": "doubao-seedance-1-0-pro",
  "image_urls": ["https://example.com/start_frame.jpg"],
  "prompt": "让图中人物动起来，背景飘落雪花"
}
```

## 注意事项

- 请确保图片 URL 是公网可访问的直接链接。
- 视频生成为耗时操作，请务必保存返回的 `data` (任务 ID) 以供后续查询。
- 如果请求失败，请参考错误码进行排查。
