```go Go theme={null}
package main

import (

func main() {

	req, _ := http.NewRequest("POST", url, bytes.NewBuffer(jsonPayload))

	req.Header.Set("Content-Type", "application/json")

	client := &http.Client{}
	resp, _ := client.Do(req)
	defer resp.Body.Close()

	var result map[string]interface{}
	json.NewDecoder(resp.Body).Decode(&result)
	fmt.Println(result)
}
```

## 认证方式

在请求头中加入 `Authorization: Bearer YOUR_API_KEY`。您可以从 [API Key 页面](https://ai.kaiho.cc/dashboard/keys) 获取您的密钥。

## 请求参数

<ParamField body="model" type="string" required>
  模型 ID：`vidu-q3-pro`
</ParamField>

<ParamField body="prompt" type="string">
  视频生成的提示词。
</ParamField>

<ParamField body="image_urls" type="array">
  输入图片的 URL 列表。支持图生视频。
</ParamField>

<ParamField body="resolution" type="string">
  分辨率。可选值：`540p`, `720p`, `1080p`。
</ParamField>

<ParamField body="duration" type="integer">
  视频时长，支持 1-16 秒范围。
</ParamField>

## 响应参数

<ResponseField name="code" type="string">
  响应状态码 * `success` - 请求成功，任务已提交
</ResponseField>

<ResponseField name="data" type="string">
  任务 ID，格式为 `task-wave{timestamp}{userId}{random}`，用于通过 [查询任务状态](/cn/api-reference/task/get-status) 接口获取生成结果
</ResponseField>

## 使用示例

### 文本生成视频
```json theme={null}
{
  "model": "vidu-q3-pro",
  "prompt": "宇航员在月球表面行走，地球在远处升起",
  "resolution": "1080p"
}
```

### 首尾帧视频生成
```json theme={null}
{
  "model": "vidu-q3-pro",
  "image_urls": ["https://example.com/start.jpg", "https://example.com/end.jpg"],
  "prompt": "镜头平滑地从起景过渡到终景"
}
```

## 注意事项

- 默认开启音频生成。
- 视频生成为耗时操作，请妥善保存 `data` 字段返回的任务 ID。
- 生成结果可通过 [查询任务状态](/cn/api-reference/task/get-status) 链接查看。
