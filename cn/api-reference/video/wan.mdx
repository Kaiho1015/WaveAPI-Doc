阿里巴巴通义万相视频生成模型,**三代演进**:

- **2.5** — 双模 T2V/I2V,480P/720P/1080P,5/10s,原生音频
- **2.6** — 影视级,720P/1080P,5/10/15s,多镜头 + 12 种特效模板;另提供 `wan2.6-i2v` / `wan2.6-i2v-flash` 两个 I2V 专用版,2-15s 任意整数,支持音频 on/off
- **2.7** — 统一 T2V/I2V/首尾帧/视频续写;另有 `wan2.7-r2v` 参考生(图+视频参考 ≤5,声音克隆)和 `wan2.7-videoedit` 视频编辑(风格迁移/内容替换)

按 **模型 × 分辨率** 计费,部分版本按时长线性叠加,详见下文定价节。

## 可用模型

| 模型 ID | 说明 |
|---|---|
| `wan2.5` | 双模 T2V/I2V,480P/720P/1080P,5/10s,原生音频 |
| `wan2.6` | 影视级,720P/1080P,5/10/15s,多镜头 + 特效模板 |
| `wan2.6-i2v` | I2V 标准版,单图生视频(高保真),720P/1080P,2-15s |
| `wan2.6-i2v-flash` | I2V 快速版,速度优先,720P/1080P,2-15s,音频 on/off |
| `wan2.7` | 统一 T2V/I2V/首尾帧/视频续写,720P/1080P,2-15s |
| `wan2.7-r2v` | R2V 参考生,多 ref(图+视频 ≤5),声音克隆 |
| `wan2.7-videoedit` | 视频编辑,风格迁移/内容替换,参考 ≤4 |

## 通用参数

<ParamField body="model" type="string" required>
  从 [可用模型](#可用模型) 列表中选一个
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作类型,各 model 支持范围见 [模型特定参数](#模型特定参数) Tab。可选值:
  - `generate` — 文生视频(T2V)
  - `image2video` — 图生视频(I2V)
  - `first_last_frame` — 首尾帧
  - `video_continuation` — 视频续写
  - `reference` — 参考生视频(R2V)
  - `reference_video` — 视频参考生(R2V 视频版)
  - `edit` / `video_edit` / `style_transfer` — 视频编辑
</ParamField>

<ParamField body="prompt" type="string">
  视频描述文本。**T2V 必填**;其他模式可选作引导
</ParamField>

<ParamField body="n" type="integer" default="1">
  并发生成数量
</ParamField>

<ParamField body="seed" type="integer" default="-1">
  随机种子,`-1` 为随机;固定值可复现相似结果
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
  画面宽高比,可选值:
  - `16:9` — 横版宽屏
  - `9:16` — 竖版长屏
  - `1:1` — 正方形
  - `4:3` — 横版
  - `3:4` — 竖版
  - `adaptive` — 跟随源素材(`wan2.6-i2v` / `wan2.6-i2v-flash` 自动 adaptive;`wan2.7-videoedit` 也可选)

  各模型实际支持值见 [模型特定参数](#模型特定参数) Tab
</ParamField>

<ParamField body="resolution" type="string">
  输出分辨率,可选值取决于模型,见各 Tab。
  - `480p` — 仅 `wan2.5`
  - `720p` — 全系列
  - `1080p` — 全系列
</ParamField>

<ParamField body="duration" type="integer" default="5">
  视频时长(秒),范围取决于模型,见各 Tab。
  - `wan2.5`:`5` 或 `10`
  - `wan2.6`:`5` / `10` / `15`
  - `wan2.6-i2v(-flash)` / `wan2.7` / `wan2.7-r2v`:`2-15` 任意整数
  - `wan2.7-videoedit`:`0-10`,`0` 表示跟随源视频
</ParamField>

<ParamField body="image_urls" type="string[]">
  参考图片 URL 数组。**I2V** 单元素;**R2V** 多元素(≤5);**VideoEdit** 作风格/内容参考(≤4)
</ParamField>

<ParamField body="first_frame_image" type="string">
  视频首帧图 URL,**首尾帧 / R2V 角色帧**用
</ParamField>

<ParamField body="last_frame_image" type="string">
  视频尾帧图 URL,需配合 `first_frame_image`
</ParamField>

<ParamField body="video_urls" type="string[]">
  参考视频 URL 数组(单元素),用于 **视频续写 / R2V 视频参考 / 视频编辑**
</ParamField>

<ParamField body="audio_urls" type="string[]">
  音频 URL 数组(单元素即可)。
  - `wan2.5` / `wan2.6` / `wan2.6-i2v(-flash)`:作驱动音 / 自定义配音(wav/mp3, 3-30s, ≤15MB)
  - `wan2.7`:文生场景作背景音;图生场景驱动音同步动作
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook 回调地址,任务终态时调用。详见 [回调机制](/cn/docs/sync-async#异步任务-webhook)
</ParamField>

<ParamField body="callback_events" type="string[]">
  订阅的回调事件类型,默认订阅终态事件
</ParamField>

## 特效模板列表

`wan2.6` / `wan2.6-i2v` / `wan2.6-i2v-flash` 在图生视频时支持以下 12 种特效模板,通过 `template` 字段指定:

### 通用类(单人/双人皆可)

| Template | 说明 |
|---|---|
| `squish` | 柔软挤压变形 |
| `rotation` | 360° 旋转环绕 |
| `poke` | 戳击反弹 |
| `inflate` | 膨胀效果 |
| `dissolve` | 溶解消散 |
| `carousel` | 旋转木马式环绕 |

### 单人类(需图中只含一个主体)

| Template | 说明 |
|---|---|
| `singleheart` | 单人爱心特效 |
| `flying` | 飞翔上升 |
| `rose` | 玫瑰花瓣环绕 |

### 双人类(需图中含两个主体)

| Template | 说明 |
|---|---|
| `hug` | 拥抱 |
| `frenchkiss` | 法式亲吻 |
| `coupleheart` | 情侣爱心特效 |

## 资源限制

| 项目 | 限制 |
|---|---|
| 参考图片 | 单张 ≤ 30MB,JPG/PNG/WEBP;R2V ≤ 5 张,VideoEdit ≤ 4 张 |
| 首尾帧图 | 同参考图,需公网可下载 |
| 源视频 | MP4/MOV,≤ 100MB,2-30 秒 |
| 音频 | WAV/MP3,3-30s,≤ 15MB |
| Prompt | ≤ 5000 字符 |
| 输出 | MP4,链接 24 小时有效,请及时下载 |
