阿里巴巴通义万相(Wan)视频生成系列,覆盖 **文生视频 / 图生视频 / 参考生视频 / 视频编辑** 四类场景,统一走异步任务入口。

五个档位:

- **`wan2.7`** — 最新主力档,文生 / 图生统一入口(传 `image_urls` 自动进图生模式,首帧 / 首尾帧),720P / 1080P,2-15 秒任意整数,支持负面提示词与智能扩写
- **`wan2.6`** — 经典稳定档,文生 / 图生,480p / 720p / 1080p,5 / 10 / 15 秒
- **`wan2.5-preview`** — 2.5 预览档,文生 / 图生(传 `image_urls` 自动进图生模式),480p / 720p / 1080p,5 / 10 秒,价位更亲民
- **`wan2.7-r2v`** — 参考生视频(R2V),以一张或多张参考图作主体 / 风格参考生成新视频,角色一致性 / 风格延续,需传 `image_urls`
- **`wan2.7-videoedit`** — 视频编辑,输入源视频 + 文本指令实现风格迁移 / 内容替换 / 整段改写,需传 `video_urls`

全系 **按秒计费**:实际费用 = 对应分辨率单价 × 时长(秒)。下表为各分辨率每秒售价(已含优惠)。

## 定价

| 模型 | 480P/p | 720P/p | 1080P/p |
|---|---|---|---|
| `wan2.7` | — | `$0.0747` / 秒 | `$0.1233` / 秒 |
| `wan2.6` | `$0.05625` / 秒 | `$0.05625` / 秒 | `$0.0945` / 秒 |
| `wan2.5-preview` | `$0.0378` / 秒 | `$0.0747` / 秒 | `$0.1233` / 秒 |
| `wan2.7-r2v` | — | `$0.0747` / 秒 | `$0.1233` / 秒 |
| `wan2.7-videoedit` | — | `$0.0747` / 秒 | `$0.1233` / 秒 |

实际扣费 = 每秒单价 × `duration`。例:`wan2.7` 1080P 生成 8 秒 = `0.1233 × 8 = $0.9864`。`wan2.7-videoedit` 按用户指定的输出时长结算。

## 可用模型

| 模型 ID | 分辨率 | 时长 | 宽高比 | 必传 | 备注 |
|---|---|---|---|---|---|
| `wan2.7` | 720P / 1080P | 2-15 秒任意整数 | 16:9 / 9:16 / 1:1 / 4:3 / 3:4 | `prompt` | 文生 / 图生统一入口,传 `image_urls` 进图生 |
| `wan2.6` | 480p / 720p / 1080p | 5 / 10 / 15 秒 | 16:9 / 9:16 / 1:1 | `prompt` | 经典稳定档,文生 / 图生 |
| `wan2.5-preview` | 480p / 720p / 1080p | 5 / 10 秒 | 16:9 / 9:16 / 1:1 | `prompt` | 2.5 预览档,文生 / 图生,传 `image_urls` 进图生 |
| `wan2.7-r2v` | 720P / 1080P | 2-15 秒任意整数 | 16:9 / 9:16 / 1:1 | `prompt` + `image_urls` | 参考生视频,角色 / 风格一致性 |
| `wan2.7-videoedit` | 720P / 1080P | 2-15 秒任意整数 | 16:9 / 9:16 / 1:1 | `prompt` + `video_urls` | 视频编辑,风格迁移 / 内容替换 |

<Note>
分辨率大小写按模型区分:`wan2.7` / `wan2.7-r2v` / `wan2.7-videoedit` 用大写 `720P` / `1080P`,`wan2.6` / `wan2.5-preview` 用小写 `480p` / `720p` / `1080p`。
</Note>

## 通用参数

<ParamField body="model" type="string" required>
  模型 ID,见 [可用模型](#可用模型) 表
</ParamField>

<ParamField body="prompt" type="string" required>
  视频内容描述。文生模式必填;图生 / 参考生 / 编辑模式可选,但建议填写以指导运镜与动作。最多 5000 字符
</ParamField>

<ParamField body="image_urls" type="string[]">
  参考 / 输入图片 URL 数组:
  - `wan2.7`:传入后进入图生模式 —— 1 张为首帧,2 张为首尾帧
  - `wan2.6`:仅支持 1 张,传入后进入图生模式
  - `wan2.5-preview`:传入后进入图生模式(图生视频)
  - `wan2.7-r2v`:**必传**,作为主体 / 风格参考(R2V)
  - `wan2.7-videoedit`:可选,叠加风格 / 内容参考图,**最多 4 张**
</ParamField>

<ParamField body="video_urls" type="string[]">
  源视频 URL 数组(单元素)。`wan2.7-videoedit` **必传**;暂不支持 base64,需公网可下载链接
</ParamField>

<ParamField body="resolution" type="string">
  输出分辨率,各模型档位见 [可用模型](#可用模型) 表:
  - `wan2.7` / `wan2.7-r2v` / `wan2.7-videoedit`:`720P` / `1080P`(默认 `1080P`)
  - `wan2.6` / `wan2.5-preview`:`480p` / `720p` / `1080p`(默认 `720p`)
</ParamField>

<ParamField body="duration" type="integer">
  视频时长(秒):
  - `wan2.7` / `wan2.7-r2v` / `wan2.7-videoedit`:`2`-`15` 任意整数(默认 `5`)
  - `wan2.6`:仅 `5` / `10` / `15`(默认 `5`)
  - `wan2.5-preview`:仅 `5` / `10`(默认 `5`)

  `wan2.7-videoedit` 按用户指定的输出时长结算。
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
  画面宽高比(图生模式下由输入图片自动决定,此参数会被忽略):
  - `wan2.7`:`16:9` / `9:16` / `1:1` / `4:3` / `3:4`
  - `wan2.6` / `wan2.5-preview` / `wan2.7-r2v` / `wan2.7-videoedit`:`16:9` / `9:16` / `1:1`
</ParamField>

<ParamField body="seed" type="integer">
  随机种子(≥0)。相同请求 + 相同 seed 会生成相似结果,不指定则每次不同
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook 回调地址,任务终态时调用。详见 [回调机制](/cn/docs/sync-async#异步任务-webhook)
</ParamField>

<ParamField body="callback_events" type="string[]">
  订阅的回调事件,默认订阅终态(`succeeded` / `failed` / `cancelled`)
</ParamField>

## 模型特定参数

<Tabs>
  <Tab title="wan2.7">
    <ParamField body="prompt_extend" type="boolean" default="true">
      提示词智能扩写,对较短提示词效果提升明显,但会增加耗时
    </ParamField>

    <ParamField body="negative_prompt" type="string">
      负面提示词,描述不希望出现的内容。示例:`"模糊、变形、低质量"`
    </ParamField>
  </Tab>

  <Tab title="wan2.6">
    Wan2.6 无额外特定参数,所有可选项在 [通用参数](#通用参数) 中。文生 / 图生由是否传 `image_urls` 自动判定。
  </Tab>

  <Tab title="wan2.5-preview">
    Wan2.5 预览档无额外特定参数,所有可选项在 [通用参数](#通用参数) 中。文生 / 图生由是否传 `image_urls` 自动判定。
  </Tab>

  <Tab title="wan2.7-r2v">
    R2V 档无额外特定参数。`image_urls` 必传,作为角色 / 风格参考,prompt 负责描述动作与镜头。
  </Tab>

  <Tab title="wan2.7-videoedit">
    VideoEdit 档无额外特定参数。`video_urls` 必传(源视频),`image_urls` 可叠加风格 / 内容参考图(≤4 张),`duration` 由用户指定输出时长。
  </Tab>
</Tabs>

## 资源限制

| 项目 | 限制 |
|---|---|
| 源视频(videoedit) | MP4 / MOV,≤ 100MB,需公网可下载 |
| 参考图片 | JPG / PNG / WEBP,需公网可下载链接(videoedit 最多 4 张) |
| Prompt | ≤ 5000 字符 |
| 输出 | MP4,链接 24 小时有效,请及时下载 |
