阿里巴巴通义万相 2.7 视频编辑专版。

**专为视频改写场景设计** — 输入一段源视频 + 文本指令(可叠加风格参考图 ≤4 张),输出风格迁移 / 内容替换 / 整段改写后的新视频。区别于 T2V/I2V,VideoEdit 必须传 `video_urls`,围绕"已有视频再加工"工作。

按 **分辨率** 计费,源视频时长线性叠加。

<Note>
万相 2.7 系列其他档(T2V / I2V / R2V)目前未上架,如有需求请联系商务。
</Note>

## 定价

| 分辨率 | 单价 |
|---|---|
| 720P | `$0.07055` / 秒 |
| 1080P | `$0.11645` / 秒 |

VideoEdit 按编辑后视频时长结算,`duration=0` 跟随源视频时,按源视频时长计费。

## 调用示例

<CodeGroup>

```bash 基础视频编辑
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "wan2.7-videoedit",
    "action": "edit",
    "prompt": "把场景改为雪天,保留人物动作",
    "video_urls": ["https://cdn.example.com/source.mp4"],
    "duration": 0,
    "resolution": "1080p"
  }'
```

```bash 加风格参考图
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "wan2.7-videoedit",
    "action": "style_transfer",
    "prompt": "将整段视频迁移为水墨画风格",
    "video_urls": ["https://cdn.example.com/source.mp4"],
    "image_urls": [
      "https://cdn.example.com/style-1.jpg",
      "https://cdn.example.com/style-2.jpg"
    ],
    "duration": 0,
    "resolution": "1080p",
    "aspect_ratio": "adaptive"
  }'
```

```bash 控制音频策略
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "wan2.7-videoedit",
    "action": "video_edit",
    "prompt": "替换背景为霓虹城市街道",
    "video_urls": ["https://cdn.example.com/source.mp4"],
    "metadata": {"audio_setting": "origin"},
    "duration": 8,
    "resolution": "1080p"
  }'
```

</CodeGroup>

<ResponseExample>
```json 提交成功
{
  "task_id": "task-wave1775285160b950328499",
  "model": "wan2.7-videoedit",
  "action": "edit",
  "status": "queued",
  "created_at": 1775285160040,
  "progress": 0
}
```

提交后用 [`GET /v1/tasks/{task_id}`](/cn/api-reference/task/status) 轮询状态,详见 [任务系统](/cn/docs/task-system)。

## 可用模型

| 模型 ID | 说明 |
|---|---|
| `wan2.7-videoedit` | 视频编辑,风格迁移 / 内容替换 / 整段改写,720P/1080P,0-10 秒,参考图 ≤4 |

## 通用参数

<ParamField body="model" type="string" required>
  固定为 `wan2.7-videoedit`
</ParamField>

<ParamField body="action" type="string" default="edit">
  操作类型,可选值:
  - `edit` — 视频编辑(默认)
  - `video_edit` — 等同于 `edit`
  - `style_transfer` — 风格迁移
</ParamField>

<ParamField body="prompt" type="string" required>
  编辑指令文本,描述目标效果(替换 / 迁移 / 改写),最多 5000 字符
</ParamField>

<ParamField body="video_urls" type="string[]" required>
  源视频 URL 数组(单元素),**必传**:
  - 暂不支持 base64
  - 需公网可下载链接
</ParamField>

<ParamField body="image_urls" type="string[]">
  风格 / 内容参考图片 URL 数组,**最多 4 张**
</ParamField>

<ParamField body="aspect_ratio" type="string" default="adaptive">
  画面宽高比,可选值:
  - `16:9` — 横版宽屏
  - `9:16` — 竖版长屏
  - `1:1` — 正方形
  - `4:3` — 横版
  - `3:4` — 竖版
  - `adaptive` — 跟随源视频比例(推荐)
</ParamField>

<ParamField body="resolution" type="string" default="720p">
  输出分辨率,可选值:
  - `720p`
  - `1080p`
</ParamField>

<ParamField body="duration" type="integer" default="0">
  视频时长(秒),范围 `0-10`:
  - `0` — 跟随源视频时长(推荐)
  - `1-10` — 指定输出时长
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook 回调地址,任务终态时调用。详见 [回调机制](/cn/docs/sync-async#异步任务-webhook)
</ParamField>

<ParamField body="callback_events" type="string[]">
  订阅的回调事件类型,默认订阅终态事件
</ParamField>

## 模型特定参数

<ParamField body="watermark" type="boolean" default="false">
  右下角添加 AI 生成水印
</ParamField>

<ParamField body="prompt_extend" type="boolean" default="true">
  提示词智能改写,后端按目标风格自动扩写编辑指令
</ParamField>

<ParamField body="metadata" type="object">
  附加参数对象,常用 `audio_setting` 控制输出音轨策略:
  - `auto` — 自动生成与编辑后视频匹配的音频(默认)
  - `origin` — 保留源视频原音轨

  示例:
  ```json
  {"metadata": {"audio_setting": "origin"}}
  ```
</ParamField>

## 资源限制

| 项目 | 限制 |
|---|---|
| 源视频 | MP4/MOV,≤ 100MB,2-30 秒,需公网可下载 |
| 风格参考图 | 最多 4 张,单张 ≤ 30MB,JPG/PNG/WEBP |
| Prompt | ≤ 5000 字符 |
| 输出 | MP4,链接 24 小时有效,请及时下载 |
