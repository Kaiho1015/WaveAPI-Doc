Google Gemini 原生多模态图像系列,**三代 Nano Banana 演进 + 双档接入**:

- **Nano Banana 2(Gemini 3.1 Flash Image Preview)** — 轻量快速档,延迟低、成本省,**独家支持 Google 文字 / 图片搜索增强**,适合预览、批量、迭代
- **Nano Banana Pro(Gemini 3 Pro Image Preview)** — 旗舰档,4K 原生输出,专业级保真度,多步规划,内置质量分析
- **Nano Banana(Gemini 2.5 Flash Image Preview)** — 老一代主力,低延迟、高性价比,多图融合,角色一致性

每代提供两档接入:**标准档**(经济、性价比线)与 **官方直连档**(高 SLA、稳定性更高,价格更高)。两档同模型同能力,区别只在通道与计费。**推荐直接选用对应的 `-official` model id 切换通道**(语义清晰),也可在标准档上设 `official_fallback: true` 在运行时切换。

按张计费,所有版本支持中英文 prompt。

## 模式速查

| 模式 | 触发字段 | 支持版本 |
|---|---|---|
| **文生图** | 仅 `prompt` | 全部 |
| **图生图(image2image)** | `+ image_urls` + `action: "image2image"` | 全部 |
| **图像编辑(指令式重绘)** | `+ image_urls` + `action: "edit"` | 全部 |
| **多图参考融合** | `+ image_urls`(多张)+ `action: "reference"` | 全部 |
| **搜索增强(grounding)** | `action: "search_grounded"` 或 `google_search: true` | **仅 3.1 Flash 系列** |

## 通用参数

<ParamField body="model" type="string" required>
  从 [可用模型](#可用模型) 列表中选一个
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作类型,可选值:
  - `generate` — 文生图(默认)
  - `image2image` — 图生图(需配合 `image_urls`)
  - `edit` — 图像编辑(指令式重绘 / 局部修改)
  - `reference` — 多图参考融合
  - `search_grounded` — 搜索增强(**仅 3.1 Flash 系列**)
</ParamField>

<ParamField body="prompt" type="string" required>
  图像描述文本,支持中英文。Gemini 系列对自然语言指令理解力强,擅长接受多步骤、多约束 prompt
</ParamField>

<ParamField body="n" type="integer" default="1">
  生成数量(单次返回多张)
</ParamField>

<ParamField body="seed" type="integer" default="-1">
  随机种子,`-1` 为随机;固定值可复现相似结果
</ParamField>

<ParamField body="aspect_ratio" type="string" default="1:1">
  画面宽高比,**取值范围取决于模型**:

  标准 10 种(全部模型支持):
  - `1:1` — 正方形
  - `2:3` / `3:2` — 竖/横版相机比
  - `3:4` / `4:3` — 竖/横版标准
  - `4:5` / `5:4` — 竖/横版社媒
  - `9:16` / `16:9` — 竖/横版宽屏
  - `21:9` — 超宽屏

  极端比例(**仅 3.1 Flash 系列**):
  - `1:4` / `4:1` — 长条 / 横幅
  - `1:8` / `8:1` — 极致长条 / 极致横幅
</ParamField>

<ParamField body="resolution" type="string" default="1K">
  输出分辨率,**取决于模型**:
  - 3 Pro:`1K` / `2K` / `4K`
  - 3.1 Flash:`0.5K` / `1K` / `2K` / `4K`
  - 2.5 Flash:`1K`
</ParamField>

<ParamField body="image_urls" type="string[]">
  参考图片 URL 数组。`image2image` / `edit` 通常一张,`reference` 多张
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook 回调地址,任务终态时调用。详见 [回调机制](/cn/docs/sync-async#异步任务-webhook)
</ParamField>

<ParamField body="callback_events" type="string[]">
  回调事件过滤,如 `["completed", "failed"]`
</ParamField>

## 资源限制

| 项目 | 限制 |
|---|---|
| 参考图片 | `reference` 模式建议 ≤ 4 张;`edit` / `image2image` 通常 1 张 |
| 单张图片大小 | ≤ 30MB,支持 JPG / PNG / WEBP |
| 输出分辨率 | 详见各 Tab(3 Pro 最高 4K / 3.1 Flash 最高 4K / 2.5 Flash 仅 1K) |
| 输出文件 | JPG,链接 24 小时有效 |
| 搜索增强 | 仅 3.1 Flash 系列;开启 `google_search` / `google_image_search` 可能略增延迟 |
