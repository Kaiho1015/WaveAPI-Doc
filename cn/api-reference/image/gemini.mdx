Google Gemini 原生多模态图像系列,**三代 Nano Banana 演进 + 双档接入**:

- **Nano Banana 2(Gemini 3.1 Flash Image Preview)** — 轻量快速档,延迟低、成本省,**标准档独家支持 Google 文字 / 图片搜索增强**,支持 `0.5K`~`4K` 分辨率,适合预览、批量、迭代
- **Nano Banana Pro(Gemini 3 Pro Image Preview)** — 旗舰档,4K 原生输出,专业级保真度,多步规划,内置质量分析
- **Nano Banana(Gemini 2.5 Flash Image Preview)** — 老一代主力,低延迟、高性价比,多图融合,角色一致性

每代提供两档接入:**标准档**(经济、性价比线)与 **官方直连档**(高 SLA、稳定性更高,价格更高)。两档同模型同能力,区别只在通道与计费。**推荐直接选用对应的 `-official` model id 切换通道**(语义清晰),也可在标准档上设 `official_fallback: true` 在运行时切换(官方直连档自身不支持该字段)。

按张计费;分辨率越高的档位单价越高。所有版本支持中英文 prompt。

## 模式速查

所有 Gemini 图像模型只有两个 action:`generate`(文生图)与 `edit`(基于参考图改写 / 局部重绘)。多图融合、图生图都走 `edit` + `image_urls`。

| 模式 | 触发字段 | 支持版本 |
|---|---|---|
| **文生图** | `action: "generate"`(默认)+ `prompt` | 全部 |
| **图像编辑 / 图生图** | `action: "edit"` + `image_urls` | 全部 |
| **多图参考融合** | `action: "edit"` + `image_urls`(多张) | 全部 |
| **局部重绘(inpaint)** | `action: "edit"` + `image_urls` + `mask_url` | 全部 |
| **搜索增强(grounding)** | `google_search: true`(可选 `google_image_search`) | **仅 3.1 Flash 标准档** |

## 调用示例

<CodeGroup>

```bash 文生图(最简,Nano Banana Pro)
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini-3-pro-image-preview",
    "prompt": "极简主义工作室人像,柔和侧光,胶片质感,4K 锐利",
    "resolution": "4K",
    "aspect_ratio": "3:2"
  }'
```

```bash 多图融合(Nano Banana Pro)
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini-3-pro-image-preview",
    "action": "edit",
    "prompt": "把第一张图的角色融入第二张图的场景,保持光影一致",
    "image_urls": [
      "https://cdn.example.com/character.jpg",
      "https://cdn.example.com/scene.jpg"
    ],
    "resolution": "4K",
    "aspect_ratio": "16:9"
  }'
```

```bash 局部重绘(inpaint,需 mask_url)
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini-2.5-flash-image-preview",
    "action": "edit",
    "prompt": "把背景换成海边日落,人物保持不变",
    "image_urls": ["https://cdn.example.com/portrait.jpg"],
    "mask_url": "https://cdn.example.com/mask.png",
    "aspect_ratio": "1:1"
  }'
```

```bash 搜索增强(3.1 Flash 标准档独家)
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \

    "prompt": "画一张本周登顶 App Store 的热门游戏的关键画面海报",
    "google_search": true,
    "google_image_search": true,
    "resolution": "2K",
    "aspect_ratio": "16:9"
  }'
```

```bash 官方直连档(高 SLA)

  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini-3-pro-image-preview-official",
    "prompt": "复杂排版海报,包含中英双语标语和密集小字,要求清晰可读",
    "resolution": "4K",
    "aspect_ratio": "2:3"
  }'
```

## 通用参数

<ParamField body="model" type="string" required>
  从 [可用模型](#可用模型) 列表中选一个
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作类型,可选值:
  - `generate` — 文生图(默认)
  - `edit` — 基于参考图改写 / 图生图 / 多图融合 / 局部重绘(需配合 `image_urls`)
</ParamField>

<ParamField body="prompt" type="string" required>
  图像描述文本,支持中英文。Gemini 系列对自然语言指令理解力强,擅长接受多步骤、多约束 prompt
</ParamField>

<ParamField body="n" type="integer" default="1">
  生成数量(单次返回多张)
</ParamField>

<ParamField body="aspect_ratio" type="string" default="1:1">
  画面宽高比,全部 6 个模型支持以下 11 种取值:
  - `auto` — 自动选择(文生图默认 `1:1` 或 `16:9`;图生图跟随输入)
  - `1:1` — 正方形,头像 / 社媒
  - `2:3` / `3:2` — 竖 / 横版相机比
  - `3:4` / `4:3` — 竖 / 横版标准
  - `4:5` / `5:4` — 竖 / 横版社媒
  - `9:16` / `16:9` — 竖 / 横版宽屏
  - `21:9` — 超宽横幅
</ParamField>

<ParamField body="resolution" type="string" default="1K">
  输出分辨率,**取决于模型**:
  - 3 Pro:`1K` / `2K` / `4K`
  - 3.1 Flash:`0.5K` / `1K` / `2K` / `4K`
  - 2.5 Flash:仅 `1K`
</ParamField>

<ParamField body="image_urls" type="string[]">
  参考图片 URL 数组(仅 `edit` 使用)。图生图 / 局部重绘通常一张,多图融合可传多张
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook 回调地址,任务终态时调用。详见 [回调机制](/cn/docs/sync-async#异步任务-webhook)
</ParamField>

<ParamField body="callback_events" type="string[]">
  回调事件过滤,如 `["completed", "failed"]`
</ParamField>

## 资源限制

| 项目 | 限制 |
|---|---|
| 参考图片 | 多图融合可传多张;`edit` 局部重绘 / 图生图通常 1 张 |
| 掩码图 | `mask_url` 仅可一张,需带 Alpha 通道,与 `image_urls` 配合 |
| 单张图片大小 | ≤ 30MB,支持 JPG / PNG / WEBP |
| 输出分辨率 | 3 Pro 最高 4K / 3.1 Flash 最高 4K(最低 0.5K)/ 2.5 Flash 仅 1K |
| 输出文件 | JPG,链接 24 小时有效 |
| 搜索增强 | 仅 3.1 Flash 标准档;开启 `google_search` / `google_image_search` 可能略增延迟 |
