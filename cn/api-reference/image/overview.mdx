## 概述

WaveAPI 支持多种图像生成模型，统一通过异步任务接口 `/v1/tasks` 调用。

## 请求端点

```
POST https://www.qingbo.dev/v1/tasks
```

## 通用请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `model` | string | 是 | 模型名称 |
| `action` | string | 否 | 操作类型，默认 `"generate"` |
| `prompt` | string | 是 | 图像描述文本 |
| `negative_prompt` | string | 否 | 不希望出现的内容 |
| `n` | integer | 否 | 生成数量，默认 1 |
| `size` | string | 否 | 宽高比，如 `"16:9"` |
| `resolution` | string | 否 | 分辨率，如 `"2K"` |
| `image_urls` | array | 否 | 参考图片 URL 数组 |
| `callback_url` | string | 否 | 完成后的回调地址 |
| `extra` | object | 否 | 模型特定的透传参数 |

## 响应格式

任务提交后返回 `task_id`，通过 `GET /v1/tasks/{task_id}` 查询结果：

```json
{
  "task_id": "task-wave1775285160b950328499",
  "model": "doubao-seedream-5.0",
  "action": "generate",
  "status": "queued",
  "created_at": 1775285160040,
  "progress": 0
}
```
