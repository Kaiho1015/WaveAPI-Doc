字节跳动 Seed 团队的图像生成模型系列,**三代演进**:

- **5.0 Lite** — 视觉推理(CoT)+ 实时联网搜索,14 张参考图,2K/3K,强身份保留,性价比线
- **4.5** — 专业生产级,原生 4K,14 张参考,94% 排版准确率,跨角度同人物一致性 90%
- **4.0** — 多模态创作,2-10 张参考图融合,文生 / 编辑 / 组图统一接口

按张计费,生成 1K-4K 图片(各版本支持范围不同)。

## 定价

| 模型 | 单价(每张) |
|---|---|
| `doubao-seedream-5.0-lite` | `$0.02975` |
| `doubao-seedream-4.5` | `$0.02975` |
| `doubao-seedream-4.0` | `$0.0238` |

## 模式速查

| 模式 | 触发字段 | 支持版本 |
|---|---|---|
| **文生图** | 仅 `prompt` | 全部 |
| **图编辑(局部重绘 / 光影修改)** | `+ image_urls`(单张)+ `action: "edit"` | 4.0 / 4.5 |
| **多图参考融合** | `+ image_urls`(多张) | 全部(4.0=2-10 / 4.5=14 / 5.0-lite=14) |
| **组图生成(主题相关一组)** | `sequential_image_generation: "auto"` | 全部 |

## 调用示例

<CodeGroup>

```bash 文生图(最简)
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "doubao-seedream-5.0-lite",
    "prompt": "未来主义城市夜景,霓虹灯映在湿润街面,电影感构图",
    "resolution": "2K",
    "aspect_ratio": "16:9"
  }'
```

```bash 多图参考融合(4.5)
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "doubao-seedream-4.5",
    "prompt": "把第一张图的角色放在第二张图的场景里",
    "image_urls": [
      "https://cdn.example.com/character.jpg",
      "https://cdn.example.com/scene.jpg"
    ],
    "resolution": "4K",
    "aspect_ratio": "16:9"
  }'
```

```bash 图像编辑(4.0)
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "doubao-seedream-4.0",
    "action": "edit",
    "prompt": "把背景换成黄昏天空,人物保持不变",
    "image_urls": ["https://cdn.example.com/portrait.jpg"],
    "resolution": "2K"
  }'
```

```bash 组图生成(主题相关 1 组)
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer $WAVE_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "doubao-seedream-4.5",
    "prompt": "电商主图四张:正面 / 侧面 / 细节 / 场景应用",
    "sequential_image_generation": "auto",
    "sequential_image_generation_options": {"max_images": 4},
    "resolution": "4K",
    "aspect_ratio": "1:1"
  }'
```

## 通用参数

<ParamField body="model" type="string" required>
  从 [可用模型](#可用模型) 列表中选一个
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作类型,可选值:
  - `generate` — 文生图(默认)
  - `image2image` — 图生图(需配合 `image_urls`)
  - `edit` — 图像编辑(局部重绘 / 光影修改,**仅 4.0 / 4.5**)
  - `reference` — 多图参考融合(**仅 4.0 / 4.5**)
  - `group` — 组图生成(全部支持)
</ParamField>

<ParamField body="prompt" type="string" required>
  图像描述文本,支持中英文。Seedream 系列对复杂排版与小字渲染准确率 ~94%
</ParamField>

<ParamField body="n" type="integer" default="1">
  生成数量(单次返回多张)
</ParamField>

<ParamField body="seed" type="integer" default="-1">
  随机种子,`-1` 为随机;固定值可复现相似结果
</ParamField>

<ParamField body="aspect_ratio" type="string" default="1:1">
  画面宽高比,可选值:
  - `1:1` — 正方形
  - `4:3` / `3:4` — 横/竖版标准
  - `16:9` / `9:16` — 横/竖版宽屏
  - `3:2` / `2:3` — 横/竖版相机比
  - `21:9` — 超宽屏(全部支持)
  - `9:21` — 超长竖图(仅 4.0 / 4.5)
  - `auto` — 智能选择(跟随参考图比例)
</ParamField>

<ParamField body="resolution" type="string" default="2K">
  输出分辨率,**取决于模型**:
  - 4.0:`1K` / `2K` / `4K`
  - 4.5:`2K` / `4K`
  - 5.0-lite:`2K` / `3K`
</ParamField>

<ParamField body="image_urls" type="string[]">
  参考图片 URL 数组,**张数取决于模型**:
  - 4.0:2-10 张(加最终输出合计 ≤15)
  - 4.5:最多 14 张
  - 5.0-lite:最多 14 张
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook 回调地址,任务终态时调用。详见 [回调机制](/cn/docs/sync-async#异步任务-webhook)
</ParamField>

## 资源限制

| 项目 | 限制 |
|---|---|
| 参考图片(4.0) | 2-10 张,加最终输出合计 ≤15 张 |
| 参考图片(4.5 / 5.0-lite) | 最多 14 张 |
| 单张图片大小 | ≤ 30MB,支持 JPG/PNG/WEBP |
| 输出分辨率 | 详见各 Tab(最高 4K) |
| 输出文件 | JPG(默认)/ PNG(5.0-lite 可选),链接 24 小时有效 |
