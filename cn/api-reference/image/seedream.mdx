## 可用模型

| 模型 ID | 说明 |
|---------|------|
| `seedream-5.0` | Seedream 5.0，支持组图、联网搜索、流式输出 |
| `seedream-4.5` | Seedream 4.5，支持组图、4K 分辨率 |
| `seedream-4.0` | Seedream 4.0，支持组图、1K~4K 分辨率 |

## 统一参数

<ParamField body="model" type="string" required>
  模型组名称：`seedream-5.0`、`seedream-4.5`、`seedream-4.0`
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作类型
</ParamField>

<ParamField body="prompt" type="string" required>
  文本提示词，支持中英文，建议不超过 300 个汉字
</ParamField>

<ParamField body="image_urls" type="string[]">
  参考图片 URL 数组，最多 14 张，每张不超过 10MB
</ParamField>

<ParamField body="size" type="string" default="1:1">
  宽高比：`1:1`、`2:3`、`3:2`、`3:4`、`4:3`、`9:16`、`16:9`、`21:9`
</ParamField>

<ParamField body="resolution" type="string" default="2K">
  分辨率，可选值取决于模型，见各模型 Tab
</ParamField>

<ParamField body="n" type="integer" default="1">
  生成数量，大于 1 时启用组图模式
</ParamField>

<ParamField body="watermark" type="boolean" default="true">
  是否添加水印
</ParamField>

<ParamField body="callback_url" type="string">
  任务完成时的 Webhook 回调地址
</ParamField>

<ParamField body="extra" type="object">
  透传给上游的额外参数，详见各模型 Tab
</ParamField>

## 生成模式

| 模式 | 参数 | 说明 |
|------|------|------|
| 文生图 | `prompt` | 纯文本生成 |
| 单图生图 | `prompt` + `image_urls`(1张) | 参考图 + 文本 |
| 多图生图 | `prompt` + `image_urls`(2-14张) | 多张参考图融合 |
| 组图 | `n > 1` | 生成一组关联图片 |

<Note>
组图模式下，输入参考图数量 + 生成图片数量 ≤ 15。
</Note>
