异步处理模式 — 提交请求后返回任务 ID，通过 [查询任务状态](/cn/api-reference/task/status) 接口获取生成结果。

<RequestExample>
```bash cURL
curl -X POST https://ai.kaiho.cc/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "flux-kontext-pro",
    "prompt": "一只橘猫趴在窗台上晒太阳，水彩画风格",
    "size": "16:9"
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    base_url="https://ai.kaiho.cc/v1",
    api_key="YOUR_API_KEY"
)

response = client.images.generate(
    model="flux-kontext-pro",
    prompt="一只橘猫趴在窗台上晒太阳，水彩画风格",
    size="16:9"
)

print(response.data[0])
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":  "flux-kontext-pro",
        "prompt": "一只橘猫趴在窗台上晒太阳，水彩画风格",
        "size":   "16:9",
    }

    body, _ := json.Marshal(payload)
    req, _ := http.NewRequest("POST", "https://ai.kaiho.cc/v1/tasks", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    resp, _ := http.DefaultClient.Do(req)
    defer resp.Body.Close()
    result, _ := io.ReadAll(resp.Body)
    fmt.Println(string(result))
}
```

```java Java
import java.net.http.*;
import java.net.URI;

public class Main {
    public static void main(String[] args) throws Exception {
        String payload = """
        {
          "model": "flux-kontext-pro",
          "prompt": "一只橘猫趴在窗台上晒太阳，水彩画风格",
          "size": "16:9"
        }
        """;

        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://ai.kaiho.cc/v1/tasks"))
            .header("Authorization", "Bearer YOUR_API_KEY")
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(payload))
            .build();

        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString());
        System.out.println(response.body());
    }
}
```

## 可用模型

| 模型 ID | 说明 |
|---------|------|
| `flux-kontext-pro` | FLUX Kontext Pro，擅长图像编辑和风格迁移 |
| `flux-kontext-max` | FLUX Kontext Max，增强版 |
| `flux-2-pro` | FLUX 2.0 Pro，通用图像生成 |
| `flux-2-dev` | FLUX 2.0 Dev，开发版 |

## 通用参数

<ParamField body="model" type="string" required>
  模型 ID，见上方模型列表
</ParamField>

<ParamField body="prompt" type="string" required>
  图像描述文本，最多 1000 个字符
</ParamField>

<ParamField body="image_urls" type="array">
  参考图片 URL 数组
</ParamField>

## 各模型差异参数

<Tabs>
  <Tab title="FLUX Kontext">
    适用于 `flux-kontext-pro` 和 `flux-kontext-max`

    <ParamField body="size" type="string" default="match_input_image">
      图像尺寸

      支持：`match_input_image`、`auto`、`1:1`、`4:3`、`3:4`、`16:9`、`9:16`、`3:2`、`2:3`、`21:9`、`9:21`
    </ParamField>

    <ParamField body="response_format" type="string" default="png">
      输出格式：`png`、`jpg`
    </ParamField>

    <ParamField body="safety_tolerance" type="integer" default="2">
      内容安全等级，范围 0-6，值越高越宽松
    </ParamField>

    <ParamField body="prompt_upsampling" type="boolean" default="false">
      启用 prompt 优化增强
    </ParamField>

    <Note>
    Kontext 系列侧重**图像编辑**，`image_urls` 最多传 1 张参考图。
    </Note>
  </Tab>

  <Tab title="FLUX 2.0">
    适用于 `flux-2-pro` 和 `flux-2-dev`

    <ParamField body="size" type="string" default="1:1">
      图像尺寸比例

      支持：`1:1`、`2:3`、`3:2`、`3:4`、`4:3`、`4:5`、`5:4`、`9:16`、`16:9`、`21:9`
    </ParamField>

    <Note>
    FLUX 2.0 系列侧重**通用图像生成**，`image_urls` 可用于图生图参考。
    </Note>
  </Tab>
</Tabs>

## 响应

返回任务 ID，使用 [查询任务状态](/cn/api-reference/task/status) 获取最终图像结果。

<ResponseField name="code" type="string">
  状态码，成功为 `"success"`
</ResponseField>

<ResponseField name="data" type="string">
  任务 ID，格式为 `task-wave{timestamp}{userId}{random}`
</ResponseField>
