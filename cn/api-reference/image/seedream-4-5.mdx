<Note>
本接口采用异步工作模式。成功提交请求后，系统将返回一个任务 ID (`task_id`)。您需要使用该 ID 调用查询接口获取生成结果。生成的图片链接有效期为 24 小时，请尽快下载保存。

[查询任务状态](/cn/api-reference/task/get-status)
</Note>

	fmt.Println(result)

## 认证方式

<ParamField header="Authorization" type="string" required>
  API 密钥，格式为 `Bearer YOUR_API_KEY`。您可以在 [控制台](https://ai.kaiho.cc/dashboard/keys) 获取您的 API Key。
</ParamField>

## 请求参数

<ParamField body="model" type="string" required>
  模型 ID，固定为 `seedream-4.5`。
</ParamField>

<ParamField body="prompt" type="string" required>
  图像生成的文本描述，最大长度 1000 字符。
</ParamField>

<ParamField body="size" type="string" default="1:1">
  生成图像的比例。
  可选值：`1:1`, `2:3`, `3:2`, `3:4`, `4:3`, `9:16`, `16:9`
</ParamField>

<ParamField body="image_urls" type="array">
  参考图像 URL 数组，支持图生图或图像编辑。
</ParamField>

## 响应参数

<ResponseField name="code" type="string">
  响应状态码

  * `success` - 请求成功，任务已提交
</ResponseField>

<ResponseField name="data" type="string">
  任务 ID，格式为 `task-wave{timestamp}{userId}{random}`

  用于通过 [查询任务状态](/cn/api-reference/task/get-status) 接口获取生成结果
</ResponseField>

## 注意事项

- 任务提交成功后并不代表生成完成，请保存 `task_id` 以查询最终结果。
- 生成的图片链接在 24 小时后失效，请务必及时下载图片。
- 模型支持图生图模式，只需在 `image_urls` 中提供有效的图片链接。
