<Note>
本接口采用异步工作模式。成功提交请求后，系统将返回一个任务 ID (`task_id`)。您需要使用该 ID 调用查询接口获取生成结果。生成的图片链接有效期为 24 小时，请尽快下载保存。

[查询任务状态](/cn/api-reference/task/get-status)
</Note>

## 认证方式

<ParamField header="Authorization" type="string" required>
  API 密钥，格式为 `Bearer YOUR_API_KEY`。您可以在 [控制台](https://qingbo.dev/dashboard/keys) 获取您的 API Key。
</ParamField>

## 请求参数

<ParamField body="model" type="string" required>
  模型 ID，固定为 `gemini-3.1-flash-image-preview`。
</ParamField>

<ParamField body="prompt" type="string" required>
  图像生成的文本描述，最大长度 1000 字符。
</ParamField>

<ParamField body="size" type="string" default="1:1">
  生成图像的比例。支持常规及极端比例。
  可选值：`1:1`, `2:3`, `3:2`, `3:4`, `4:3`, `4:5`, `5:4`, `9:16`, `16:9`, `21:9`, `1:4`, `4:1`, `1:8`, `8:1`
</ParamField>

<ParamField body="resolution" type="string" default="1K">
  生成分辨率。可选值：`1K`, `2K`, `4K`。
</ParamField>

<ParamField body="image_urls" type="array">
  参考图像 URL 数组，最大支持 14 张图片，用于风格或角色一致性。
</ParamField>

<ParamField body="google_search_enhancement" type="boolean" default="false">
  是否启用谷歌搜索增强，以提高生成内容的真实性。
</ParamField>

<ParamField body="n" type="number" default="1">
  生成数量，目前固定为 1。
</ParamField>

## 响应参数

<ResponseField name="code" type="string">
  响应状态码

  * `success` - 请求成功，任务已提交
</ResponseField>

<ResponseField name="data" type="string">
  任务 ID，格式为 `task-wave{timestamp}{userId}{random}`

  用于通过 [查询任务状态](/cn/api-reference/task/get-status) 接口获取生成结果
</ResponseField>

## 注意事项

- 支持最高 4K 分辨率的高清图像输出。
- 模型支持极端宽高比（如 1:8 或 8:1），适用于全景图或长条形设计。
- 通过 `image_urls` 传入多达 14 张参考图，可以实现极佳的角色或风格一致性。
