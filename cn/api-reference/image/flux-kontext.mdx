<Note>
  本接口采用异步模式。提交成功后将返回 `task_id`，您需要通过任务状态接口获取生成结果。生成结果的下载链接有效期为 24 小时。
</Note>

```bash cURL theme={null}

     -H "Content-Type: application/json" \

       "prompt": "将头发颜色改为蓝色",

       "size": "match_input_image"
     }'
```

```go Go theme={null}
package main

import (
	"bytes"
	"encoding/json"
	"fmt"
	"net/http"
)

func main() {
	url := "https://qingbo.dev/v1/tasks"
	payload := map[string]interface{}{
		"model":      "flux-kontext-pro",
		"prompt":     "将头发颜色改为蓝色",
		"image_urls": []string{"https://example.com/image.jpg"},
		"size":       "match_input_image",
	}
	jsonPayload, _ := json.Marshal(payload)

	req, _ := http.NewRequest("POST", url, bytes.NewBuffer(jsonPayload))
	req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
	req.Header.Set("Content-Type", "application/json")

	client := &http.Client{}
	resp, _ := client.Do(req)
	defer resp.Body.Close()

	var result map[string]interface{}
	json.NewDecoder(resp.Body).Decode(&result)
	fmt.Println(result)
}
```

## 认证方式

<ParamField header="Authorization" type="string" placeholder="Bearer YOUR_API_KEY" required>
  在 [API Key 页面](https://qingbo.dev/dashboard/keys) 获取您的密钥，并以 `Bearer YOUR_API_KEY` 的格式放入请求头中。
</ParamField>

## 请求参数

<ParamField body="model" type="string" required>
  使用的模型 ID。可选值：`flux-kontext-pro`, `flux-kontext-max`。
</ParamField>

<ParamField body="prompt" type="string" required>
  图像生成的文本提示词。
</ParamField>

<ParamField body="image_urls" type="array">
  包含图片 URL 的数组。最多支持 1 张图片。必须是可公开访问的 URL，不支持 base64。
</ParamField>

<ParamField body="size" type="string" default="match_input_image">
  生成图像的比例或尺寸。可选值：`match_input_image` (存在输入图时的默认值), `auto`, `1:1`, `4:3`, `3:4`, `16:9`, `9:16`, `3:2`, `2:3`, `21:9`, `9:21`。
</ParamField>

<ParamField body="response_format" type="string" default="png">
  返回结果的图像格式。可选值：`png`, `jpg`。
</ParamField>

<ParamField body="safety_tolerance" type="integer" default="2">
  内容安全容差等级，范围 0-6。
</ParamField>

<ParamField body="prompt_upsampling" type="boolean" default="false">
  是否开启提示词优化。
</ParamField>

## 响应参数

<ResponseField name="code" type="string">
  响应状态码

  * `success` - 请求成功，任务已提交
</ResponseField>

<ResponseField name="data" type="string">
  任务 ID，格式为 `task-wave{timestamp}{userId}{random}`

  用于通过 [查询任务状态](/cn/api-reference/task/get-status) 接口获取生成结果
</ResponseField>

## 注意事项

- 接口返回 `task_id` 后，请前往 [查询任务状态](/cn/api-reference/task/get-status) 页面了解如何获取最终图像。
- 图像编辑功能需要提供 `image_urls`。
- 生成的下载链接具有时效性，请及时转存。
