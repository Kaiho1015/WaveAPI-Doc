## 请求参数

<ParamField body="model" type="string" default="gpt-4o-image" required>
  图像生成模型名称

  示例：`"gpt-4o-image"`
</ParamField>

<ParamField body="prompt" type="string" required>
  图像生成的文字描述

  最大 1000 个字符
</ParamField>

<ParamField body="size" type="string">
  图像生成尺寸

  支持的比例：

  * `1:1` - 正方形
  * `2:3` - 竖版
  * `3:2` - 横版
</ParamField>

<ParamField body="n" type="integer">
  生成图像的数量

  支持 1、2、4，费用将按数量预扣

  默认值：1

  **⚠️ 注意：** 必须输入纯数字（如 `1`），不要使用引号包裹
</ParamField>

<ParamField body="image_urls" type="array<url>">
  参考图像 URL 列表，用于图生图或图片编辑

  <Expandable title="详细说明">
    数组中的每个元素为字符串，支持两种格式：

    **1. 完整图像 URL**

    * 公开可访问的图像 URL（http:// 或 https://）
    * 示例：`"https://example.com/image.jpg"`

    **2. Base64 编码格式**

    * **必须使用完整的 Data URI 格式**
    * 格式：`data:image/{format};base64,{base64data}`
    * 支持的图像格式：jpeg、png、webp
    * 示例：`"data:image/jpeg;base64,/9j/4AAQSkZJRgABAQEAYABg..."`
    * ⚠️ 注意：必须包含 `data:image/jpeg;base64,` 前缀

    **限制：**

    * 单张图片最大 10MB
    * 支持格式：.jpeg、.jpg、.png、.webp
  </Expandable>

  **限制：** 最多 5 张图片
</ParamField>

<ParamField body="mask_url" type="string">
  遮罩图像 URL

  * 必须为 PNG 格式
  * 尺寸必须与参考图像一致
  * 最大 4MB
</ParamField>

## 响应参数

<ResponseField name="code" type="string">
  响应状态码

  * `success` - 请求成功，任务已提交
</ResponseField>

<ResponseField name="data" type="string">
  任务 ID，格式为 `task-wave{timestamp}{userId}{random}`

  用于通过 [查询任务状态](/cn/api-reference/task/get-status) 接口获取生成结果
</ResponseField>

## 注意事项

1. **异步处理：** 任务提交后异步处理，请通过 [查询任务状态](/cn/api-reference/task/get-status) 获取生成结果
2. **链接有效期：** 生成的图片链接有效期为 24 小时，请及时下载保存
3. **图片 URL 要求：** 输入的参考图片必须为公开可访问的 URL 或 Base64 格式
4. **费用说明：** 当 `n` 大于 1 时，费用将按数量预扣
