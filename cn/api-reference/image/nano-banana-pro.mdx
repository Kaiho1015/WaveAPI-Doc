## 请求参数

<ParamField body="model" type="string" default="gemini-3-pro-image-preview" required>
  图像生成模型名称

  示例：`"gemini-3-pro-image-preview"`
</ParamField>

<ParamField body="prompt" type="string" required>
  图像生成的文字描述

  最大 1000 个字符
</ParamField>

<ParamField body="size" type="string">
  图像生成尺寸

  支持的比例：

  * `1:1` - 正方形
  * `2:3` - 竖版
  * `3:2` - 横版
  * `3:4` - 竖版 3:4
  * `4:3` - 横版 4:3
  * `4:5` - 竖版 4:5
  * `5:4` - 横版 5:4
  * `9:16` - 竖版全屏
  * `16:9` - 横版宽屏
  * `21:9` - 超宽屏
</ParamField>

<ParamField body="resolution" type="string" default="1K">
  输出图像分辨率

  支持的值：

  * `1K` - 1K 分辨率（默认）
</ParamField>

<ParamField body="n" type="integer">
  生成图像的数量

  固定为 1

  **⚠️ 注意：** 必须输入纯数字（如 `1`），不要使用引号包裹
</ParamField>

<ParamField body="image_urls" type="array<url>">
  参考图像 URL 列表，用于图生图或图片编辑

  * 最多 14 张图片
  * 每张图片不超过 10MB
  * 支持格式：.jpeg、.jpg、.png、.webp
</ParamField>

## 响应参数

<ResponseField name="code" type="string">
  响应状态码

  * `success` - 请求成功，任务已提交
</ResponseField>

<ResponseField name="data" type="string">
  任务 ID，格式为 `task-wave{timestamp}{userId}{random}`

  用于通过 [查询任务状态](/cn/api-reference/task/get-status) 接口获取生成结果
</ResponseField>

## 注意事项

1. **异步处理：** 任务提交后异步处理，请通过 [查询任务状态](/cn/api-reference/task/get-status) 获取生成结果
2. **链接有效期：** 生成的图片链接有效期为 24 小时，请及时下载保存
3. **专业品质：** Nano Banana Pro 基于 Gemini 3 Pro，生成质量更高，适合专业级图像创作
4. **参考图片：** 支持最多 14 张参考图片，可用于风格/角色一致性控制
