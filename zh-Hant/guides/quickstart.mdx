## 前置要求

- 一個 WaveAPI 帳戶（[註冊](https://ai.kaiho.cc/login)）
- API 金鑰
- 基礎的程式設計知識

## 取得 API 金鑰

<Steps>
  <Step title="登入控制台">
    存取 [Dashboard](https://ai.kaiho.cc/dashboard/keys) 並登入
  </Step>
  <Step title="建立金鑰">
    在金鑰管理頁面點擊「建立新金鑰」
  </Step>
  <Step title="儲存金鑰">
    複製並安全儲存你的 API 金鑰，金鑰只顯示一次
  </Step>
</Steps>

<Warning>
請勿在前端程式碼或公開儲存庫中暴露 API Key。
</Warning>

## 傳送第一個請求

WaveAPI 完全相容 OpenAI SDK，只需替換 `base_url` 即可使用。

<CodeGroup>

```python Python
from openai import OpenAI

client = OpenAI(
    base_url="https://ai.kaiho.cc/v1",
    api_key="YOUR_API_KEY"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "user", "content": "你好，介绍一下你自己"}
    ]
)

print(response.choices[0].message.content)
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'https://ai.kaiho.cc/v1',
  apiKey: 'YOUR_API_KEY'
});

const response = await client.chat.completions.create({
  model: 'gpt-4o',
  messages: [
    { role: 'user', content: '你好，介绍一下你自己' }
  ]
});

console.log(response.choices[0].message.content);
```

```bash cURL
curl https://ai.kaiho.cc/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "gpt-4o",
    "messages": [
      {"role": "user", "content": "你好，介绍一下你自己"}
    ]
  }'
```

</CodeGroup>
