非同步處理模式 — 提交請求後回傳任務 ID，透過 [查詢任務狀態](/zh-Hant/api-reference/task/status) 介面取得生成結果。

<RequestExample>
```bash cURL
curl -X POST https://qingbo.dev/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "flux-kontext-pro",
    "prompt": "一只橘猫趴在窗台上晒太阳，水彩画风格",
    "size": "16:9"
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    base_url="https://qingbo.dev/v1",
    api_key="YOUR_API_KEY"
)

response = client.images.generate(
    model="flux-kontext-pro",
    prompt="一只橘猫趴在窗台上晒太阳，水彩画风格",
    size="16:9"
)

print(response.data[0])
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":  "flux-kontext-pro",
        "prompt": "一只橘猫趴在窗台上晒太阳，水彩画风格",
        "size":   "16:9",
    }

    body, _ := json.Marshal(payload)
    req, _ := http.NewRequest("POST", "https://qingbo.dev/v1/tasks", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    resp, _ := http.DefaultClient.Do(req)
    defer resp.Body.Close()
    result, _ := io.ReadAll(resp.Body)
    fmt.Println(string(result))
}
```

```java Java
import java.net.http.*;
import java.net.URI;

public class Main {
    public static void main(String[] args) throws Exception {
        String payload = """
        {
          "model": "flux-kontext-pro",
          "prompt": "一只橘猫趴在窗台上晒太阳，水彩画风格",
          "size": "16:9"
        }
        """;

        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://qingbo.dev/v1/tasks"))
            .header("Authorization", "Bearer YOUR_API_KEY")
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(payload))
            .build();

        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString());
        System.out.println(response.body());
    }
}
```

## 可用模型

| 模型 ID | 說明 |
|---------|------|
| `flux-kontext-pro` | FLUX Kontext Pro，擅長圖像編輯和風格遷移 |
| `flux-kontext-max` | FLUX Kontext Max，增強版 |
| `flux-2-pro` | FLUX 2.0 Pro，通用圖像生成 |
| `flux-2-dev` | FLUX 2.0 Dev，開發版 |

## 通用參數

<ParamField body="model" type="string" required>
  模型 ID，見上方模型列表
</ParamField>

<ParamField body="prompt" type="string" required>
  圖像描述文字，最多 1000 個字元
</ParamField>

<ParamField body="image_urls" type="array">
  參考圖片 URL 陣列
</ParamField>

## 各模型差異參數

<Tabs>
  <Tab title="FLUX Kontext">
    適用於 `flux-kontext-pro` 和 `flux-kontext-max`

    <ParamField body="size" type="string" default="match_input_image">
      圖像尺寸

      支援：`match_input_image`、`auto`、`1:1`、`4:3`、`3:4`、`16:9`、`9:16`、`3:2`、`2:3`、`21:9`、`9:21`
    </ParamField>

    <ParamField body="response_format" type="string" default="png">
      輸出格式：`png`、`jpg`
    </ParamField>

    <ParamField body="safety_tolerance" type="integer" default="2">
      內容安全等級，範圍 0-6，值越高越寬鬆
    </ParamField>

    <ParamField body="prompt_upsampling" type="boolean" default="false">
      啟用 prompt 最佳化增強
    </ParamField>

    <Note>
    Kontext 系列側重**圖像編輯**，`image_urls` 最多傳 1 張參考圖。
    </Note>
  </Tab>

  <Tab title="FLUX 2.0">
    適用於 `flux-2-pro` 和 `flux-2-dev`

    <ParamField body="size" type="string" default="1:1">
      圖像尺寸比例

      支援：`1:1`、`2:3`、`3:2`、`3:4`、`4:3`、`4:5`、`5:4`、`9:16`、`16:9`、`21:9`
    </ParamField>

    <Note>
    FLUX 2.0 系列側重**通用圖像生成**，`image_urls` 可用於圖生圖參考。
    </Note>
  </Tab>
</Tabs>

## 回應

回傳任務 ID，使用 [查詢任務狀態](/zh-Hant/api-reference/task/status) 取得最終圖像結果。

<ResponseField name="code" type="string">
  狀態碼，成功為 `"success"`
</ResponseField>

<ResponseField name="data" type="string">
  任務 ID，格式為 `task-wave{timestamp}{userId}{random}`
</ResponseField>
