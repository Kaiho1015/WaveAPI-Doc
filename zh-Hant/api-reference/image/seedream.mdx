非同步處理模式 — 提交請求後回傳任務 ID，透過 [查詢任務狀態](/zh-Hant/api-reference/task/status) 介面取得生成結果。圖片連結有效期 24 小時，請及時儲存。

<RequestExample>
```bash cURL
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "doubao-seedream-5.0",
    "action": "generate",
    "prompt": "一只柴犬穿宇航服站在月球上",
    "size": "16:9",
    "resolution": "2K",
    "watermark": false
  }'
```

## 可用模型

| 模型 ID | 說明 |
|---------|------|
| `doubao-seedream-5.0` | Seedream 5.0，支援組圖、聯網搜尋、串流輸出 |
| `doubao-seedream-4.5` | Seedream 4.5，支援組圖、4K 解析度 |
| `doubao-seedream-4.0` | Seedream 4.0，支援組圖、1K~4K 解析度 |

## 統一參數

<ParamField body="model" type="string" required>
  模型組名稱：`doubao-seedream-5.0`、`doubao-seedream-4.5`、`doubao-seedream-4.0`
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作類型
</ParamField>

<ParamField body="prompt" type="string" required>
  文字提示詞，支援中英文，建議不超過 300 個漢字
</ParamField>

<ParamField body="image_urls" type="string[]">
  參考圖片 URL 陣列，最多 14 張，每張不超過 10MB
</ParamField>

<ParamField body="size" type="string" default="1:1">
  寬高比：`1:1`、`2:3`、`3:2`、`3:4`、`4:3`、`9:16`、`16:9`、`21:9`
</ParamField>

<ParamField body="resolution" type="string" default="2K">
  解析度，可選值取決於模型，見各模型 Tab
</ParamField>

<ParamField body="n" type="integer" default="1">
  生成數量，大於 1 時啟用組圖模式
</ParamField>

<ParamField body="watermark" type="boolean" default="true">
  是否新增浮水印
</ParamField>

<ParamField body="callback_url" type="string">
  任務完成時的 Webhook 回呼位址
</ParamField>

<ParamField body="extra" type="object">
  透傳給上游的額外參數，詳見各模型 Tab
</ParamField>

## 生成模式

| 模式 | 參數 | 說明 |
|------|------|------|
| 文生圖 | `prompt` | 純文字生成 |
| 單圖生圖 | `prompt` + `image_urls`(1張) | 參考圖 + 文字 |
| 多圖生圖 | `prompt` + `image_urls`(2-14張) | 多張參考圖融合 |
| 組圖 | `n > 1` | 生成一組關聯圖片 |

<Note>
組圖模式下，輸入參考圖數量 + 生成圖片數量 ≤ 15。
</Note>
