## 認證方式

## 請求參數

<ParamField body="model" type="string" default="gpt-4o-image" required>
  圖像生成模型名稱

  範例：`"gpt-4o-image"`
</ParamField>

<ParamField body="prompt" type="string" required>
  圖像生成的文字描述

  最大 1000 個字元
</ParamField>

<ParamField body="size" type="string">
  圖像生成尺寸

  支援的比例：

  * `1:1` - 正方形
  * `2:3` - 直版
  * `3:2` - 橫版
</ParamField>

<ParamField body="n" type="integer">
  生成圖像的數量

  支援 1、2、4，費用將按數量預扣

  預設值：1

  **注意：** 必須輸入純數字（如 `1`），不要使用引號包裹
</ParamField>

<ParamField body="image_urls" type="array<url>">
  參考圖像 URL 列表，用於圖生圖或圖片編輯

  <Expandable title="詳細說明">
    陣列中的每個元素為字串，支援兩種格式：

    **1. 完整圖像 URL**

    * 公開可存取的圖像 URL（http:// 或 https://）
    * 範例：`"https://example.com/image.jpg"`

    **2. Base64 編碼格式**

    * **必須使用完整的 Data URI 格式**
    * 格式：`data:image/{format};base64,{base64data}`
    * 支援的圖像格式：jpeg、png、webp
    * 範例：`"data:image/jpeg;base64,/9j/4AAQSkZJRgABAQEAYABg..."`
    * 注意：必須包含 `data:image/jpeg;base64,` 前綴

    **限制：**

    * 單張圖片最大 10MB
    * 支援格式：.jpeg、.jpg、.png、.webp
  </Expandable>

  **限制：** 最多 5 張圖片
</ParamField>

<ParamField body="mask_url" type="string">
  遮罩圖像 URL

  * 必須為 PNG 格式
  * 尺寸必須與參考圖像一致
  * 最大 4MB
</ParamField>

## 回應參數

<ResponseField name="code" type="string">
  回應狀態碼

  * `success` - 請求成功，任務已提交
</ResponseField>

<ResponseField name="data" type="string">
  任務 ID，格式為 `task-wave{timestamp}{userId}{random}`

  用於透過 [查詢任務狀態](/zh-Hant/api-reference/task/get-status) 介面取得生成結果
</ResponseField>

## 注意事項

1. **非同步處理：** 任務提交後非同步處理，請透過 [查詢任務狀態](/zh-Hant/api-reference/task/get-status) 取得生成結果
2. **連結有效期：** 生成的圖片連結有效期為 24 小時，請及時下載儲存
3. **圖片 URL 要求：** 輸入的參考圖片必須為公開可存取的 URL 或 Base64 格式
4. **費用說明：** 當 `n` 大於 1 時，費用將按數量預扣
