- 完全相容 Claude Messages API 格式
- 支援多輪對話和單次查詢
- 支援文字、圖像等多模態內容

    base_url="https://www.qingbo.dev/v1",

)

    model="claude-sonnet-4-5-20250929",
    max_tokens=1024,

    messages=[

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":      "claude-sonnet-4-5-20250929",
        "max_tokens": 1024,
        "system":     "你是一个专业的AI助手。",
        "messages": []map[string]string{
            {"role": "user", "content": "解释一下冒泡排序算法。"},
        },
    }

    body, _ := json.Marshal(payload)
    req, _ := http.NewRequest("POST", "https://www.qingbo.dev/v1/messages", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")
    req.Header.Set("anthropic-version", "2023-06-01")

    resp, err := http.DefaultClient.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    result, _ := io.ReadAll(resp.Body)
    fmt.Println(string(result))
}
```

import java.net.http.*;

## Authorizations

<ParamField header="x-api-key" type="string" required>
  API 金鑰用於身份驗證

  存取 [API Key 管理頁面](https://qingbo.dev/dashboard/keys) 取得您的 API Key

  在請求標頭中新增：

  ```
  x-api-key: YOUR_API_KEY
  ```
</ParamField>

<ParamField header="anthropic-version" type="string" required>
  API 版本號

  指定要使用的 Claude API 版本

  範例：`2023-06-01`
</ParamField>

## Body

<ParamField body="model" type="string" required>
  模型名稱

  * `claude-haiku-4-5-20251001` - Claude 4.5 快速回應版本
  * `claude-sonnet-4-5-20250929` - Claude 4.5 平衡版本
  * `claude-opus-4-1-20250805` - 最強大的 Claude 4.1 旗艦模型
  * `claude-opus-4-1-20250805-thinking` - Claude 4.1 Opus 深度思考版
  * `claude-sonnet-4-5-20250929-thinking` - Claude 4.5 Sonnet 深度思考版
</ParamField>

<ParamField body="messages" type="array" required>
  訊息列表，支援交替的 `user` 和 `assistant` 角色

  <Expandable title="訊息物件屬性">
    <ParamField body="role" type="string" required>
      角色：`user`（使用者輸入）或 `assistant`（模型回覆，用於多輪對話或預填充）
    </ParamField>

    <ParamField body="content" type="string" required>
      訊息內容，支援字串或內容區塊陣列（多模態）
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="max_tokens" type="integer" required>
  最大生成 token 數

  生成停止前的最大 token 數量。模型可能會在達到此限制前停止。

  不同模型有不同的最大值，請參考模型文件。

  最小值：1
</ParamField>

<ParamField body="system" type="string | array">
  系統提示詞

  系統提示詞用於設定 Claude 的角色、個性、目標和指令。

  **字串格式：**

  ```json
  {
    "system": "你是一位专业的Python编程导师"
  }
  ```

  **結構化格式：**

  ```json
  {
    "system": 
    [
      {
        "type": "text",
        "text": "你是一位专业的Python编程导师"
      }
    ]
  }
  ```
</ParamField>

<ParamField body="temperature" type="number">
  溫度參數，範圍 0-1

  控制輸出的隨機性：

  * 低值（如 0.2）：更確定、更保守
  * 高值（如 0.8）：更隨機、更有創意

  預設值：1.0
</ParamField>

<ParamField body="top_p" type="number">
  核取樣參數，範圍 0-1

  使用 nucleus sampling。建議使用 `temperature` 或 `top_p` 其中之一，不要同時使用。

  預設值：1.0
</ParamField>

<ParamField body="top_k" type="integer">
  Top-K 取樣

  只從機率最高的 K 個選項中取樣，用於移除「長尾」低機率回應。

  建議僅在進階用例中使用。
</ParamField>

<ParamField body="stream" type="boolean">
  是否啟用串流輸出

  * `true`: 透過伺服器傳送事件 (SSE) 逐步回傳回應
  * `false`: 一次性回傳完整回應

  預設值：false
</ParamField>

<ParamField body="stop_sequences" type="array">
  停止序列

  自訂文字序列，遇到時停止生成。最多 4 個序列，每個最長 32 個 token。
</ParamField>

<ParamField body="metadata" type="object">
  中繼資料

  用於追蹤或標識請求的物件。
</ParamField>

## Response

<ResponseField name="id" type="string">
  訊息的唯一識別碼
</ResponseField>

<ResponseField name="type" type="string">
  物件類型，固定為 `message`
</ResponseField>

<ResponseField name="role" type="string">
  角色類型，固定為 `assistant`
</ResponseField>

<ResponseField name="content" type="array">
  訊息內容陣列

  <Expandable title="屬性">
    <ResponseField name="type" type="string">
      內容類型

      * `text` - 文字內容
      * `tool_use` - 工具使用（如果啟用了工具）
    </ResponseField>

    <ResponseField name="text" type="string">
      文字內容（當 type 為 text 時）
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="model" type="string">
  實際使用的模型名稱
</ResponseField>

<ResponseField name="stop_reason" type="string">
  停止原因

  可能的值：

  * `end_turn` - 自然結束
  * `max_tokens` - 達到最大 token 限制
  * `stop_sequence` - 遇到停止序列
  * `tool_use` - 工具使用
</ResponseField>

<ResponseField name="stop_sequence" type="string">
  觸發停止的序列（如果有）
</ResponseField>

<ResponseField name="usage" type="object">
  token 使用統計

  <Expandable title="屬性">
    <ResponseField name="input_tokens" type="integer">
      輸入 token 數
    </ResponseField>

    <ResponseField name="output_tokens" type="integer">
      輸出 token 數
    </ResponseField>
  </Expandable>
</ResponseField>

## 使用範例

### 單次對話

```json theme={null}
{
  "model": "claude-sonnet-4-5-20250929",
  "max_tokens": 1024,
  "messages": [
    {"role": "user", "content": "解释一下量子计算"}
  ]
}
```

### 多輪對話

```json theme={null}
{
  "model": "claude-sonnet-4-5-20250929",
  "max_tokens": 1024,
  "messages": [
    {"role": "user", "content": "你好"},
    {"role": "assistant", "content": "你好！我是Claude。"},
    {"role": "user", "content": "能解释一下AI吗？"}
  ]
}
```

### 使用系統提示詞

```json theme={null}
{
  "model": "claude-sonnet-4-5-20250929",
  "max_tokens": 1024,
  "system": "你是一位经验丰富的数据科学家，专长包括统计分析和机器学习。",
  "messages": [
    {"role": "user", "content": "如何选择合适的机器学习算法？"}
  ]
}
```

### 預填充回應

```json theme={null}
{
  "model": "claude-sonnet-4-5-20250929",
  "max_tokens": 1024,
  "messages": [
    {"role": "user", "content": "列出5个Python最佳实践"},
    {"role": "assistant", "content": "以下是5个Python最佳实践：\n\n1."}
  ]
}
```
