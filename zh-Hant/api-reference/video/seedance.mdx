非同步處理模式 — 提交請求後回傳任務 ID，透過 [查詢任務狀態](/zh-Hant/api-reference/task/status) 介面取得生成結果。影片連結有效期 24 小時，請及時下載。

<RequestExample>
```bash cURL

  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "seedance-2.0",
    "action": "generate",
    "prompt": "一只柴犬穿宇航服在月球上行走，电影感光影",
    "aspect_ratio": "16:9",
    "duration": 5,
    "resolution": "1080p",
    "watermark": false,
    "extra": {
      "generate_audio": true
    }
  }'
```

## 可用模型

| 模型 ID | 說明 |
|---------|------|
| `seedance-2.0` | Seedance 2.0，支援原生音訊、首尾幀、多模態參考、最高 2K |
| `seedance-2.0-fast` | Seedance 2.0 快速版 |
| `seedance-1.5-pro` | Seedance 1.5 專業版 |
| `seedance-1.0-pro-fast` | Seedance 1.0 快速版 |
| `seedance-1.0-pro-quality` | Seedance 1.0 高畫質版 |

## 統一參數

<ParamField body="model" type="string" required>
  模型 ID
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作類型
</ParamField>

<ParamField body="prompt" type="string" required>
  影片描述文字，支援中英文。2.0 支援在 prompt 中用 `@image1`、`@video1`、`@audio1` 引用參考素材
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
  寬高比：`16:9`、`9:16`、`4:3`、`3:4`、`21:9`、`1:1`
</ParamField>

<ParamField body="duration" type="integer" default="5">
  影片時長（秒），可選範圍取決於模型，見各模型 Tab
</ParamField>

<ParamField body="resolution" type="string">
  輸出解析度，可選值取決於模型，見各模型 Tab
</ParamField>

<ParamField body="image_urls" type="string[]">
  參考圖片 URL 陣列，數量限制取決於模型版本
</ParamField>

<ParamField body="first_frame" type="string">
  首幀圖片 URL
</ParamField>

<ParamField body="last_frame" type="string">
  尾幀圖片 URL（需配合 `first_frame`）
</ParamField>

<ParamField body="seed" type="integer" default="-1">
  隨機種子，範圍 [-1, 2^32-1]。`-1` 為隨機。相同種子 + 相同請求可獲得相似結果
</ParamField>

<ParamField body="watermark" type="boolean" default="false">
  是否新增浮水印
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook 回呼位址
</ParamField>

<ParamField body="extra" type="object">
  透傳給上游的額外參數，詳見各模型 Tab
</ParamField>

## 模型特定參數

<Tabs>
  <Tab title="Seedance 2.0">
    **duration 範圍：** 4-15 秒

    **resolution 可選值：** `480p`、`720p`、`1080p`、`2K`

    **size 額外支援：** `adaptive`（自適應，匹配輸入素材寬高比）

    **輸入限制：** 圖片最多 9 張（≤ 30MB）、影片最多 3 個（≤ 50MB，總時長 ≤ 15s）、音訊最多 3 個（MP3/WAV，≤ 15MB，總時長 ≤ 15s）

    <ParamField body="extra.generate_audio" type="boolean" default="false">
      原生音訊影片聯合生成，支援對白口型同步（8+ 語言）
    </ParamField>

    <ParamField body="extra.return_last_frame" type="boolean" default="false">
      回傳末幀圖片 URL，便於影片片段銜接續拍
    </ParamField>

    <ParamField body="extra.tools" type="array">
      模型工具，支援聯網搜尋

      ```json
      "extra": { "tools": [{"type": "web_search"}] }
      ```
    </ParamField>

    <ParamField body="extra.service_tier" type="string">
      服務等級，控制延遲與成本的平衡
    </ParamField>

    <Note>
    2.0 使用首尾幀模式（`first_frame` + `last_frame`）時，不能同時傳 `video_urls` 和 `audio_urls`。
    </Note>
  </Tab>

  <Tab title="Seedance 2.0 Fast">
    **duration 範圍：** 4-15 秒

    **resolution 可選值：** `480p`、`720p`、`1080p`、`2K`

    參數與 Seedance 2.0 相同，處理速度更快。

    <ParamField body="extra.generate_audio" type="boolean" default="false">
      原生音訊影片聯合生成
    </ParamField>

    <ParamField body="extra.return_last_frame" type="boolean" default="false">
      回傳末幀圖片
    </ParamField>
  </Tab>

  <Tab title="Seedance 1.5 Pro">
    **duration 範圍：** 4-12 秒

    **resolution 可選值：** `480p`、`720p`、`1080p`

    **輸入限制：** 圖片最多 2 張（1 張首幀 + 1 張尾幀）

    <ParamField body="extra.audio" type="boolean" default="false">
      是否生成伴隨音訊
    </ParamField>

    <ParamField body="extra.camerafixed" type="boolean" default="false">
      固定機位模式，開啟後攝影機位置不變
    </ParamField>
  </Tab>

  <Tab title="Seedance 1.0 Pro">
    **兩個子版本：** `seedance-1.0-pro-fast`（快速）和 `seedance-1.0-pro-quality`（高畫質）

    **duration 範圍：** 2-12 秒

    **resolution 可選值：** `480p`、`720p`、`1080p`

    **輸入限制：** 圖片最多 1 張

    <Note>
    1.0 Quality 版本支援 `last_frame`（尾幀）和 `reference`（風格參考）角色。使用 reference 角色時不支援 1080p 解析度。
    </Note>
  </Tab>

## 生成模式

| 模式 | 參數 | 支援版本 |
|------|------|---------|
| 文生影片 | `prompt` | 全部 |
| 圖生影片 | `prompt` + `image_urls`(1張) 或 `first_frame` | 全部 |
| 首尾幀 | `first_frame` + `last_frame` | 1.0-quality / 1.5 / 2.0 |
| 多模態參考 | `image_urls` + `video_urls` + `audio_urls` | 僅 2.0 |
