## 概述

WaveAPI 支援多種影片生成模型，統一透過非同步任務介面 `/v1/tasks` 呼叫。影片生成通常耗時較長，請使用輪詢或 Webhook 回呼取得結果。

## 請求端點

```
POST https://www.qingbo.dev/v1/tasks
```

## 通用請求參數

| 參數 | 類型 | 必填 | 說明 |
|------|------|------|------|
| `model` | string | 是 | 模型名稱 |
| `action` | string | 否 | 預設 `"generate"` |
| `prompt` | string | 是 | 影片描述文字 |
| `duration` | integer | 否 | 影片時長（秒） |
| `aspect_ratio` | string | 否 | 寬高比，如 `"16:9"` |
| `resolution` | string | 否 | 解析度，如 `"1080p"` |
| `first_frame` | string | 否 | 首幀圖片 URL |
| `last_frame` | string | 否 | 尾幀圖片 URL |
| `image_urls` | array | 否 | 參考圖片 URL 陣列 |
| `callback_url` | string | 否 | 完成後的回呼位址 |
| `extra` | object | 否 | 模型特定的透傳參數 |

## 回應格式

```json
{
  "task_id": "task-wave1775285160b950328499",
  "model": "veo-3.1-fast",
  "action": "generate",
  "status": "queued",
  "created_at": 1775285160040,
  "progress": 0
}
```

## 生成模式

| 模式 | 參數 | 說明 |
|------|------|------|
| 文生影片 | `prompt` | 純文字描述生成影片 |
| 圖生影片 | `prompt` + `first_frame` 或 `image_urls` | 基於圖片生成影片 |
| 首尾幀 | `first_frame` + `last_frame` | 指定首尾畫面生成過渡影片 |

<Note>
並非所有模型都支援全部模式，請查看各模型文件了解具體支援情況。
</Note>
