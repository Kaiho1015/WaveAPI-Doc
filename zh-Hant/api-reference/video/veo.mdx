非同步處理模式 — 提交請求後回傳任務 ID，透過 [查詢任務狀態](/zh-Hant/api-reference/task/status) 介面取得生成結果。影片連結有效期 24 小時，請及時下載。

<RequestExample>
```bash cURL
curl -X POST https://ai.kaiho.cc/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "veo-3.1",
    "action": "generate",
    "prompt": "A golden retriever running on a beach at sunset, cinematic lighting, slow motion",
    "aspect_ratio": "16:9",
    "duration": 8,
    "resolution": "1080p"
  }'
```

```python Python
import requests

response = requests.post(
    "https://ai.kaiho.cc/v1/tasks",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "veo-3.1",
        "action": "generate",
        "prompt": "A golden retriever running on a beach at sunset, cinematic lighting, slow motion",
        "aspect_ratio": "16:9",
        "duration": 8,
        "resolution": "1080p"
    }
)

task_id = response.json()["task_id"]
print(f"任務已提交: {task_id}")
```

## 可用模型

| 模型 ID | 說明 |
|---------|------|
| `veo-3.1` | VEO 3.1，支援 4K、原生音訊 |
| `veo-3.1-fast` | VEO 3.1 快速版 |
| `veo-3.0` | VEO 3.0 |

## 統一參數

<ParamField body="model" type="string" required>
  模型 ID
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作類型
</ParamField>

<ParamField body="prompt" type="string" required>
  影片描述文字。VEO 3+ 支援在 prompt 中描述音效和對白，模型會自動生成對應音訊
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
  寬高比：`16:9`、`9:16`
</ParamField>

<ParamField body="duration" type="integer" default="8">
  影片時長（秒）：`4`、`6`、`8`
</ParamField>

<ParamField body="resolution" type="string" default="720p">
  輸出解析度，可選值取決於模型，見各模型 Tab
</ParamField>

<ParamField body="negative_prompt" type="string">
  不希望出現的內容，直接描述即可（不要用否定語句）
</ParamField>

<ParamField body="seed" type="integer">
  隨機種子，提高可重複性（不保證完全一致）。僅 VEO 3.1 系列支援
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook 回呼位址
</ParamField>

<ParamField body="extra" type="object">
  透傳給 Google VEO 的額外參數，詳見各模型 Tab
</ParamField>

## 模型特定參數

<Tabs>
  <Tab title="VEO 3.1">
    **resolution 可選值：** `720p`、`1080p`、`4k`

    <ParamField body="extra.personGeneration" type="string" default="allow_all">
      人物生成控制：

      - `allow_all` — 允許所有人物
      - `allow_adult` — 僅允許成年人
      - `dont_allow` — 禁止人物生成
    </ParamField>

    **獨有能力：**
    - 原生音訊生成（對白 + 環境音 + 音效，無需額外參數）
    - 4K 解析度輸出
  </Tab>

  <Tab title="VEO 3.1 Fast">
    **resolution 可選值：** `720p`、`1080p`、`4k`

    參數與 VEO 3.1 相同，處理速度更快。

    <ParamField body="extra.personGeneration" type="string" default="allow_all">
      人物生成控制：`allow_all`、`allow_adult`、`dont_allow`
    </ParamField>
  </Tab>

  <Tab title="VEO 3.0">
    **resolution 可選值：** `720p`、`1080p`

    <ParamField body="extra.personGeneration" type="string" default="allow_all">
      人物生成控制：`allow_all`、`allow_adult`、`dont_allow`
    </ParamField>
  </Tab>
</Tabs>

## 約束條件

| 條件 | 要求 |
|------|------|
| 解析度 `1080p` 或 `4k` | `duration` 必須為 `8` |
| 解析度 `4k` | 僅 VEO 3.1 和 3.1 Fast 支援 |
| 輸出幀率 | 24 FPS |
| 輸出格式 | MP4 |

<Note>
VEO 3+ 原生支援音訊生成，無需額外參數。在 prompt 中描述音效即可（如 "the sound of waves crashing"）。
</Note>
