非同步處理模式 — 提交請求後回傳任務 ID，透過 [查詢任務狀態](/zh-Hant/api-reference/task/status) 介面取得生成結果。

<Warning>
Sora 2 系列模型將於 2026 年 9 月 24 日停止服務。
</Warning>

<RequestExample>
```bash cURL
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "sora-2-pro",
    "action": "generate",
    "prompt": "A cat playing piano in a jazz club, cinematic lighting",
    "aspect_ratio": "16:9",
    "duration": 8,
    "resolution": "1080p"
  }'
```

```python Python
import requests

response = requests.post(
    "https://www.qingbo.dev/v1/tasks",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "sora-2-pro",
        "action": "generate",
        "prompt": "A cat playing piano in a jazz club, cinematic lighting",
        "aspect_ratio": "16:9",
        "duration": 8,
        "resolution": "1080p"
    }
)

task_id = response.json()["task_id"]
print(f"任務已提交: {task_id}")
```

```go Go
import (
```

```java Java
import java.net.http.*;
import java.net.URI;

public class Main {
    public static void main(String[] args) throws Exception {
        String payload = """
        {
          "model": "sora-2-pro",
          "action": "generate",
          "prompt": "A cat playing piano in a jazz club, cinematic lighting",
          "aspect_ratio": "16:9",

        """;

        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://www.qingbo.dev/v1/tasks"))
            .header("Authorization", "Bearer YOUR_API_KEY")
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(payload))
            .build();

        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString());
        System.out.println(response.body());
    }
}
```

## 可用模型

| 模型 ID | resolution 支援 | 說明 |
|---------|----------------|------|
| `sora-2-pro` | `720p`、`1080p` | 高畫質，支援 1080p |
| `sora-2` | `720p` | 速度更快 |

## 統一參數

<ParamField body="model" type="string" required>
  模型 ID
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作類型：`generate`、`remix`、`extend`
</ParamField>

<ParamField body="prompt" type="string" required>
  影片描述文字
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook 回呼位址
</ParamField>

<ParamField body="extra" type="object">
  透傳給 Sora 的額外參數
</ParamField>

## 各操作類型參數

<Tabs>
  <Tab title="生成 (generate)">
    文生影片或圖生影片。

    <ParamField body="aspect_ratio" type="string" default="9:16">
      寬高比：`16:9`、`9:16`
    </ParamField>

    <ParamField body="duration" type="integer" default="4">
      影片時長（秒）：`4`、`8`、`12`、`16`、`20`
    </ParamField>

    <ParamField body="resolution" type="string" default="720p">
      輸出解析度。`1080p` 僅 `sora-2-pro` 支援
    </ParamField>

    <ParamField body="image_urls" type="string[]">
      參考圖片 URL，用於首幀引導。圖片寬高比需與 `aspect_ratio` 一致
    </ParamField>

    <ParamField body="extra.characters" type="array">
      角色資產陣列，最多 2 個。角色名稱必須在 prompt 中被提及

      <Expandable title="屬性">
        <ParamField body="name" type="string" required>
          角色名稱
        </ParamField>
        <ParamField body="asset_id" type="string" required>
          角色資產 ID
        </ParamField>
      </Expandable>
    </ParamField>
  </Tab>

  <Tab title="Remix">
    基於已有影片進行修改。需提供原始影片的 task_id。

    <ParamField body="extra.video_id" type="string" required>
      要 remix 的原始影片 ID
    </ParamField>

    <ParamField body="extra.strength" type="string" default="Mild">
      變化程度：`Subtle`、`Mild`、`Strong`，或數字 `1`-`8`
    </ParamField>

    <Note>
    Remix 即將被 Edits 替代。
    </Note>
  </Tab>

  <Tab title="續寫 (extend)">
    延續已有影片，保持運動、鏡頭和場景的連續性。

    <ParamField body="duration" type="integer" required>
      續寫片段時長（秒）：`4`、`8`、`12`、`16`、`20`
    </ParamField>

    <ParamField body="extra.video_id" type="string" required>
      要續寫的原始影片 ID（必須是已完成的 Sora 影片）
    </ParamField>

    **限制：**
    - 每條影片鏈最多續寫 6 次（總計 120 秒）
    - 不支援 `image_urls` 和 `characters`
  </Tab>
</Tabs>

</ResponseField>
