<Note>
  本接口采用异步模式。提交成功后将返回 `task_id`，您需要通过任务状态接口获取生成结果。生成结果的下载链接有效期为 24 小时。
</Note>

<RequestExample>

```bash cURL theme={null}
curl -X POST "https://www.qingbo.dev/v1/tasks" \
     -H "Authorization: Bearer YOUR_API_KEY" \
     -H "Content-Type: application/json" \
     -d '{
       "model": "doubao-seedream-4.0",
       "prompt": "水墨画风格的山水风景",
       "size": "16:9"
     }'
```

```python Python theme={null}
import requests

url = "https://www.qingbo.dev/v1/tasks"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "model": "doubao-seedream-4.0",
    "prompt": "水墨画风格的山水风景",
    "size": "16:9"
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
```

```go Go theme={null}
package main

import (
	"bytes"
	"encoding/json"
	"fmt"
	"net/http"
)

func main() {
	url := "https://www.qingbo.dev/v1/tasks"
	payload := map[string]interface{}{
		"model":  "doubao-seedream-4.0",
		"prompt": "水墨画风格的山水风景",
		"size":   "16:9",
	}
	jsonPayload, _ := json.Marshal(payload)

	req, _ := http.NewRequest("POST", url, bytes.NewBuffer(jsonPayload))
	req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
	req.Header.Set("Content-Type", "application/json")

	client := &http.Client{}
	resp, _ := client.Do(req)
	defer resp.Body.Close()

	var result map[string]interface{}
	json.NewDecoder(resp.Body).Decode(&result)
	fmt.Println(result)
}
```

## 认证方式

<ParamField header="Authorization" type="string" placeholder="Bearer YOUR_API_KEY" required>
  在 [API Key 页面](https://qingbo.dev/dashboard/keys) 获取您的密钥，并以 `Bearer YOUR_API_KEY` 的格式放入请求头中。
</ParamField>

## 请求参数

<ParamField body="model" type="string" required>
  使用的模型 ID：`doubao-seedream-4.0`。
</ParamField>

<ParamField body="prompt" type="string" required>
  图像生成的文本提示词。最大长度 1000 字符。
</ParamField>

<ParamField body="image_urls" type="array">
  参考图片 URL 数组。用于图生图或图像编辑模式。
</ParamField>

<ParamField body="size" type="string" default="1:1">
  生成图像的比例。可选值：`1:1`, `2:3`, `3:2`, `3:4`, `4:3`, `9:16`, `16:9`。
</ParamField>

## 响应参数

<ResponseField name="code" type="string">
  响应状态码

  * `success` - 请求成功，任务已提交
</ResponseField>

<ResponseField name="data" type="string">
  任务 ID，格式为 `task-wave{timestamp}{userId}{random}`

  用于通过 [查询任务状态](/cn/api-reference/task/get-status) 接口获取生成结果
</ResponseField>

## 注意事项

- 接口返回 `task_id` 后，请前往 [查询任务状态](/cn/api-reference/task/get-status) 页面了解如何获取最终图像。
- 支持文生图、图生图及图像编辑。
- 生成的下载链接具有时效性，请及时转存。
