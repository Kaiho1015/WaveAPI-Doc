非同期処理モード — リクエスト送信後にタスク ID が返されます。[タスクステータス照会](/jp/api-reference/task/status) インターフェースで生成結果を取得してください。動画リンクの有効期限は24時間です。

<RequestExample>
```bash cURL
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "minimax-hailuo-2.3",
    "action": "generate",
    "prompt": "一只白猫慵懒地躺在阳光下的窗台上",
    "duration": 6,
    "resolution": "1080p"
  }'
```

```python Python
import requests

response = requests.post(
    "https://www.qingbo.dev/v1/tasks",
    headers={"Authorization": "Bearer YOUR_API_KEY"},

        "action": "generate",
        "prompt": "一只白猫慵懒地躺在阳光下的窗台上",
        "duration": 6,
        "resolution": "1080p"
    }
)

task_id = response.json()["task_id"]
print(f"タスクを送信しました: {task_id}")
```

```java Java
import java.net.http.*;
import java.net.URI;

public class Main {
    public static void main(String[] args) throws Exception {
        String payload = """
        {
          "model": "minimax-hailuo-2.3",
          "action": "generate",
          "prompt": "一只白猫慵懒地躺在阳光下的窗台上",

          "resolution": "1080p"
        }
        """;

        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://www.qingbo.dev/v1/tasks"))
            .header("Authorization", "Bearer YOUR_API_KEY")
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(payload))
            .build();

        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString());
        System.out.println(response.body());
    }
}
```

## 利用可能なモデル

| モデル ID | デフォルト解像度 | 対応する長さ | 説明 |
|---------|-----------|---------|------|
| `minimax-hailuo-2.3` | 768P | 6秒 / 10秒 | Hailuo 2.3、キャラクター参照に対応 |
| `minimax-hailuo-2.3-fast` | 768P | 6秒 / 10秒 | Hailuo 2.3 高速版、画像から動画のみ対応 |

## 統一パラメータ

<ParamField body="model" type="string" required>
  モデル ID
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作タイプ
</ParamField>

<ParamField body="prompt" type="string" required>
  動画の説明テキスト
</ParamField>

<ParamField body="duration" type="integer" default="6">
  動画の長さ（秒）：`6`、`10`
</ParamField>

<ParamField body="resolution" type="string" default="768p">
  出力解像度、選択可能な値はモデルにより異なります。各モデルの Tab を参照
</ParamField>

<ParamField body="image_urls" type="string[]">
  先頭フレーム画像 URL、画像から動画生成に使用
</ParamField>

<ParamField body="last_frame" type="string">
  末尾フレーム画像 URL、`image_urls` と併用が必要
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook コールバックアドレス
</ParamField>

<ParamField body="extra" type="object">
  Hailuo にパススルーされる追加パラメータ、各モデルの Tab を参照
</ParamField>

    </ParamField>

## 生成モード

| モード | パラメータ | 対応モデル |
|------|------|---------|
| テキストから動画 | `prompt` | 全て |
| 画像から動画 | `prompt` + `image_urls`(1枚) | 全て |
| キャラクター参照 | `prompt` + `extra.subject_reference` | minimax-hailuo-2.3 |
