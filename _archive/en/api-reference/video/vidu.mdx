Async processing mode -- submit a request and receive a task ID. Use the [Query Task Status](/en/api-reference/task/status) endpoint to retrieve generation results. Video links are valid for 24 hours.

<RequestExample>
```bash cURL
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "viduq3",
    "action": "generate",
    "prompt": "城市夜景延时摄影，霓虹灯光流转",
    "aspect_ratio": "16:9",
    "duration": 5,
    "resolution": "1080p"
  }'
```

```java Java
import java.net.http.*;
import java.net.URI;

public class Main {
    public static void main(String[] args) throws Exception {

        {
          "model": "viduq3",
          "action": "generate",
          "prompt": "城市夜景延时摄影，霓虹灯光流转",

        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://www.qingbo.dev/v1/tasks"))
            .header("Authorization", "Bearer YOUR_API_KEY")
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(payload))
            .build();

        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString());
        System.out.println(response.body());
    }
}
```

## Available Models

| Model ID | Description |
|----------|-------------|
| `viduq3` | Vidu Q3, image-to-video only (requires 1-7 reference images) |
| `viduq3-pro` | Vidu Q3 Pro, highest quality |
| `viduq3-turbo` | Vidu Q3 Turbo, speed-optimized |

## Common Parameters

<ParamField body="model" type="string" required>
  Model ID
</ParamField>

<ParamField body="action" type="string" default="generate">
  Operation type
</ParamField>

<ParamField body="prompt" type="string" required>
  Video description text
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
  Aspect ratio: `16:9`, `9:16`, `1:1`
</ParamField>

<ParamField body="duration" type="integer" default="5">
  Video duration (seconds)
</ParamField>

<ParamField body="resolution" type="string" default="1080p">
  Resolution: `720p`, `1080p`
</ParamField>

<ParamField body="seed" type="integer">
  Random seed, `0` for random
</ParamField>

<ParamField body="watermark" type="boolean">
  Whether to add a watermark. Supports custom watermark images and positioning (via extra)
</ParamField>

<ParamField body="image_urls" type="string[]">
  Reference image URL array. The number of images determines the generation mode:
  - 1 image: Image-to-video
  - 2 images: First/last frame mode
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook callback URL
</ParamField>

<ParamField body="extra" type="object">
  Pass-through parameters
</ParamField>

## Model-Specific Parameters

<ParamField body="extra.movement_amplitude" type="string" default="auto">
  Motion amplitude: `auto`, `small`, `medium`, `large`
</ParamField>

<ParamField body="extra.bgm" type="boolean" default="false">
  Whether to add background music
</ParamField>

<ParamField body="extra.wm_url" type="string">
  Custom watermark image URL (requires `watermark: true`)
</ParamField>

<ParamField body="extra.wm_position" type="integer">
  Watermark position: `1` (top-left), `2` (top-right), `3` (bottom-left), `4` (bottom-right)
</ParamField>

## Generation Modes

| Mode | Parameters | Description |
|------|-----------|-------------|
| Text-to-video | `prompt` | Pure text generation |
| Image-to-video | `prompt` + `image_urls` (1 image) | Single image reference |
| First/Last frame | `image_urls` (2 images) | Transition between start and end frames |
