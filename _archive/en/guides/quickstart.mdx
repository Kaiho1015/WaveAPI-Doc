## Prerequisites

- A WaveAPI account ([Sign up](https://qingbo.dev/login))
- An API key
- Basic programming knowledge

## Obtain an API Key

<Steps>
  <Step title="Log in to the Console">
    Visit the [Dashboard](https://qingbo.dev/dashboard/keys) and log in
  </Step>
  <Step title="Create a Key">
    Click "Create New Key" on the key management page
  </Step>
  <Step title="Save Your Key">
    Copy and securely store your API key — it is only displayed once
  </Step>
</Steps>

<Warning>
Do not expose your API key in front-end code or public repositories.
</Warning>

## Send Your First Request

WaveAPI is fully compatible with the OpenAI SDK — simply replace the `base_url` to get started.

<CodeGroup>

```python Python
from openai import OpenAI

client = OpenAI(
    base_url="https://www.qingbo.dev/v1",
    api_key="YOUR_API_KEY"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "user", "content": "你好，介绍一下你自己"}
    ]
)

print(response.choices[0].message.content)
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'https://www.qingbo.dev/v1',
  apiKey: 'YOUR_API_KEY'
});

const response = await client.chat.completions.create({
  model: 'gpt-4o',
  messages: [
    { role: 'user', content: '你好，介绍一下你自己' }
  ]
});

console.log(response.choices[0].message.content);
```

```bash cURL
curl https://www.qingbo.dev/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "gpt-4o",
    "messages": [
      {"role": "user", "content": "你好，介绍一下你自己"}
    ]
  }'
```

</CodeGroup>
