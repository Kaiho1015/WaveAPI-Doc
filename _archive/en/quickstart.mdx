## 📋 Prerequisites

Before you begin, you'll need:

- A WaveAPI account
- An API key (obtain from the [Dashboard](https://qingbo.dev/token))
- Basic programming knowledge

## 1️⃣ Get Your API Key

<Steps>
  <Step title="Login to Console">
    Visit [WaveAPI Dashboard](https://qingbo.dev/token) and log in
  </Step>
  <Step title="Generate Key">
    Click "Create New Key" on the API Key Management page
  </Step>
  <Step title="Save Your Key">
    Copy and securely save your API key
  </Step>
</Steps>

<Warning>
  **Security Note:** Never expose your API key in public code or on the client side
</Warning>

## 2️⃣ Make Your First Request

### Using cURL

```bash
curl https://www.qingbo.dev/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "gpt-4o",
    "messages": [
      {
        "role": "user",
        "content": "Hello, please introduce yourself"
      }
    ]
  }'
```

### Using Python

```python
import openai

client = openai.OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://www.qingbo.dev/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "user", "content": "Hello, please introduce yourself"}
    ]
)

print(response.choices[0].message.content)
```

### Using JavaScript/Node.js

```javascript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'YOUR_API_KEY',
  baseURL: 'https://www.qingbo.dev/v1'
});

async function main() {
  const response = await client.chat.completions.create({
    model: 'gpt-4o',
    messages: [
      { role: 'user', content: 'Hello, please introduce yourself' }
    ]
  });
  
  console.log(response.choices[0].message.content);
}

main();
```
