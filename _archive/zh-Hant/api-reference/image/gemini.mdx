非同步處理模式 — 提交請求後回傳任務 ID，透過 [查詢任務狀態](/zh-Hant/api-reference/task/status) 介面取得生成結果。

<RequestExample>
```bash cURL
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini-3.1-flash-image-preview",
    "prompt": "一只橘猫趴在窗台上晒太阳，水彩画风格",
    "size": "16:9"
  }'
```

```python Python
from openai import OpenAI

client = OpenAI(
    base_url="https://www.qingbo.dev/v1",
    api_key="YOUR_API_KEY"
)

response = client.images.generate(
    model="gemini-3.1-flash-image-preview",
    prompt="一只橘猫趴在窗台上晒太阳，水彩画风格",
    size="16:9"
)

print(response.data[0])
```

```go Go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "model":  "gemini-3.1-flash-image-preview",
        "prompt": "一只橘猫趴在窗台上晒太阳，水彩画风格",
        "size":   "16:9",
    }

    body, _ := json.Marshal(payload)
    req, _ := http.NewRequest("POST", "https://www.qingbo.dev/v1/tasks", bytes.NewBuffer(body))
    req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
    req.Header.Set("Content-Type", "application/json")

    resp, _ := http.DefaultClient.Do(req)
    defer resp.Body.Close()
    result, _ := io.ReadAll(resp.Body)
    fmt.Println(string(result))
}
```

```java Java
import java.net.http.*;
import java.net.URI;

public class Main {
    public static void main(String[] args) throws Exception {
        String payload = """
        {
          "model": "gemini-3.1-flash-image-preview",
          "prompt": "一只橘猫趴在窗台上晒太阳，水彩画风格",
          "size": "16:9"
        }
        """;

        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://www.qingbo.dev/v1/tasks"))
            .header("Authorization", "Bearer YOUR_API_KEY")
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(payload))
            .build();

        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString());
        System.out.println(response.body());
    }
}
```

## 可用模型

| 模型 ID | 說明 |
|---------|------|
| `gemini-2.5-flash-image` | Gemini 2.5 Flash 圖像預覽版 |
| `gemini-3-pro-image-preview` | Gemini 3 Pro 圖像預覽版 |
| `gemini-3.1-flash-image-preview` | Gemini 3.1 Flash 圖像預覽版，支援高解析度和搜尋增強 |

## 通用參數

<ParamField body="model" type="string" required>
  模型 ID，見上方模型列表
</ParamField>

<ParamField body="prompt" type="string" required>
  圖像描述文字，最多 1000 個字元
</ParamField>

<ParamField body="image_urls" type="array">
  參考圖片 URL 陣列，最多 14 張，每張不超過 10MB
</ParamField>

<ParamField body="n" type="integer" default="1">
  生成數量，固定為 1
</ParamField>

## 回應

回傳任務 ID，使用 [查詢任務狀態](/zh-Hant/api-reference/task/status) 取得最終圖像結果。

<ResponseField name="code" type="string">
  狀態碼，成功為 `"success"`
</ResponseField>

<ResponseField name="data" type="string">
  任務 ID，格式為 `task-wave{timestamp}{userId}{random}`
</ResponseField>
