## 概述

WaveAPI 支援多種圖像生成模型，統一透過非同步任務介面 `/v1/tasks` 呼叫。

## 請求端點

```
POST https://www.qingbo.dev/v1/tasks
```

## 通用請求參數

| 參數 | 類型 | 必填 | 說明 |
|------|------|------|------|
| `model` | string | 是 | 模型名稱 |
| `action` | string | 否 | 操作類型，預設 `"generate"` |
| `prompt` | string | 是 | 圖像描述文字 |
| `negative_prompt` | string | 否 | 不希望出現的內容 |
| `n` | integer | 否 | 生成數量，預設 1 |
| `size` | string | 否 | 寬高比，如 `"16:9"` |
| `resolution` | string | 否 | 解析度，如 `"2K"` |
| `image_urls` | array | 否 | 參考圖片 URL 陣列 |
| `callback_url` | string | 否 | 完成後的回呼位址 |
| `extra` | object | 否 | 模型特定的透傳參數 |

## 回應格式

任務提交後回傳 `task_id`，透過 `GET /v1/tasks/{task_id}` 查詢結果：

```json
{
  "task_id": "task-wave1775285160b950328499",
  "model": "doubao-seedream-5.0",
  "action": "generate",
  "status": "queued",
  "created_at": 1775285160040,
  "progress": 0
}
```
