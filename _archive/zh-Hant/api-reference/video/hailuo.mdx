非同步處理模式 — 提交請求後回傳任務 ID，透過 [查詢任務狀態](/zh-Hant/api-reference/task/status) 介面取得生成結果。影片連結有效期 24 小時。

<RequestExample>
```bash cURL
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "minimax-hailuo-2.3",
    "action": "generate",
    "prompt": "一只白猫慵懒地躺在阳光下的窗台上",
    "duration": 6,
    "resolution": "1080p"
  }'
```

```python Python
import requests

        "resolution": "1080p"
    }

print(f"任務已提交: {task_id}")
```

import java.net.http.*;
import java.net.URI;

public class Main {

    }

## 可用模型

| 模型 ID | 預設解析度 | 支援時長 | 說明 |
|---------|-----------|---------|------|
| `minimax-hailuo-2.3` | 768P | 6s / 10s | 海螺 2.3，支援角色參考 |
| `minimax-hailuo-2.3-fast` | 768P | 6s / 10s | 海螺 2.3 快速版，僅支援圖生影片 |

## 統一參數

<ParamField body="model" type="string" required>
  模型 ID
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作類型
</ParamField>

<ParamField body="prompt" type="string" required>
  影片描述文字
</ParamField>

<ParamField body="duration" type="integer" default="6">
  影片時長（秒）：`6`、`10`
</ParamField>

<ParamField body="resolution" type="string" default="768p">
  輸出解析度，可選值取決於模型，見各模型 Tab
</ParamField>

<ParamField body="image_urls" type="string[]">
  首幀圖片 URL，用於圖生影片
</ParamField>

<ParamField body="last_frame" type="string">
  尾幀圖片 URL，需配合 `image_urls`
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook 回呼位址
</ParamField>

<ParamField body="extra" type="object">
  透傳給海螺的額外參數，詳見各模型 Tab
</ParamField>

## 生成模式

| 模式 | 參數 | 支援模型 |
|------|------|---------|
| 文生影片 | `prompt` | 全部 |
| 圖生影片 | `prompt` + `image_urls`(1張) | 全部 |
| 角色參考 | `prompt` + `extra.subject_reference` | minimax-hailuo-2.3 |
