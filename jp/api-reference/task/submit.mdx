---

## エンドポイント

```
POST https://www.qingbo.dev/v1/tasks
```

画像、動画、または音声の生成タスクを送信し、すぐに `task_id` を返します。

## リクエストパラメータ

<ParamField body="model" type="string" required>
  モデル名、例：`"veo3"`, `"gpt-4o-image"`, `"sora2"`
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作タイプ：`generate`, `edit`, `upscale`, `variation`, `blend`
</ParamField>

<ParamField body="prompt" type="string">
  生成の説明テキスト
</ParamField>

<ParamField body="callback_url" type="string">
  タスク完了時の Webhook コールバックアドレス
</ParamField>

<ParamField body="extra" type="object">
  モデル固有パラメータのパススルーフィールド、上流モデルに直接渡されます
</ParamField>

その他のパラメータは [画像生成の概要](/jp/api-reference/image/overview) または [動画生成の概要](/jp/api-reference/video/overview) を参照してください。

## レスポンス

```json
{
  "task_id": "task_01K8SGYNNNVBQTXNR4MM964S7K",
  "model": "veo3",
  "action": "generate",
  "status": "queued",
  "created_at": 1720000000,
  "progress": 0
}
```

## 例

<CodeGroup>

```bash cURL
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "veo3",
    "action": "generate",
    "prompt": "A cat walking on the beach at sunset",
    "duration": 5
  }'
```

```python Python
import requests

response = requests.post(
    "https://www.qingbo.dev/v1/tasks",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "veo3",
        "action": "generate",
        "prompt": "A cat walking on the beach at sunset",
        "duration": 5
    }
)

task_id = response.json()["task_id"]
```

</CodeGroup>
