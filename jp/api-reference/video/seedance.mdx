非同期処理モード — リクエスト送信後にタスク ID が返されます。[タスクステータス照会](/jp/api-reference/task/status) インターフェースで生成結果を取得してください。動画リンクの有効期限は24時間です。速やかにダウンロードしてください。

<RequestExample>
```bash cURL
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "seedance-2.0",
    "action": "generate",
    "prompt": "一只柴犬穿宇航服在月球上行走，电影感光影",
    "aspect_ratio": "16:9",
    "duration": 5,
    "resolution": "1080p",
    "watermark": false,
    "extra": {
      "generate_audio": true
    }
  }'
```

## 利用可能なモデル

| モデル ID | 説明 |
|---------|------|
| `seedance-2.0` | Seedance 2.0、ネイティブ音声・先頭末尾フレーム・マルチモーダル参照・最大 2K に対応 |
| `seedance-2.0-fast` | Seedance 2.0 高速版 |
| `seedance-1.5-pro` | Seedance 1.5 プロフェッショナル版 |
| `seedance-1.0-pro-fast` | Seedance 1.0 高速版 |
| `seedance-1.0-pro-quality` | Seedance 1.0 高画質版 |

## 統一パラメータ

<ParamField body="model" type="string" required>
  モデル ID
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作タイプ
</ParamField>

<ParamField body="prompt" type="string" required>
  動画の説明テキスト、中国語・英語に対応。2.0 ではプロンプト内で `@image1`、`@video1`、`@audio1` を使用して参照素材を引用できます
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
  アスペクト比：`16:9`、`9:16`、`4:3`、`3:4`、`21:9`、`1:1`
</ParamField>

<ParamField body="duration" type="integer" default="5">
  動画の長さ（秒）、選択可能な範囲はモデルにより異なります。各モデルの Tab を参照
</ParamField>

<ParamField body="resolution" type="string">
  出力解像度、選択可能な値はモデルにより異なります。各モデルの Tab を参照
</ParamField>

<ParamField body="image_urls" type="string[]">
  参照画像 URL 配列、数量制限はモデルバージョンにより異なります
</ParamField>

<ParamField body="first_frame" type="string">
  先頭フレーム画像 URL
</ParamField>

<ParamField body="last_frame" type="string">
  末尾フレーム画像 URL（`first_frame` と併用が必要）
</ParamField>

<ParamField body="seed" type="integer" default="-1">
  ランダムシード、範囲 [-1, 2^32-1]。`-1` はランダム。同じシード + 同じリクエストで類似の結果が得られます
</ParamField>

<ParamField body="watermark" type="boolean" default="false">
  透かしを追加するかどうか
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook コールバックアドレス
</ParamField>

<ParamField body="extra" type="object">
  上流にパススルーされる追加パラメータ、各モデルの Tab を参照
</ParamField>

## モデル固有のパラメータ

<Tabs>
  <Tab title="Seedance 2.0">
    **duration の範囲：** 4-15秒

    **resolution の選択可能な値：** `480p`、`720p`、`1080p`、`2K`

    **size の追加対応：** `adaptive`（自適応、入力素材のアスペクト比に合わせる）

    **入力制限：** 画像最大9枚（≤ 30MB）、動画最大3つ（≤ 50MB、合計長さ ≤ 15秒）、音声最大3つ（MP3/WAV、≤ 15MB、合計長さ ≤ 15秒）

    <ParamField body="extra.generate_audio" type="boolean" default="false">
      ネイティブ音声・映像同時生成、セリフのリップシンクに対応（8言語以上）
    </ParamField>

    <ParamField body="extra.return_last_frame" type="boolean" default="false">
      末尾フレーム画像 URL を返却、動画クリップの連続撮影に便利
    </ParamField>

    <ParamField body="extra.tools" type="array">
      モデルツール、ウェブ検索に対応

      ```json
      "extra": { "tools": [{"type": "web_search"}] }
      ```
    </ParamField>

    <ParamField body="extra.service_tier" type="string">
      サービスティア、レイテンシーとコストのバランスを制御
    </ParamField>

    <Note>
    2.0 で先頭末尾フレームモード（`first_frame` + `last_frame`）を使用する場合、`video_urls` と `audio_urls` は同時に指定できません。
    </Note>
  </Tab>

  <Tab title="Seedance 2.0 Fast">
    **duration の範囲：** 4-15秒

    **resolution の選択可能な値：** `480p`、`720p`、`1080p`、`2K`

    パラメータは Seedance 2.0 と同じですが、処理速度が速くなります。

    <ParamField body="extra.generate_audio" type="boolean" default="false">
      ネイティブ音声・映像同時生成
    </ParamField>

    <ParamField body="extra.return_last_frame" type="boolean" default="false">
      末尾フレーム画像を返却
    </ParamField>
  </Tab>

  <Tab title="Seedance 1.5 Pro">
    **duration の範囲：** 4-12秒

    **resolution の選択可能な値：** `480p`、`720p`、`1080p`

    **入力制限：** 画像最大2枚（先頭フレーム1枚 + 末尾フレーム1枚）

    <ParamField body="extra.audio" type="boolean" default="false">
      伴奏音声を生成するかどうか
    </ParamField>

    <ParamField body="extra.camerafixed" type="boolean" default="false">
      固定カメラモード、有効にするとカメラ位置が固定されます
    </ParamField>
  </Tab>

  <Tab title="Seedance 1.0 Pro">
    **2つのサブバージョン：** `seedance-1.0-pro-fast`（高速）と `seedance-1.0-pro-quality`（高画質）

    **duration の範囲：** 2-12秒

    **resolution の選択可能な値：** `480p`、`720p`、`1080p`

    **入力制限：** 画像最大1枚

    <Note>
    1.0 Quality バージョンは `last_frame`（末尾フレーム）と `reference`（スタイル参照）ロールに対応しています。reference ロール使用時は 1080p 解像度に非対応です。
    </Note>
  </Tab>
</Tabs>

## 生成モード

| モード | パラメータ | 対応バージョン |
|------|------|---------|
| テキストから動画 | `prompt` | 全て |
| 画像から動画 | `prompt` + `image_urls`(1枚) または `first_frame` | 全て |
| 先頭末尾フレーム | `first_frame` + `last_frame` | 1.0-quality / 1.5 / 2.0 |
| マルチモーダル参照 | `image_urls` + `video_urls` + `audio_urls` | 2.0 のみ |
