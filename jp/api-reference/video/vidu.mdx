非同期処理モード — リクエスト送信後にタスク ID が返されます。[タスクステータス照会](/jp/api-reference/task/status) インターフェースで生成結果を取得してください。動画リンクの有効期限は24時間です。

<RequestExample>
```bash cURL
curl -X POST https://qingbo.dev/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "viduq3",
    "action": "generate",
    "prompt": "城市夜景延时摄影，霓虹灯光流转",
    "aspect_ratio": "16:9",
    "duration": 5,
    "resolution": "1080p"
  }'
```

## 利用可能なモデル

| モデル ID | 説明 |
|---------|------|
| `viduq3` | Vidu Q3、ネイティブ音声・16秒・1080p |
| `viduq3-pro` | Vidu Q3 Pro、最高画質 |
| `viduq3-turbo` | Vidu Q3 Turbo、速度優先 |
| `viduq3-mix` | Vidu Q3 Mix、複数参照画像・複数エンティティの一貫性 |
| `viduq2` | Vidu Q2 |
| `viduq1` | Vidu Q1 |

## 統一パラメータ

<ParamField body="model" type="string" required>
  モデル ID
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作タイプ
</ParamField>

<ParamField body="prompt" type="string" required>
  動画の説明テキスト
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
  アスペクト比：`16:9`、`9:16`、`1:1`
</ParamField>

<ParamField body="duration" type="integer" default="5">
  動画の長さ（秒）
</ParamField>

<ParamField body="resolution" type="string" default="1080p">
  解像度：`720p`、`1080p`
</ParamField>

<ParamField body="seed" type="integer">
  ランダムシード、`0` はランダム
</ParamField>

<ParamField body="watermark" type="boolean">
  透かしを追加するかどうか。カスタム透かし画像と位置にも対応（extra 経由）
</ParamField>

<ParamField body="image_urls" type="string[]">
  参照画像 URL 配列、枚数により生成モードが決まります：
  - 1枚 → 画像から動画
  - 2枚 → 先頭末尾フレームモード
  - 3枚以上 → 参照画像ガイド（自動的に viduq2 に切り替え）
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook コールバックアドレス
</ParamField>

<ParamField body="extra" type="object">
  パススルーパラメータ
</ParamField>

## モデル固有のパラメータ

<ParamField body="extra.movement_amplitude" type="string" default="auto">
  動きの幅：`auto`、`small`、`medium`、`large`
</ParamField>

<ParamField body="extra.bgm" type="boolean" default="false">
  BGM を追加するかどうか
</ParamField>

<ParamField body="extra.wm_url" type="string">
  カスタム透かし画像 URL（`watermark: true` が必要）
</ParamField>

<ParamField body="extra.wm_position" type="integer">
  透かし位置：`1`（左上）、`2`（右上）、`3`（左下）、`4`（右下）
</ParamField>

## 生成モード

| モード | パラメータ | 説明 |
|------|------|------|
| テキストから動画 | `prompt` | テキストのみで生成 |
| 画像から動画 | `prompt` + `image_urls`(1枚) | 単一画像参照 |
| 先頭末尾フレーム | `image_urls`(2枚) | 開始・終了画面の遷移 |
| 参照画像ガイド | `image_urls`(3枚以上) | 複数画像参照（自動的に viduq2 に切り替え） |
