非同期処理モード — リクエスト送信後にタスク ID が返されます。[タスクステータス照会](/jp/api-reference/task/status) インターフェースで生成結果を取得してください。

<Warning>
Sora 2 シリーズモデルは2026年9月24日にサービスを終了します。
</Warning>

<RequestExample>
```bash cURL
curl -X POST https://ai.kaiho.cc/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "sora-2-pro",
    "action": "generate",
    "prompt": "A cat playing piano in a jazz club, cinematic lighting",
    "aspect_ratio": "16:9",
    "duration": 8,
    "resolution": "1080p"
  }'
```

```python Python
        "prompt": "A cat playing piano in a jazz club, cinematic lighting",
        "aspect_ratio": "16:9",
        "duration": 8,
        "resolution": "1080p"
    }
)

task_id = response.json()["task_id"]
print(f"タスクを送信しました: {task_id}")
```

```java Java
import java.net.http.*;
import java.net.URI;

public class Main {
    public static void main(String[] args) throws Exception {
        String payload = """
        {
          "model": "sora-2-pro",
          "action": "generate",
          "prompt": "A cat playing piano in a jazz club, cinematic lighting",
          "aspect_ratio": "16:9",

        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://ai.kaiho.cc/v1/tasks"))
            .header("Authorization", "Bearer YOUR_API_KEY")
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(payload))
            .build();

        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString());
        System.out.println(response.body());
    }
}
```

## 利用可能なモデル

| モデル ID | resolution 対応 | 説明 |
|---------|----------------|------|
| `sora-2-pro` | `720p`、`1080p` | 高画質、1080p 対応 |
| `sora-2` | `720p` | 速度重視 |

## 統一パラメータ

<ParamField body="model" type="string" required>
  モデル ID
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作タイプ：`generate`、`remix`、`extend`
</ParamField>

<ParamField body="prompt" type="string" required>
  動画の説明テキスト
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook コールバックアドレス
</ParamField>

<ParamField body="extra" type="object">
  Sora にパススルーされる追加パラメータ
</ParamField>

## 各操作タイプのパラメータ

<Tabs>
  <Tab title="生成 (generate)">
    テキストから動画または画像から動画。

    <ParamField body="aspect_ratio" type="string" default="9:16">
      アスペクト比：`16:9`、`9:16`
    </ParamField>

    <ParamField body="duration" type="integer" default="4">
      動画の長さ（秒）：`4`、`8`、`12`、`16`、`20`
    </ParamField>

    <ParamField body="resolution" type="string" default="720p">
      出力解像度。`1080p` は `sora-2-pro` のみ対応
    </ParamField>

    <ParamField body="image_urls" type="string[]">
      参照画像 URL、先頭フレームガイダンス用。画像のアスペクト比は `aspect_ratio` と一致する必要があります
    </ParamField>

    <ParamField body="extra.characters" type="array">
      キャラクターアセット配列、最大2つ。キャラクター名はプロンプト内で言及する必要があります

      <Expandable title="属性">
        <ParamField body="name" type="string" required>
          キャラクター名
        </ParamField>
        <ParamField body="asset_id" type="string" required>
          キャラクターアセット ID
        </ParamField>
      </Expandable>
    </ParamField>
  </Tab>

  <Tab title="Remix">
    既存の動画を基に修正。元の動画の task_id が必要です。

    <ParamField body="extra.video_id" type="string" required>
      リミックスする元の動画 ID
    </ParamField>

    <ParamField body="extra.strength" type="string" default="Mild">
      変化の度合い：`Subtle`、`Mild`、`Strong`、または数値 `1`-`8`
    </ParamField>

    <Note>
    Remix は今後 Edits に置き換えられる予定です。
    </Note>
  </Tab>

  <Tab title="延長 (extend)">
    既存の動画を延長し、動き、カメラワーク、シーンの連続性を維持します。

    <ParamField body="duration" type="integer" required>
      延長部分の長さ（秒）：`4`、`8`、`12`、`16`、`20`
    </ParamField>

    <ParamField body="extra.video_id" type="string" required>
      延長する元の動画 ID（完了済みの Sora 動画である必要があります）
    </ParamField>

    **制限：**
    - 各動画チェーンで最大6回まで延長可能（合計120秒）
    - `image_urls` と `characters` は非対応
  </Tab>
</Tabs>
