非同期処理モード — リクエスト送信後にタスク ID が返されます。[タスクステータス照会](/jp/api-reference/task/status) インターフェースで生成結果を取得してください。動画リンクの有効期限は24時間です。速やかにダウンロードしてください。

<RequestExample>
```bash cURL
curl -X POST https://ai.kaiho.cc/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "veo-3.1",
    "action": "generate",
    "prompt": "A golden retriever running on a beach at sunset, cinematic lighting, slow motion",
    "aspect_ratio": "16:9",
    "duration": 8,
    "resolution": "1080p"
  }'
```

```python Python
import requests

response = requests.post(
    "https://ai.kaiho.cc/v1/tasks",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "veo-3.1",
        "action": "generate",
        "prompt": "A golden retriever running on a beach at sunset, cinematic lighting, slow motion",
        "aspect_ratio": "16:9",
        "duration": 8,
        "resolution": "1080p"
    }
)

task_id = response.json()["task_id"]
print(f"タスクを送信しました: {task_id}")
```

## 利用可能なモデル

| モデル ID | 説明 |
|---------|------|
| `veo-3.1` | VEO 3.1、4K・ネイティブ音声に対応 |
| `veo-3.1-fast` | VEO 3.1 高速版 |
| `veo-3.0` | VEO 3.0 |

## 統一パラメータ

<ParamField body="model" type="string" required>
  モデル ID
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作タイプ
</ParamField>

<ParamField body="prompt" type="string" required>
  動画の説明テキスト。VEO 3+ ではプロンプト内で効果音やセリフを記述でき、モデルが自動的に対応する音声を生成します
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
  アスペクト比：`16:9`、`9:16`
</ParamField>

<ParamField body="duration" type="integer" default="8">
  動画の長さ（秒）：`4`、`6`、`8`
</ParamField>

<ParamField body="resolution" type="string" default="720p">
  出力解像度、選択可能な値はモデルにより異なります。各モデルの Tab を参照
</ParamField>

<ParamField body="negative_prompt" type="string">
  表示させたくない内容を直接記述してください（否定文は使用しないでください）
</ParamField>

<ParamField body="seed" type="integer">
  ランダムシード、再現性を向上（完全一致は保証されません）。VEO 3.1 シリーズのみ対応
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook コールバックアドレス
</ParamField>

<ParamField body="extra" type="object">
  Google VEO にパススルーされる追加パラメータ、各モデルの Tab を参照
</ParamField>

## モデル固有のパラメータ

<Tabs>
  <Tab title="VEO 3.1">
    **resolution の選択可能な値：** `720p`、`1080p`、`4k`

    <ParamField body="extra.personGeneration" type="string" default="allow_all">
      人物生成の制御：

      - `allow_all` — 全ての人物を許可
      - `allow_adult` — 成人のみ許可
      - `dont_allow` — 人物生成を禁止
    </ParamField>

    **固有の機能：**
    - ネイティブ音声生成（セリフ + 環境音 + 効果音、追加パラメータ不要）
    - 4K 解像度出力
  </Tab>

  <Tab title="VEO 3.1 Fast">
    **resolution の選択可能な値：** `720p`、`1080p`、`4k`

    パラメータは VEO 3.1 と同じですが、処理速度が速くなります。

    <ParamField body="extra.personGeneration" type="string" default="allow_all">
      人物生成の制御：`allow_all`、`allow_adult`、`dont_allow`
    </ParamField>
  </Tab>

  <Tab title="VEO 3.0">
    **resolution の選択可能な値：** `720p`、`1080p`

    <ParamField body="extra.personGeneration" type="string" default="allow_all">
      人物生成の制御：`allow_all`、`allow_adult`、`dont_allow`
    </ParamField>
  </Tab>
</Tabs>

## 制約条件

| 条件 | 要件 |
|------|------|
| 解像度 `1080p` または `4k` | `duration` は `8` である必要があります |
| 解像度 `4k` | VEO 3.1 および 3.1 Fast のみ対応 |
| 出力フレームレート | 24 FPS |
| 出力フォーマット | MP4 |

<Note>
VEO 3+ はネイティブで音声生成に対応しており、追加パラメータは不要です。プロンプト内で効果音を記述するだけです（例："the sound of waves crashing"）。
</Note>
