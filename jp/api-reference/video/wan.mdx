非同期処理モード — リクエスト送信後にタスク ID が返されます。[タスクステータス照会](/jp/api-reference/task/status) インターフェースで生成結果を取得してください。動画リンクの有効期限は24時間です。

<RequestExample>
```bash cURL
curl -X POST https://ai.kaiho.cc/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "wan2.5-t2v",
    "action": "generate",
    "prompt": "春天的樱花树下，花瓣随风飘落",
    "aspect_ratio": "16:9",
    "duration": 5
  }'
```

```python Python
import requests

response = requests.post(
    "https://ai.kaiho.cc/v1/tasks",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "wan2.5-t2v",
        "action": "generate",
        "prompt": "春天的樱花树下，花瓣随风飘落",
        "aspect_ratio": "16:9",
        "duration": 5
    }
)

task_id = response.json()["task_id"]
print(f"タスクを送信しました: {task_id}")
```

```java Java
import java.net.http.*;
import java.net.URI;

public class Main {
    public static void main(String[] args) throws Exception {
        String payload = """
        {
          "model": "wan2.5-t2v",
          "action": "generate",
          "prompt": "春天的樱花树下，花瓣随风飘落",
          "aspect_ratio": "16:9",
          "duration": 5
        }
        """;

        HttpClient client = HttpClient.newHttpClient();
        HttpRequest request = HttpRequest.newBuilder()
            .uri(URI.create("https://ai.kaiho.cc/v1/tasks"))
            .header("Authorization", "Bearer YOUR_API_KEY")
            .header("Content-Type", "application/json")
            .POST(HttpRequest.BodyPublishers.ofString(payload))
            .build();

        HttpResponse<String> response = client.send(request,
            HttpResponse.BodyHandlers.ofString());
        System.out.println(response.body());
    }
}
```

## 利用可能なモデル

| モデル ID | 説明 |
|---------|------|
| `wan2.5-t2v` | Wan 2.5 テキストから動画 |
| `wan2.5-i2v` | Wan 2.5 画像から動画 |
| `wan2.2-i2v-flash` | Wan 2.2 画像から動画 高速版 |
| `wan2.2-i2v-plus` | Wan 2.2 画像から動画 強化版 |

## 統一パラメータ

<ParamField body="model" type="string" required>
  モデル ID
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作タイプ
</ParamField>

<ParamField body="prompt" type="string" required>
  動画の説明テキスト
</ParamField>

<ParamField body="aspect_ratio" type="string" default="16:9">
  アスペクト比、バックエンドが自動的に最適な解像度にマッピングします
</ParamField>

<ParamField body="duration" type="integer" default="5">
  動画の長さ（秒）
</ParamField>

<ParamField body="resolution" type="string">
  解像度：`480p`、`720p`、`1080p`
</ParamField>

<ParamField body="seed" type="integer">
  ランダムシード、範囲 0-2147483647
</ParamField>

<ParamField body="watermark" type="boolean" default="false">
  透かしを追加するかどうか
</ParamField>

<ParamField body="image_urls" type="string[]">
  参照画像 URL（画像から動画モデルで使用）
</ParamField>

<ParamField body="first_frame" type="string">
  先頭フレーム画像 URL
</ParamField>

<ParamField body="last_frame" type="string">
  末尾フレーム画像 URL
</ParamField>

<ParamField body="negative_prompt" type="string">
  表示させたくない内容、最大500文字
</ParamField>

<ParamField body="callback_url" type="string">
  Webhook コールバックアドレス
</ParamField>

<ParamField body="extra" type="object">
  パススルーパラメータ
</ParamField>

## 生成モード

| モード | モデル | パラメータ |
|------|------|------|
| テキストから動画 | `wan2.5-t2v` | `prompt` |
| 画像から動画 | `wan2.5-i2v` / `wan2.2-i2v-*` | `prompt` + `image_urls` |
| 先頭末尾フレーム | `wan2.5-i2v` | `first_frame` + `last_frame` |
