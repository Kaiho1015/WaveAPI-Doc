<Note>
  **非同期処理モード：** リクエスト送信後にタスク ID が返されます。[タスクステータス照会](/jp/api-reference/task/get-status) インターフェースで生成結果を取得してください。生成された画像リンクの有効期限は24時間です。速やかに保存してください。
</Note>

## 認証方式

<ParamField header="Authorization" type="string" required>
  全てのインターフェースで Bearer Token 認証が必要です

  API Key の取得：

  [API Key 管理ページ](https://qingbo.dev/dashboard/keys) にアクセスして API Key を取得してください

  リクエストヘッダーに以下を追加してください：

  ```
  Authorization: Bearer YOUR_API_KEY
  ```
</ParamField>

## リクエストパラメータ

<ParamField body="model" type="string" default="gpt-4o-image" required>
  画像生成モデル名

  例：`"gpt-4o-image"`
</ParamField>

<ParamField body="prompt" type="string" required>
  画像生成のテキスト説明

  最大1000文字
</ParamField>

<ParamField body="size" type="string">
  画像生成サイズ

  対応するアスペクト比：

  * `1:1` - 正方形
  * `2:3` - 縦型
  * `3:2` - 横型
</ParamField>

<ParamField body="n" type="integer">
  生成する画像の数

  1、2、4 に対応、料金は数量に応じて先払いされます

  デフォルト値：1

  **注意：** 数値（例：`1`）を直接入力してください。引用符で囲まないでください
</ParamField>

<ParamField body="image_urls" type="array<url>">
  参照画像 URL リスト、画像から画像生成や画像編集に使用

  <Expandable title="詳細説明">
    配列の各要素は文字列で、2つのフォーマットに対応：

    **1. 完全な画像 URL**

    * パブリックにアクセス可能な画像 URL（http:// または https://）
    * 例：`"https://example.com/image.jpg"`

    **2. Base64 エンコード形式**

    * **完全な Data URI 形式を使用する必要があります**
    * 形式：`data:image/{format};base64,{base64data}`
    * 対応画像フォーマット：jpeg、png、webp
    * 例：`"data:image/jpeg;base64,/9j/4AAQSkZJRgABAQEAYABg..."`
    * 注意：`data:image/jpeg;base64,` プレフィックスを必ず含めてください

    **制限：**

    * 1枚あたり最大 10MB
    * 対応フォーマット：.jpeg、.jpg、.png、.webp
  </Expandable>

  **制限：** 最大5枚
</ParamField>

<ParamField body="mask_url" type="string">
  マスク画像 URL

  * PNG 形式である必要があります
  * サイズは参照画像と一致する必要があります
  * 最大 4MB
</ParamField>

## レスポンスパラメータ

<ResponseField name="code" type="string">
  レスポンスステータスコード

  * `success` - リクエスト成功、タスク送信済み
</ResponseField>

<ResponseField name="data" type="string">
  タスク ID、形式は `task-wave{timestamp}{userId}{random}`

  [タスクステータス照会](/jp/api-reference/task/get-status) インターフェースで生成結果を取得するために使用します
</ResponseField>

## 注意事項

1. **非同期処理：** タスク送信後に非同期で処理されます。[タスクステータス照会](/jp/api-reference/task/get-status) で生成結果を取得してください
2. **リンクの有効期限：** 生成された画像リンクの有効期限は24時間です。速やかにダウンロードして保存してください
3. **画像 URL の要件：** 入力する参照画像はパブリックにアクセス可能な URL または Base64 形式である必要があります
4. **料金について：** `n` が1より大きい場合、料金は数量に応じて先払いされます
