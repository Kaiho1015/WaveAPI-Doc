非同期処理モード — リクエスト送信後にタスク ID が返されます。[タスクステータス照会](/jp/api-reference/task/status) インターフェースで生成結果を取得してください。画像リンクの有効期限は24時間です。速やかに保存してください。

<RequestExample>
```bash cURL
curl -X POST https://www.qingbo.dev/v1/tasks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "seedream-5.0",
    "action": "generate",
    "prompt": "一只柴犬穿宇航服站在月球上",
    "size": "16:9",
    "resolution": "2K",
    "watermark": false
  }'
```

## 利用可能なモデル

| モデル ID | 説明 |
|---------|------|
| `seedream-5.0` | Seedream 5.0、グループ画像、ウェブ検索、ストリーミング出力に対応 |
| `seedream-4.5` | Seedream 4.5、グループ画像、4K 解像度に対応 |
| `seedream-4.0` | Seedream 4.0、グループ画像、1K~4K 解像度に対応 |

## 統一パラメータ

<ParamField body="model" type="string" required>
  モデルグループ名：`seedream-5.0`、`seedream-4.5`、`seedream-4.0`
</ParamField>

<ParamField body="action" type="string" default="generate">
  操作タイプ
</ParamField>

<ParamField body="prompt" type="string" required>
  テキストプロンプト、中国語・英語に対応、300文字以内を推奨
</ParamField>

<ParamField body="image_urls" type="string[]">
  参照画像 URL 配列、最大14枚、1枚あたり10MB以下
</ParamField>

<ParamField body="size" type="string" default="1:1">
  アスペクト比：`1:1`、`2:3`、`3:2`、`3:4`、`4:3`、`9:16`、`16:9`、`21:9`
</ParamField>

<ParamField body="resolution" type="string" default="2K">
  解像度、選択可能な値はモデルにより異なります。各モデルの Tab を参照
</ParamField>

<ParamField body="n" type="integer" default="1">
  生成数、1より大きい場合はグループ画像モードが有効になります
</ParamField>

<ParamField body="watermark" type="boolean" default="true">
  透かしを追加するかどうか
</ParamField>

<ParamField body="callback_url" type="string">
  タスク完了時の Webhook コールバックアドレス
</ParamField>

<ParamField body="extra" type="object">
  上流にパススルーされる追加パラメータ、各モデルの Tab を参照
</ParamField>

## 生成モード

| モード | パラメータ | 説明 |
|------|------|------|
| テキストから画像 | `prompt` | テキストのみで生成 |
| 単一画像から画像 | `prompt` + `image_urls`(1枚) | 参照画像 + テキスト |
| 複数画像から画像 | `prompt` + `image_urls`(2-14枚) | 複数の参照画像を融合 |
| グループ画像 | `n > 1` | 一連の関連画像を生成 |

<Note>
グループ画像モードでは、入力参照画像数 + 生成画像数 ≤ 15 です。
</Note>
