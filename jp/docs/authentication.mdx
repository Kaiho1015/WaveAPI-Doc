## 認証方式

全ての WaveAPI リクエストは Bearer Token 認証を使用します：

```bash
Authorization: Bearer YOUR_API_KEY
```

## API Key の取得

<Steps>
  <Step title="アカウント登録">
    [Dashboard](https://qingbo.dev/dashboard/keys) にアクセスして登録してください
  </Step>
  <Step title="キーの作成">
    キー管理ページで「新しいキーを作成」をクリックしてください
  </Step>
  <Step title="キーの保存">
    コピーして安全に保存してください。キーは一度しか表示されません
  </Step>
</Steps>

## Base URL

```
https://qingbo.dev/v1
```

全てのエンドポイントはこの URL を基準としています。既に OpenAI SDK をご利用の場合は、`base_url` を置き換えるだけで利用できます。

<Warning>
フロントエンドのコードや公開リポジトリで API Key を公開しないでください。
</Warning>
