## 利用可能なモデルの照会

```bash
curl https://qingbo.dev/v1/models \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## 個別モデルの照会

```bash
curl https://qingbo.dev/v1/models/{model_name} \
  -H "Authorization: Bearer YOUR_API_KEY"
```

## 課金タイプ

| タイプ | 説明 | 対象 |
|------|------|------|
| Token 課金 | 入力/出力 Token 数に基づく | テキスト生成、埋め込み |
| 回数課金 | リクエストごとの固定料金 | 一部の特殊エンドポイント |
| 画像課金 | 生成数と解像度に基づく | 画像生成 |
| 動画課金 | 長さと解像度に基づく | 動画生成 |
| 音声課金 | 音声の長さに基づく | TTS、STT |
