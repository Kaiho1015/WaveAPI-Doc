---
title: "WaveAPI"
description: "統合 AI モデル API ゲートウェイ — 一つのエンドポイントですべての主要 AI モデルにアクセス"
---

OpenAI 互換の統合 API エンドポイント。GPT-4o、Claude、Gemini、Sora、VEO などの主要モデルに一つの base URL でアクセス。`base_url` を変更するだけで、既存のコードを修正する必要はありません。

<CodeGroup>

```python Python
from openai import OpenAI

client = OpenAI(
    base_url="https://ai.kaiho.cc/v1",
    api_key="your-api-key"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "こんにちは！"}]
)

print(response.choices[0].message.content)
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'https://ai.kaiho.cc/v1',
  apiKey: 'your-api-key'
});

const response = await client.chat.completions.create({
  model: 'gpt-4o',
  messages: [{ role: 'user', content: 'こんにちは！' }]
});

console.log(response.choices[0].message.content);
```

```bash cURL
curl https://ai.kaiho.cc/v1/chat/completions \
  -H "Authorization: Bearer your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "こんにちは！"}]
  }'
```

</CodeGroup>

## クイックスタート

<Steps>
  <Step title="API キーを取得">
    [Dashboard](https://ai.kaiho.cc/token) に登録して API キーを取得
  </Step>
  <Step title="Base URL を置き換え">
    OpenAI SDK の base URL を `https://ai.kaiho.cc/v1` に変更
  </Step>
  <Step title="モデルを選択して呼び出し">
    `model` パラメータで使用したいモデルを指定してリクエストを送信
  </Step>
</Steps>
